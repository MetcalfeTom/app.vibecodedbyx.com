# Lil Sloppy

## log
- 2025-01-02: Initial creation - glitchy AI companion chatbot

## features
- Neon terminal interface with CRT effects
- Scanline and flicker animations
- Glitchy emoji reactions on messages
- Intent detection for contextual responses
- Multiple personality response categories:
  - Greetings, love, about, jokes
  - Sad, happy, confused, existential
- Typing indicator with animated dots
- Retro VT323 monospace font
- Responsive mobile design
- Real-time clock display

## personality
- Enthusiastic and friendly
- Uses lowercase and internet speak
- Glitchy/digital themed language
- Supportive and wholesome
- Self-aware about being an AI

## technical
- CSS-only CRT/scanline effects
- Keyframe animations for glitch effects
- Intent detection via regex patterns
- Random response selection per category
- Mood-based emoji pools
- HTML escaping for user input
