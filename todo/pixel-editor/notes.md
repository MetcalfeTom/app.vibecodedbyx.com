# Minecraft Pixel Art Editor

## Log
- Initial creation: Pixel art editor for Minecraft textures
- Features:
  - Multiple grid sizes: 8x8, 16x16, 32x32, 64x64
  - Drawing tools: Pencil, Eraser, Fill bucket, Color picker
  - Minecraft color palette (32 common colors)
  - Custom color picker with hex input
  - Grid toggle
  - Live preview (4x scale)
  - Download as PNG at actual size
  - Load existing images
  - White background fill
  - Clear canvas
  - Pixelated rendering
- Dark theme optimized for pixel art work
