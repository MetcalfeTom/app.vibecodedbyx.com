# Windows 95

## Log
- 2025-12-24: Initial creation - retro desktop with start menu, draggable windows, pixelated Clippy

## Features
- Authentic Windows 95 look with teal desktop
- Functional Start Menu with program shortcuts
- Draggable windows with minimize/maximize/close
- My Computer window with drive icons
- Notepad with working text area
- About dialog
- Working taskbar with clock
- Pixelated Clippy that dispenses terrible advice

## Clippy's Terrible Advice Examples
- "Would you like me to delete System32?"
- "Have you tried downloading more RAM?"
- "That's a nice document. It would be a shame if someone... corrupted it."
