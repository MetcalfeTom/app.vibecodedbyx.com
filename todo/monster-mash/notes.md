# Monster Mash

## log
- 2025-12-28: Initial creation
  - Kid-friendly monster builder
  - Emoji-based parts: body, head, eyes, mouth, arms, legs, accessories
  - 10 color options with visual feedback
  - Wiggly CSS animations (slow, fast, bounce, spin)
  - Web Audio API sound effects (boing, pop, woosh, blurp, monster)
  - Randomize button with silly generated names
  - Extra Wiggle mode for faster animations
  - Sound toggle for quiet mode
  - Click monster for sound effect
  - Playful Fredoka font
  - Gradient background with floating shapes
  - Mobile responsive

## notes
### Part Categories:
- Body: shapes and hearts
- Head: creature faces (alien, robot, ghost, etc.)
- Eyes: various eye styles
- Mouth: expressions
- Arms: hands and appendages
- Legs: feet and movement items
- Accessory: hats and decorations

### Sound Types:
- `pop`: Part selection
- `woosh`: Color change / randomize
- `boing`: Toggle wiggle
- `blurp`: Click monster
- `monster`: Make Sound button (multi-tone chaos)
