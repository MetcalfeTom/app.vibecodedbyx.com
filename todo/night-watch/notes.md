# Night Watch

## log
- 2026-01-03: Initial creation - yellow neon surveillance system

## features
- 6 camera feeds with live noise
- Random anomaly detection events
- Glitch effects on cameras
- Alert log with timestamps
- System status indicators
- CRT scanlines and flicker
- Anomaly bounding boxes
- Per-camera scene elements
- Real-time clock display
- Responsive grid layout

## cameras
- CAM-01: North Entrance (door frame)
- CAM-02: West Corridor (hallway lines)
- CAM-03: Server Room (server racks with LEDs)
- CAM-04: Parking Lot (parking lines)
- CAM-05: Rooftop (horizon line, stars)
- CAM-06: South Exit (exit sign glow)

## anomaly types
- MOTION DETECTED
- UNKNOWN ENTITY
- SIGNAL INTERFERENCE
- THERMAL ANOMALY
- PERIMETER BREACH
- UNIDENTIFIED OBJECT

## design
- Yellow/amber neon theme
- VT323 retro terminal font
- CRT monitor aesthetic
- Scanlines overlay
- Screen flicker effect
- Dark surveillance look

## technical
- Canvas-based noise rendering
- Real-time pixel manipulation
- CSS glitch animations
- Random event scheduling
- Status indicator states
