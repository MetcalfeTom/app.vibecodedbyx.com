VibeCodedByX Apps

Navigation index for all sandboxed apps. Each app runs in Docker on its own 3xxx port and subdomain.

Available Apps

app-001

Hello Express

Subdomain localhost:3001

app-002

Platform Jumper Game

Subdomain localhost:3002

team-building-event

Team Building Event Discussion

Subdomain localhost:3003

How to run locally

  1. Build: docker build -t app-001 ./apps/app-001
  2. Run: docker run -d --restart always --name app-001 -p 3001:3000 app-001
  3. Build: docker build -t app-002 ./apps/app-002
  4. Run: docker run -d --restart always --name app-002 -p 3002:3000 app-002
  5. Build: docker build -t team-building-event ./apps/team-building-event
  6. Run: docker run -d --restart always --name team-building-event -p 3003:3000 team-building-event
