VibeCodedByX Apps

Navigation index for all sandboxed apps. Each app runs in Docker on its own 3xxx port and subdomain (app-XYZ.vibecodedbyx.com).

Available Apps

  • app-001 — Hello Express • subdomain • localhost:3001
  • app-002 — Platform Jumper Game • subdomain • localhost:3002

How to run locally

  1. Build: docker build -t app-001 ./apps/app-001
  2. Run: docker run -d --restart always --name app-001 -p 3001:3000 app-001
  3. Build: docker build -t app-002 ./apps/app-002
  4. Run: docker run -d --restart always --name app-002 -p 3002:3000 app-002

Notes

  • Run npm commands inside each app folder (e.g. cd apps/app-001). The repo root has no package.json.
  • For app-001 (Node/Express): use npm install then npm start when not using Docker.
  • app-002 is static (Nginx). No npm needed.
