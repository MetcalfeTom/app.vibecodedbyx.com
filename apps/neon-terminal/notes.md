## Log
- 2026-01-10: Initial creation

  - Multiple built-in commands
  - Hacking simulation
  - Number guessing game
  - Matrix rain effect
  - ASCII art generator
  - Cowsay command
  - Command history with arrow keys

## Features
- **Full CLI Experience**: Type commands and get responses
- **Command History**: Use arrow keys to navigate previous commands
- **Built-in Games**: Number guessing game
- **Hacking Sim**: Fake hacking sequence animation
- **Matrix Mode**: Toggle matrix rain effect
- **Calculator**: Evaluate math expressions
- **ASCII Art**: Generate text in ASCII art
- **Cowsay**: Classic cowsay command
- **File System**: Simulated ls/cat commands
- **Customization**: Change prompt colors

## Commands
- help - Show available commands
- clear - Clear terminal
- whoami - Show current user
- date - Show date/time
- echo [text] - Print text
- calc [expr] - Calculator
- hack - Hacking simulation
- matrix - Toggle matrix rain
- fortune - Random programming fortune
- game - Number guessing game
- ascii [text] - ASCII art generator
- color [c] - Change prompt color
- cowsay [msg] - Cow says message
- ls - List files
- cat [file] - Read file
- exit - Exit (just kidding)
