# Window War

Catch runaway windows before they minimize your game. Click them before they escape!

## log
- 2026-04-12: Initial build. Fake OS desktop with taskbar, start button, clock, desktop icons. 10 themed window types (error.exe, runaway.app, virus_scan.bat, budget.xlsx, memories.dll, IMPORTANT.txt, task_mgr.exe, feelings.log, cmd.exe, love_letter.doc) each with unique title bar color, funny body text, and personality. Windows spawn from screen edges and drift toward taskbar to "minimize." Random dodge behavior makes windows evasive. Urgency shake and fade as windows approach timeout. Click/tap to catch for points (time bonus + wave bonus). 5 lives, lose one per escaped window. Waves every 15s increase speed, spawn rate, and window count. Catch pop animation, minimize shrink animation. Taskbar shows active window list. Catch/escape/gameover SFX. Title screen and game over with stats. IBM Plex Sans + IBM Plex Mono typography, Windows-inspired blue desktop aesthetic.

## features
- 10 themed runaway window types with unique content
- Windows spawn from edges, drift toward taskbar
- Random dodge behavior (windows try to evade clicks)
- Urgency shake + fade near timeout
- Click/tap to catch windows
- Time bonus scoring (faster catch = more points)
- 5 lives system
- Wave progression (speed, count, spawn rate increase)
- Fake OS desktop with taskbar, start button, clock, icons
- Active window list in taskbar
- Catch pop and minimize shrink animations
- Sound effects (catch, escape, game over)
- Title screen and game over with stats
- Touch support
