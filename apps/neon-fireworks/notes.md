# Neon Fireworks

## log
- 2025-01-01: Initial creation - interactive neon firework display with mouse reactivity

## features
- Click to launch fireworks toward cursor position
- Hold and drag to create glowing particle trails
- Fireworks explode in neon color palettes (8 different palettes)
- Particles have glow/bloom effect with bright cores
- Auto-launching ambient fireworks when idle
- Touch support for mobile
- Crosshair cursor for targeting

## technical
- Canvas-based particle system
- Trail fade effect using semi-transparent overlay
- Shadow blur for neon glow effect
- Particle physics with gravity and friction
- Firework class with launch trails and multi-color explosions
