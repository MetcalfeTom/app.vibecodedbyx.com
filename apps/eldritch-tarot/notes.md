# eldritch-tarot

## log
- 2026-05-26: Initial build per missmeowd's giggles. Full 22-card Major Arcana reskinned for things-that-should-not-be-named: Wanderer / Magus / Veiled / Mother / Tyrant / Cult / Pact / Wraith / Scale / Lantern / Wheel / Maw / Suspended / Shroud / Alembic / Horned / Collapse / Burning / Hollow / Eye / Reckoning / Void. Each card has: roman numeral, name, hand-crafted inline SVG sigil (100×100 viewBox, current-color stroked + filled with parchment-ink contrast), upright pool of 3-4 cryptic predictions, reversed pool of 2-3 darker predictions. **Card mechanics**: CSS 3D card flip via `transform-style:preserve-3d` + `backface-visibility:hidden` on `.face.back`/`.face.front`. Back: radial purple→void gradient with double-bordered frame + The Void sigil centered (a circle of circles). Front: parchment gradient (#ebd9a8→#c4b287) with ink border + repeating-linear grain mixed-blend multiply + corner blood-radial blotch; numeral top, sigil center, gothic name bottom. Reversed cards add `~` markers around the name and rotate the sigil + numeral + name 180°. Card animations: 0.7s `dealIn` keyframe (drop-from-above + rotate + scale-fade-in with stagger), 0.85s ease flip with cubic-bezier(0.7,0.05,0.3,1). **Two draw modes**: Single Card (one card centered, click to flip) and Past·Present·Future spread (3 cards in row with stagger-dealt 220ms apart, labeled above). Deck shuffles via Fisher-Yates on init + via Shuffle button; deck index pops on draw, autoshuffles when empty. 33% chance per card to deal reversed. **Reading display**: each revealed card spawns a `.reading` block below — left blood-red border, fade-up animation, position label (small caps candle-color), gothic card name in blood with optional reversed marker, then 2 random predictions from the appropriate pool (no repeats within a single reading), Cormorant italic with drop-cap candle-color first letter via `::first-letter`. **Ambient drone**: 4-voice chord (55/73.42/110/138.59 Hz sine+triangle with 0.5% detune) through resonant lowpass swept by 0.04-0.16Hz LFOs ±180Hz, 2.4s fade-in, 1.6s fade-out on toggle. **One-shot SFX**: shuffle = 4 staggered bandpass noise bursts (90ms apart, freq 1800→2400Hz); deal = single 3200Hz bandpassed noise click; flip = 620→240Hz triangle blip + lowpass-filtered noise thunk delayed 120ms. **Atmosphere**: candle-flicker keyframe on body::before (opacity wobble 3.2s ease), 6 base + endless 2.2s-cadence floating amber motes (3px glowing dots, 9s rise from below viewport with horizontal drift), radial-blended void bg with 4 layered glows (candle base + magenta top + blood side + violet side). **Defensive init pattern**: error toast on global error, AudioContext + matchMedia in try/catch. **Typography**: UnifrakturCook for title + card names + drop-caps (gothic blackletter), IM Fell English italic for position labels + tagline + button labels (Victorian print feel), Cormorant Garamond italic for prediction text + body, IBM Plex Mono for HUD numerals. **Palette**: void #0c0509 + parchment #d8c89a + blood #9a1830 + candle #e3a73f + violet #3a1b4a + ghost #c8b0e8 — high contrast, ritual-room feel. **WCAG basics**: rem units, semantic main/h1/section/button, aria-live on reading area, aria-pressed on drone + mute toggles, role=button + tabindex + keyboard reveal (Space/Enter) on cards, aria-label updates from "Face-down card" → card name on flip, focus-visible 3px candle outline, ≥2.75rem interactive targets, prefers-reduced-motion kills card flip animation duration to 200ms + disables candle flicker + mote drift. **Mobile**: cards shrink from 11rem×16.5rem to 8.6rem×13rem at ≤640px, name + numeral scale down. **JSDOM smoke test**: zero errors, deck initializes to 22-sealed, draw-one renders 1 card and drops deck to 21, click reveals card + adds flipped class + spawns reading with card name, draw-three renders 3 cards labeled Past/Present/Future.

## issues
- Card flip uses 3D CSS transforms — older Safari without `-webkit-backface-visibility` might show both faces. I included the prefix, should be fine.
- SVG sigils are inline strings rather than `<svg>` source files — slightly larger HTML but no external requests, plays nicely with `currentColor` for the parchment-ink theme.
