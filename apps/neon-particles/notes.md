# Neon Particle Storm

## Log
- 2024-12-24: Initial creation - neon particle system with mouse following and smooth trails

## Features
- High-performance canvas-based particle system
- Smooth trails with fade effect
- Multiple neon colors (pink, cyan, magenta, green, orange, purple)
- Particles have spinning motion and glow effects
- Click/tap for burst effect
- Mobile touch support
- Initial burst animation on load
