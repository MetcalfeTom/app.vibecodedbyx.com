# Task Manager

Kill malware pop-ups by clicking X or End Task before your CPU overloads.

## log
- 2026-04-17: Initial build. Retro Windows-style desktop with taskbar, icons, and teal background. 16 hilarious malware pop-up templates (virus alerts, crypto miners, ransomware, hot singles, FBI warnings, hamster alerts, etc.). Click X button or End Task to kill pop-ups. Clicking the fake OK/action buttons spawns MORE pop-ups as punishment. CPU load meter rises with active pop-ups — 100% triggers BSOD game over. Wave system: every 8 kills speeds up spawns and increases max popups. Draggable windows. Score = 100 + wave×20 per kill. Pop-ups have death animation, spawn animation, and random shake.

## features
- Retro Windows XP-style desktop aesthetic
- 16 unique malware pop-up templates with humor
- Click X button or End Task to kill pop-ups
- Fake buttons spawn MORE pop-ups (punishment mechanic)
- CPU load meter: each popup adds 12% load
- BSOD game over screen when CPU hits 100%
- Wave system (every 8 kills): faster spawns, more max popups
- Draggable pop-up windows
- Score with wave multiplier
- Pop-up color variants (red, blue, green, orange, purple)
- Spawn/death/shake animations
- Desktop icons, taskbar with clock, Start button
- Mobile tap support
