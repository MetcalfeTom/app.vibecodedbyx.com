# Tickle the Grape

## log
- 2026-01-05: Initial creation - neon tickle simulator with laughing grape

## features
- Click/tap the grape to tickle it
- Wiggle animation on tickle
- Confetti explosion particles
- Giggle sound effects (Web Audio API)
- Floating giggle text ("Hehe!", "Teehee!", etc.)
- Random laughing face overlay
- Combo meter for rapid clicking
- MEGA MODE at 100% combo (rainbow effects, more confetti)
- Mobile-friendly touch support

## design
- Neon purple/pink color scheme
- Grid background pattern
- Pulsing glow around grape
- Bubblegum Sans + Fredoka fonts
- Smooth CSS animations
