## issues
- **Native `<marquee>` is deprecated** but still renders in all modern browsers. Crucial for authenticity. If a future browser kills it, fallback to CSS keyframe scroll.
- **Background animation may strain low-end devices** — 4 layered animations + cursor trail + emoji confetti. If perf complaints, lower bg-shift/tile-roll cadence.
- **Cursor trail spawns DOM nodes constantly** but throttled 1/5 frames + 800ms cleanup. Check Memory tab if a session runs > 10 min.
- **Lightning + honk-alert use mix-blend / inset shadows** which can flicker on Safari iOS — acceptable, adds to charm.
- **window.status is a no-op in modern browsers** but the call is harmless and adds period authenticity.
- **Counter increments on its own** (+1–3 every 4.5s) so users see numbers go up. Clearly fake, stylistically appropriate.
- **No persistence intentionally** — every visit starts fresh at 1337.
- **Audio left out by design** — no auto-playing midi (browsers block it anyway, and the joke works without sound). Could add toggleable midi-style synth later.
