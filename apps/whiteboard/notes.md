## features
- Drawing Tools:
  - Pen with smooth lines
  - Eraser (3x brush size)
  - 6 colors (black, red, blue, green, orange, purple)
  - Adjustable brush size (2-30px)
- Sticky Notes:
  - 6 colors (yellow, pink, blue, green, orange, purple)
  - Drag anywhere on canvas
  - Editable text (Caveat handwriting font)
  - Delete button
  - Color cycle button
- Canvas:
  - Dotted grid background
  - Full screen
  - Touch support
- Actions:
  - Undo (removes last stroke)
  - Clear all (drawings + notes)

## design
- Inter + Caveat fonts
- Clean minimal UI
- Floating toolbar
- Soft shadows
- Grid paper background

## controls
- P: Pen tool
- E: Eraser tool
- S: Sticky note tool
- Ctrl+Z: Undo
- Click on canvas: draw or place sticky
- Drag notes: move them around
