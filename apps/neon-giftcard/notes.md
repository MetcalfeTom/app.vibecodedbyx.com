# Neon Gift Card Generator

## log
- 2026-01-15: Added evading download button and 90s stickers
  - Download button jumps away from mouse 5 times before letting you click
  - Taunting messages: "Too slow!", "Can't catch me!", etc.
  - 16 draggable 90s clip art stickers (fire, skulls, lightning, aliens, etc.)
  - Double-click stickers to remove them
  - Stickers included in PNG download
- 2026-01-15: Initial creation
  - Upload any embarrassing photo
  - Comic Sans text overlay with neon glow
  - Drag text to reposition anywhere
  - 8 quick message presets (Birthday, Congrats, Get Well, Sorry, Xmas, Thanks, Love, Achievement)
  - 6 neon color options for text
  - Text size slider
  - Download as PNG
  - Magenta neon border with glow
  - Orbitron font for UI, Comic Sans for card text
  - Mobile responsive with touch drag support

## features
- Photo upload (any image)
- Comic Sans MS text with neon shadow effects
- Draggable text positioning (mouse and touch)
- Quick preset messages
- Text color picker (6 neon colors)
- Text size control
- 16 draggable 90s clip art stickers
- Evading download button (jumps 5x before you can click)
- PNG download with all effects + stickers
- Neon pink glowing border
- Mobile friendly
