# Plant CLI

## log
- 2026-01-12: Created neon productivity plant CLI game

## features
- Virtual pet plant that grows with task completion
- 8 plant growth stages (seed to legendary)
- ASCII art plant display with neon colors
- Task management (add, complete, list)
- XP system with level-ups
- Water your plant mechanic (+XP, cooldown)
- 5-task streak bonuses
- Name your plant
- Progress saved to localStorage
- Terminal/CLI aesthetic with scanlines
- Commands: help, add, done, tasks, water, status, name, clear, reset
- Type anything to add it as a task
- Click "done" buttons or use "done #" command
