# GTA Portfolio

## log
- 2026-04-02: Created GTA Portfolio — Vice City / GTA 6 themed developer portfolio for fictional "VICE_DEV". Single-file HTML app with Bebas Neue + Teko + Source Sans 3 typography. Neon pink/orange gradient palette, parallax silhouette, loading screen, Intersection Observer fade-ins, scanline overlay, palm tree SVG decorations, wanted-star skills, mission-card projects, timeline experience, pause-menu contact section.

## features
- GTA-style loading screen with progress bar and rotating tips
- Animated hero entrance (slide + fade + scale)
- Parallax scrolling (silhouette at 0.3x, background at 0.5x)
- Neon flicker CSS animation on select headings
- Intersection Observer for section fade-in on scroll
- Wanted-level star ratings for skills (pure CSS)
- Mission briefing project cards with hover glow + "MISSION PASSED" stamps
- Vertical timeline with pulsing neon dots
- Pause-menu styled contact box
- Scanline/VHS overlay effect
- Custom scrollbar with neon gradient
- Palm tree SVG silhouettes as hero decoration
- Palm tree emoji dividers between sections
- Mobile responsive
