# neon-cat-cafe

## log
- 2026-05-27: Initial build. **Top-down pixel cafe** at 28×18 tiles (24px each) = 672×432 canvas with `image-rendering: pixelated`. Hand-authored ASCII tile map for the cafe layout: outer brick walls, 3 glowing pink window panes + 6-tile neon "MEOW CAFE" sign across the top wall, 4 round wooden tables (2-tile vertical T/t pieces) with adjacent chairs, a pink cat bed, a green food bowl, a wooden scratching post with yellow ball on top, and a right-edge column of potted plants. Each tile type renders procedurally with 24px chunk pixel art — wall has brick lines + magenta top stripe, tables have 2-row top+leg distinction with gold rim glow, chairs have cyan accent strip, food bowl has green kibble, scratch post has wood + ball, window has starfield + pink glow, neon sign hue-cycles every frame and pulses. **Cat AI**: each cat picks a wander target (~70% random floor, ~30% feature: bed/food/scratch/chair) and walks at 32-40 u/s along axis-tested cat-walkable tiles. Arriving at bed → 70% sleep state (8-14s, periodic Z particles, curled-up sprite); arriving at food → 4s eat state + happiness +0.15; arriving at scratch → sit state + happiness +0.10; arriving at chair → 50% sit (3-7s). State machine: `wander | sit | sleep | eat | pet`. **8 cat types**: Slate (gray, base), Coral (pink tuxedo), Mint (green stripes), Azure (cyan spots), Butter (yellow stripes), Grape (violet tuxedo), Shadow (dark with lime eyes + glow halo), Rainbow (cycling hue stripes, 4% chance per spawn, +8 points/pet). **Milestone unlocks** tied to lifetime points: 10/30/80/200/500/1000 thresholds — when crossed, banner pops "UNLOCKED · X CAT" + ascending-arpeggio chime + immediate bonus spawn. **Cat sprite** drawn procedurally per frame ~16px tall — body + lighter underside + darker back stripe + pattern overlay (stripes/spots/tuxedo white chest+socks/rainbow diagonal hue stripes), head with ears + inner-ear pink + 2 eyes (state-dependent: open w/ eye color + catchlight, blink crescents at random 1.8-5s intervals, happy-crescents during pet), pink nose, 2-frame leg shuffle synced to walkPhase, wagging tail. Sleeping cats render as curled-wider blob with closed eyes + tail wrapped + tiny ear bumps. Rainbow and Shadow cats get an aura glow halo behind the sprite. **Pet mechanic**: click/tap within 18px of cat center → nearest cat triggers `pet` state for 1.6s (cooldown 1.2s per cat). Pet rewards base points (1/2/3/4/8 depending on type), +1 bonus if happiness > 0.85. Pet spawns 5+ heart particles (per-cat-hue) + sndPurr (two-tone triangle chirp pitched by hue) + floating "+N" DOM popup that animates up & fades. Rainbow pets show "+N ✨" in gold. **Treat button**: spend 10 points to drop an orange fish treat at a random floor spot. Nearest cats path to it, eat = +5 points + +0.40 happiness + 6 hearts. Treats last 20s. Button auto-disables when insufficient funds. **Persistence**: localStorage `neon-cat-cafe-v1` saves total points, total pets, unlocked types Set, rainbow-seen flag. Unlocks re-apply silently on load so old players don't re-trigger banners. **Spawning**: 1 cat at start (always Slate), auto-spawn every 12-20s up to max `3 + unlockedTypes.size` (caps at 8). **Particles**: hearts (5x5 pixel heart, hue-tinted, rises with gravity), Z's (lime Press Start 2P "z" rising and slowing), sparks (small color squares for ambient bursts). **Sound** (Web Audio synth, no samples): pet chirp = triangle 600+hue*0.6 → 1.5× + sine harmonic 1.5× → 2.2×; unlock = 4-note ascending C-major arpeggio; treat = 880→1320 triangle + 660→440 sine. **Idle quote ticker** rotates every 10s through 9 cozy observations ("a tail flicks.", "a small dream begins."). **HUD top**: purr-points (gold, persistent) / cats X/MAX (cyan) / pets (lime). Bottom strip: quote + Treat button + mute. **Defensive init**: try/catch on matchMedia, canvas, every handler; magenta error toast on window.error. **WCAG basics**: rem units, semantic main/h1, role=application + descriptive aria-label on canvas, role=group on HUDs, aria-live on quote + unlock banner, aria-pressed on mute, aria-label on every button, focus-visible 2-3px outlines, ≥2.5rem touch targets, prefers-reduced-motion damps dt ×0.85 + shortens floater animation. **Aesthetic**: Bowlby One SC pink→magenta→gold gradient title, Press Start 2P pixel tag + quote font, Audiowide button labels, Share Tech Mono HUD numerals. Dark void bg with pink + cyan radial glows. Sprites + tiles use `image-rendering: pixelated` for crisp NES-style pixel boundaries.

## issues
- Cat sprites are drawn fresh each frame rather than baked to offscreen canvases — fine for 8 cats but profile if scaling to 20+.
- Cats can pass through chairs (intentional — chairs are non-blocking, so cats can sit on them).
- The 2-tile bed pattern in LAYOUT is currently represented as two single-tile bed sprites side-by-side; cat AI handles both as separate bed tiles. Looks fine visually since each cell is independently rendered.
- Treat magnetism makes ALL cats path to a treat — could feel chaotic if many cats present. Acceptable for the "cafe excitement" feel.
