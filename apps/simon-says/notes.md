# Neon Simon

## log
- 2026-01-12: Created neon Simon Says game with glowing panels and synth sounds

## features
- Classic Simon Says gameplay
- 4 neon glowing panels (green, red, yellow, blue)
- Synth sound effects using Web Audio API
- Each color has its own musical note (G4, E4, C4, A3)
- Intense glow effects when panels activate
- Score tracking with high score in localStorage
- Sound toggle button
- Keyboard controls (Q/W/A/S or 1/2/3/4)
- Space/Enter to start game
- Game over animation (all panels flash)
- Error sound on wrong input
- Status messages for game state
- Mobile-friendly with touch support
- Neon cyberpunk aesthetic
