## features
- Babylon.js 3D scene
- Arc rotate camera (drag to rotate, scroll to zoom)
- Grid ground plane
- Starter cyan cube
- Spawn colorful boxes (button or spacebar)
- Reset scene (button or R key)
- Glow layer for emissive materials
- FPS counter
- Hemisphere + point lighting

## controls
- Mouse drag: rotate camera
- Scroll: zoom in/out
- Space: spawn random box
- R: reset scene
- Spawn Box button: add random box
- Reset button: clear and reset

- Space Grotesk font
