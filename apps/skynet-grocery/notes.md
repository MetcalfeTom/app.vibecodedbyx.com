# Skynet Grocery

## log
- 2026-04-02: Initial build. Single-file HTML app — Terminator-themed grocery list with glowing red eye, AI commentary, localStorage persistence. Orbitron + Share Tech Mono typography. Dark metallic aesthetic with circuit board background, scanline overlay, red ambient glow.

## features
- Red glowing eye with pulse animation, scan line, and mouse tracking
- Terminal-style commentary box with typewriter effect
- Specific responses for ~24 common grocery items (milk, bread, eggs, coffee, beer, etc.)
- 25+ generic menacing AI responses for other items
- localStorage persistence
- Checkbox to mark acquired (strikethrough), delete with red flash animation
- Items slide in on add
- Stats bar (total, acquired, status)
- All-complete and empty state messages
- Mobile responsive
