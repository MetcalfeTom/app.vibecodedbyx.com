# World Crisis Dashboard

## log
- 2026-01-19: Initial creation - absurd world crisis monitoring dashboard

## features
- Fake crisis monitoring dashboard
- 10 absurd crisis types:
  - Rubber Duck Invasion
  - Spaghetti Rain Event
  - Sock Puppet Uprising
  - Giant Toast Emergency
  - Cheese Tsunami Warning
  - Aggressive Accordion Outbreak
  - Penguin Traffic Jam
  - Balloon Animal Swarm
  - Glitter Contamination
  - Yodeling Epidemic
- Live updating statistics
- Breaking news ticker
- Global threat map with hotspots
- Live alert feed with new alerts every 8s
- Regional status breakdown
- Weekly absurdity index chart
- Threat levels: Critical, High, Moderate, Low

## design
- Military/command center aesthetic
- Share Tech Mono + Rajdhani fonts
- Green terminal text on dark background
- CRT scanline overlay
- Pulsing hotspots on map
- Red accent color for critical alerts
- Blinking status indicators

## data updates
- Clock updates every second
- Stats fluctuate every 3 seconds
- New alert added every 8 seconds
- Randomized alert selection

## panels
- Global Threat Map: Visual hotspots
- Crisis Overview: 4 key statistics
- Active Incidents: Top 4 crises with details
- Weekly Absurdity Index: Bar chart
- Live Alert Feed: Scrolling alerts
- Regional Status: 5 world regions
