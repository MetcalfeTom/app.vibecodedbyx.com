# Silence Meter

## log
- 2026-01-14: Created silence meter that screams and deletes its own buttons

## features
- Timer counting seconds of silence
- Scream intensity bar that fills up over time
- 9 escalating scream levels from "..." to "SILENCE HAS WON"
- Audio screams get louder/more frequent with time
- Buttons start self-deleting after 25 seconds
- Chaos levels with visual effects:
  - Background gets redder
  - Title starts shaking
  - Glitch effects
  - Warning overlay flashes
  - Floating chaos text spawns
  - Watching eyes appear
- 8 action buttons:
  - Break Silence: resets timer
  - Scream: -5 seconds
  - Whisper: +3 seconds (makes it worse)
  - Panic: -10 seconds with rapid screams
  - Reset: restores all buttons
  - Mute: toggle sound
  - Add Time: +15 seconds (why would you?)
  - Sacrifice: delete a button for -20 seconds
- Deleted button counter
- Creepster font for horror aesthetic
- Red/black color scheme
- Mobile responsive
