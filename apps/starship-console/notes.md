# Starship Console

Neon starship bridge console with flickering buttons, scrolling starfield map, and full systems readout.

## log
- 2026-04-12: Initial build. 3-panel layout: left systems readout, center scrolling starfield, right controls + comms log. Ship systems: power reactor, shields, hull, fuel, oxygen, temperature, heading, sector. 8 interactive console buttons (shields, weapons, scan, cloak, comms, auto-nav, warp, red alert) with LED indicators, flicker animations, and state toggles. Starfield: 3 parallax layers (380 stars total) with twinkle, 6 nebula blobs, 12 named celestial objects (planets, stations, anomalies, asteroids), grid overlay, scan pulse ring. Warp drive: charge bar, warp streaks on stars. Red alert mode with screen flash. WASD/arrow manual flight. Random events in comms log (12 types). Clock, coordinate display, sector naming. Rajdhani + Share Tech Mono typography, cyan/magenta neon on dark blue-black.

## features
