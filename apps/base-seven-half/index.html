HOW TO PLAY

In base 7½, each digit position represents a power of 7.5:

7.5⁰ = 1
7.5¹ = 7.5
7.5² = 56.25
7.5³ = 421.875

All 15 digits (0–14) use custom Argam glyphs — hand-drawn SVG symbols forming a complete alien numeral system. Each digit has a unique stroke-based form. The decimal value is shown below each glyph. Place a digit in each position so the weighted sum matches the target.

Example: target 23
→ digit 3 in the 7.5¹ position: 3 × 7.5 = 22.5
→ need 0.5 more... but digits are integers!
→ this is the puzzle — not every number is exactly representable.
→ get as close as possible, or find one that hits exactly.

Target 16: digit 2 in 7.5¹ + digit 1 in 7.5⁰
→ 2×7.5 + 1×1 = 15 + 1 = 16 ✓

The fractional base means some targets have exact solutions and some require creative approximation. Targets are always chosen to be exactly representable. Select a digit, then click a position to place it.
