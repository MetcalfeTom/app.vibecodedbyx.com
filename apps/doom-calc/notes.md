# Doom Calc

## log
- 2026-04-04: Initial build — sinister calculator with dark red gothic theme that always gives wrong answers. 10 corruption methods (off-by-one, digit swap, sign flip, sneaky factor, 666, digit drop, reversed digits, wrong digit, spooky numbers, tiny offset). 20 whisper messages. UnifrakturCook + Crimson Pro + DM Mono typography, blood drip animations, ornamental corners, ambient display flicker.

## features
- Full calculator: +, -, ×, ÷, %, ±, chained operations
- 10 different corruption algorithms — never gives the right answer
- Gothic whisper messages after each calculation (20 unique)
- Dark red/bone color scheme with blood drip CSS animations
- Ornamental corner decorations
- Display flicker and shake effects
- Keyboard support (0-9, +, -, *, /, Enter, Esc)
- Mobile responsive
- Scanline overlay

## issues
- Very rarely, a corruption might accidentally produce the correct answer
- Long decimal results can overflow display width
- No memory/history function
