# Tic Tac Toe

## log
- 2025-12-29: Initial creation - polished glassmorphism tic tac toe with smooth animations

## features
- Glassmorphism design with frosted glass cards
- Animated gradient background with floating orbs
- Smooth piece placement animations (scale + rotate)
- Animated win line with gradient
- Score tracking (X wins, O wins, draws)
- Active player indicator with glow effect
- Celebration particles on win
- Fully responsive for mobile
- Outfit font for modern typography
