# Holo Terrarium

A holographic glass jar with glowing moss and code-fragment fireflies.

## log
- 2026-04-13: Initial build. Glass terrarium jar with cork lid, holographic sheen, dirt substrate. 60 moss patches with swaying stems, glowing tips, side-growing variants. 24 code-fragment fireflies that wander, pulse, and flash code snippets (30 Rust/functional-style fragments). Floating dust/pollen particles. Tap to release luminous spores. Mouse stirs fireflies and triggers code reveals. Scanline holographic overlay. Megrim + IBM Plex Mono typography, deep black/green bioluminescent aesthetic.

## features
- Glass jar with holographic sheen and cork lid
- 60 glowing moss patches with swaying stems and tip glow
- Side-growing moss variants
- 24 code-fragment fireflies with pulsing glow
- 30 code snippets (Rust/functional style) that flash near fireflies
- Tap to release luminous spores
- Mouse proximity stirs fireflies and triggers code reveals
- Floating dust/pollen particles
- Bumpy dirt substrate
- Holographic scanline overlay
- Touch support
