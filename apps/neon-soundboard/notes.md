# Neon Soundboard

## log
- 2026-01-15: Initial creation
  - 3 synthesized sounds: GUH, FAH, OOF
  - Web Audio API synthesized vocals (no external files)
  - Neon glowing buttons (red/orange, cyan/blue, purple/pink)
  - Keyboard shortcuts (G, F, O keys)
  - Ripple effect on click
  - Button glow animation when playing
  - Bangers font for comic impact
  - Mobile touch support

## features
- GUH: Low grunt sound (red/orange button)
- FAH: Breathy F sound (cyan/blue button)
- OOF: Impact grunt with thump (purple/pink button)
- All sounds synthesized with Web Audio API
- Keyboard shortcuts: G, F, O
- Visual feedback with glow and ripple effects
- Mobile responsive

## controls
- Click/tap buttons to play sounds
- Press G, F, O keys on keyboard
- Touch friendly for mobile
