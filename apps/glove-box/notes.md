# Glove Box

16x16 neon-grid pixel sandbox where the captain can unleash forbidden power safely.

## log
- 2026-01-25: Initial creation
  - Default 16x16 canvas (forbidden power resolution)
  - NES-inspired 56-color palette
  - Drawing tools: pen, eraser, fill bucket, eyedropper
  - Shape tools: line, rectangle, circle (with preview)
  - Brush sizes: 1px, 2px, 4px
  - Canvas sizes: 16x16, 32x32, 64x64, 128x128
  - Undo/redo with 50-state history
  - Export to PNG at 1:1 pixel ratio
  - Mobile responsive with touch support
  - Keyboard shortcuts (Ctrl+Z, Ctrl+Shift+Z, Ctrl+S)
  - Scanline CRT effect overlay
  - Neon pink/cyan aesthetic

## features
- Neon-grid canvas with visible lines
- Bresenham line algorithm for smooth drawing
- Midpoint circle algorithm for circles
- Flood fill with stack-based algorithm
- Real-time shape preview (semi-transparent)
- Custom color picker
- "Forbidden Power" tagline

## design
- Press Start 2P font (Google Fonts)
- Dark background with neon accents
