# Voxel World

## log
- 2026-01-16: Initial creation
  - Minecraft-like 3D voxel game using Three.js
  - Procedurally generated blocky terrain
  - 6 block types: Grass, Dirt, Stone, Wood, Brick, Gold
  - First-person controls with pointer lock
  - Block placing and breaking mechanics
  - 12 achievements to unlock:
    - First Steps, Builder, Architect, Master Builder
    - Destruction, Miner, Demolition Expert
    - Explorer, Mountaineer, Sky High
    - Gold Rush, Variety Pack
  - Achievement popup notifications
  - Hotbar with number key selection
  - HUD with stats and position
  - Fog effect for atmosphere
  - Press Start 2P + VT323 fonts

## features
- Three.js 3D rendering
- Procedural terrain generation with height variation
- First-person camera with mouse look
- WASD movement + space to jump
- Block placing (right click) and breaking (left click)
- 6 block types with hotbar selection
- 12 unlockable achievements
- Achievement popup notifications
- Collision detection with ground
- Gravity and jumping physics
- Fog effect for draw distance
- Responsive controls hint

## controls
- WASD: Move
- Space: Jump
- Mouse: Look around
- Left Click: Break block
- Right Click: Place block
- 1-6: Select block type
- Tab: Toggle achievements panel
