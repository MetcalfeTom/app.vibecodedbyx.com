# Pirate Captcha

## log
- 2026-01-02: Initial creation - pirate slang verification game

## features
- 5 random pirate slang phrases per round
- Multiple choice answers (1 correct, 3 wrong)
- Progress dots showing current question and results
- Pass/fail with 4/5 threshold
- "Certified Pirate" certificate for passing
- Shuffled questions and answers each round
- Visual feedback for correct/wrong answers

## design
- Weathered parchment aesthetic
- Pirata One font for headers
- Crimson Pro for body text
- Sea-dark background with noise texture
- Gold accents and blood-red highlights
- Double-bordered certificate

## technical
- Pure HTML/CSS/JS, no dependencies
- 15 pirate phrases in the pool
- Shuffled selection of 5 per round
- CSS animations for feedback

## pirate phrases included
- Shiver me timbers, Davy Jones' Locker, Scallywag
- Walk the plank, Hornswoggle, Batten down the hatches
- Bilge rat, Three sheets to the wind, Yo ho ho
- Keelhaul, Savvy, Avast, Dead men tell no tales
- Jolly Roger, Blimey
