// BOMB DEFUSAL MANUAL v3.1

Each bomb has a serial number, indicators (lit or unlit), and 7 modules to defuse. 3 strikes = detonation. Scroll down. Read everything.

MODULE 1: WIRES

Cut the correct wires based on how many wires are present and their colors.

3 wires:
> If no red wires, cut the 2nd wire.
> Otherwise, if the last wire is white, cut the last wire.
> Otherwise, if there are more than one blue wire, cut the last blue wire.
> Otherwise, cut the last wire.
4 wires:
> If there are more than one red wire AND the last digit of the serial is odd, cut the last red wire.
> Otherwise, if the last wire is yellow AND there are no red wires, cut the 1st wire.
> Otherwise, if there is exactly one blue wire, cut the 1st wire.
> Otherwise, if there are more than one yellow wire, cut the last wire.
> Otherwise, cut the 2nd wire.
5 wires:
> If the last wire is black AND the serial number contains a vowel, cut the 4th wire.
> Otherwise, if there is exactly one red wire AND more than one yellow wire, cut the 1st wire.
> Otherwise, if there are no black wires, cut the 2nd wire.
> Otherwise, cut the 1st wire.
6 wires:
> If there are no yellow wires AND the serial number contains a vowel, cut the 3rd wire.
> Otherwise, if there is exactly one yellow wire AND more than one white wire, cut the 4th wire.
> Otherwise, if there are no red wires, cut the last wire.
> Otherwise, cut the 4th wire.

Wire positions are numbered left-to-right starting at 1.

MODULE 2: THE BUTTON

The button has a color and a label. Follow these rules:

> If the button is blue and the label says "Abort", hold the button.
> If there are more than 1 lit indicators and the label says "Detonate", press and release immediately.
> If the button is white and indicator CAR is lit, hold the button.
> If there are more than 2 lit indicators and indicator FRK is lit, press and release immediately.
> If the button is yellow, hold the button.
> If the button is red and the label says "Hold", press and release immediately.
> Otherwise, hold the button.

If you need to HOLD: A colored strip will light up. Release when the countdown timer shows the matching digit:

> Blue strip → release when timer has a 4
> Yellow strip → release when timer has a 5
> Green strip → release when timer has a 1
> Any other color → release when timer has a 1

MODULE 3: KEYPADS

Press the 4 symbols in the correct order. Find the column below that contains all 4 symbols, then press them top-to-bottom as they appear in that column.

Only one column will contain all 4 of your symbols. The order matters!

MODULE 4: SIMON SIGNALS

Colored lights flash in a sequence. Repeat using translated colors based on the serial number.

If serial has a vowel:
> Red → Blue | Blue → Red | Green → Yellow | Yellow → Green
If serial has NO vowel:
> Red → Red | Blue → Yellow | Green → Green | Yellow → Blue

Complete 3 stages. Each stage adds one more flash.

MODULE 6: ROTARY DIAL

Rotate the dial to each target number shown, then lock it in. Complete 3 positions to defuse.

> The dial shows numbers 0-9 around the ring.
> Drag to rotate. The red marker at top indicates the selected number.
> A target number is shown — rotate to it and press LOCK IN.
> Tolerance: must be within ±5° of the exact position.
> Wrong lock-in = strike. 3 correct locks = defused.

The starting position is randomized each time. Careful — the numbers go clockwise.

MODULE 7: COLOR SEQUENCE

Press the colored buttons in a specific order across 4 rounds. The correct button depends on the round number and the color shown last round.

Round 1:
> If red is in position 2, press position 2.
> If blue is in position 1, press position 1.
> Otherwise, press position 4.
Round 2:
> If last answer was red, press the blue button.
> If there are two yellow buttons, press position 1.
> Otherwise, press the same position as Round 1.
Round 3:
> If last answer was blue, press position 3.
> If red is in position 1, press the green button.
> Otherwise, press position 4.
Round 4:
> If yellow is not present, press position 2.
> If there is exactly one red and last answer was green, press position 1.
> Otherwise, press the same position as Round 2.

Positions are numbered 1-4 left-to-right. "Press the blue button" means find whichever position has blue.
