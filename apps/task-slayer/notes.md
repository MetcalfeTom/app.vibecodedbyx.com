# task-slayer

## log
- 2026-04-27: Created. **Gamified to-do list where every task is a boss you slay for XP**. Dark fantasy RPG aesthetic, Cinzel display + Press Start 2P pixel stats + Fraunces italic flavor + IBM Plex Mono body. **HUD strip**: 78px gold radial-gradient circular Lv badge (with embossed inset dashed ring + ink shadow), XP bar (gold gradient w/ animated diagonal stripe shimmer + 2px ink border + inset shadow), rank title from `RANKS[level-1]` (Wandering Squire → Eternal Reaper, 10 tiers), 3 stat blocks: SLAIN crimson / ALIVE violet / STRIKES gold. **Summon panel**: ⚔ title + task name input (80 char cap) + 4 difficulty pills (Trivial green / Standard cyan / Heroic gold / Legendary violet, each with active border-glow color) + crimson Cinzel SUMMON button with 3px ink drop-shadow. **Boss cards** in `repeat(auto-fill, minmax(280px, 1fr))` grid. Each card: 88×88 pixelated sprite canvas + name (Cinzel 17px) + difficulty tag (Press Start 2P 7px in tier color) + crimson HP bar + HP/XP readout + ⚔ Strike / ✓ instant-slay / ✕ banish row. Cards have left tier-color stripe + radial crimson glow + 6px ink offset shadow. Legendary cards add violet glow halo. **Procedural sprites** (16×16 pixelated): FNV-1a hash of `name+id` seeds mulberry32 PRNG → 1 of 5 silhouettes (blob/imp/lich/beast/abomination), per-shape symmetric body-width function over y rows, optional horns + legs + void tendrils. Shading pass: cells with no fill above → highlight color (5), cells with no fill left → light (4), bottom row → shadow (2). 4-neighbor outline pass marks transparent-adjacent cells as outline (1). Eyes drawn at row 4–5 (white sclera color 6 + pupil 7), mouth either solid or fang accent (color 8). Per-difficulty palette: forest (green/cream), frost (navy/cyan), flame (rust/amber), void (purple/magenta + pink eyes). All rendered to `<canvas width=16 height=16>` with `image-rendering: pixelated` upscaled to 88px. **Strike mechanic**: `baseDmg = 8 + level*2`, multiplied by `0.7 + Math.random()*0.6` for variance, 12% crit chance for `*1.8`. Floating damage popup (`-N` or `CRIT! -N` in red) animates -66px over 0.95s with `dmgPop` keyframe (scale 0.6→1.18→0.95). Sprite shake animation ±3px x via Web Animations API. **Strike SFX**: sawtooth thwack 880*power → 160Hz exp ramp through 0.18s + bandpass-filtered noise burst 2200Hz Q=1.6. **HP scaling** per difficulty: Trivial 30/+18XP, Standard 80/+55XP, Heroic 200/+180XP, Legendary 480/+520XP, all multiplied by `1 + (level-1)*0.06` so endgame stays challenging. **Slay flow**: 22 colored gibs (per-difficulty palette colors 3–5) burst from card center via WAAPI translate+rotate over 900–1300ms. Card itself runs `bossDie` keyframe (scale 1→1.08→0.9→0.4 + rotate + hue-rotate + opacity 0 + blur, 1s). **Death SFX**: sine boom 180→40Hz over 0.55s + 4-note descending triangle arp [880, 660, 440, 220Hz @ 70ms intervals]. **XP rain**: on slay, spawn `min(20, 4+xp/30)` Press Start 2P "+NXP" text orbs at boss position, each with 40ms stagger animates via WAAPI to the level badge position with 0.15→1.1→0.6 scale curve over 900ms. **XP table**: `xpForLevel(lv) = floor(100 * 1.42^(lv-1))` — L1→100, L2→142, L3→202, L10→2,876, L20→81k. While-loop unwinds multi-level-ups from huge XP awards. **Level up** triggers fullscreen radial-gradient gold/crimson flash overlay + Cinzel 9vw "LEVEL UP" + LV.N + Fraunces italic rank title; `lvlFlash` 1.6s opacity keyframe + `lvlText` 0.4→1.1→1 scale + rotate keyframe. **Level up SFX**: 5-note major arpeggio C5/E5/G5/C6/E6 in triangle+sine octave layer @ 90ms intervals. **Filter pills**: ALL/TRIVIAL/STANDARD/HEROIC/LEGENDARY at top of arena, sets `display:none` on non-matching cards. **CULL ALL**: confirm → banish all (no XP). **Persistence**: full state in `localStorage['task-slayer-v1']` `{bosses[], level, xp, totalXp, slain, strikes}`. **Seed**: new player gets 2 starter bosses ("Open the email Maria sent" trivial + "Refactor that gnarly function" standard). Mobile breakpoint @680px stacks hero panel + summon row vertically. Pollinations OG.

## issues
- **HP scaling stays modest** (`+6%/level`) so high-level players still demolish trivial tasks in 2–3 strikes. Intentional — small tasks should feel small.
- **Audio-gated** — first pointerdown unlocks AudioContext. Mute is implicit (tab muted in browser).
- **Sprite seeded by name+id** — same task name = same id substring (random), so sprites are unique even for duplicate names. Renaming requires deletion + resummon.
- **CULL ALL** uses `confirm()` dialog — slightly old-school but prevents accidental nuke.
- **Banish (✕) gives 0 XP** — only Strike or instant-slay (✓) award. Banish is for misclicks/abandoned tasks; some players might mistake it for delete-with-credit. Tooltip says "no XP".
- **No edit-task** — to rename, banish + resummon. Could add later.
- **gibs animate from center of card** but card may have moved due to grid reflow — visually acceptable since they spawn before reflow.
- **xpForLevel growth (×1.42)** is aggressive — hits L20 at ~125K XP. Tunable.
