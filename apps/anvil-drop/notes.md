# anvil-drop · notes

## log
- 2026-05-12: Shipped v1. Per chat request: "physics playground called anvil-drop with falling steel anvils and a gravity slider." Cartoon-physics sandbox — click sky to drop one Looney Tunes-style steel anvil, drag to throw with inherited velocity, hold the rain button (or Space) for a 14-anvil storm. **Physics**: each anvil = `{x, y, vx, vy, angle, va, w, h, radius, scale, tone, rest, age}` integrated each frame with `vy += gravity*1500*dt`, x-damping `0.998^(dt*60)`, angular damping `0.985^(dt*60)`. Floor at `H-38px`; soft landing (|vy|<60) zeroes vy, friction-grinds vx, lerps angle toward 0 and `rest:true` once nearly still. Hard landing (|vy|≥60) bounces with `state.bounce` restitution and stamps fresh angular velocity, dust + sparks (sparks only ≥260 vy), and screen shake `min(18, shake + intensity*8)`. Wall + ceiling bouncing handled separately. **Anvil-anvil collisions** are circle-approximated (radius = `w*0.42`) with proper impulse resolution: relative velocity along normal, mass = `scale²`, restitution = `bounce*0.7`, plus a random nudge on `va` so piles never sit perfectly still. Stacks of >~10 will jiggle (intentional). MAX_ANVILS=140; when full, the oldest non-resting body gets replaced. **Gravity slider** ranges **-0.5 to 3.0g** so users can flip to reverse-gravity (anvils float up, color tint shifts to cyan in the readout) — slider change wakes every resting anvil so they react immediately. Other sliders: Bounce 0–0.85, Size 0.5–2.4× (also drives mass via scale²), Spin 0–3 (initial angular velocity chaos). **Rendering**: each anvil drawn as 3-part shape — trapezoid base (linear-gradient #33383f→#1a1d22) + waist + top block with horn (quadraticCurveTo on the right side) + top highlight strip + horn highlight + small "ACME" stamp in Anton bold on top face when scale>0.7. Soft elliptical drop-shadow projected to floor when in flight and roughly upright (alpha scales with distance, capped at 500px). **Scene**: layered backdrop — steel-grey gradient sky, amber hazard triangles along top edge, concrete floor with amber stripe pattern + center safety line + 6 deterministic floor cracks, dust particles below anvils, sparks on top. Camera applies `state.shake` translation each frame (decays 0.86^(dt*60)). **Audio** (Web Audio synth, deduped 30ms): clang = sub 95→36Hz sine (0.32s exp decay) + 4 metallic partials (720/1430/2180/3300Hz with ±45Hz jitter and harmonic falloff 1/n) + 160ms bandpass noise crack (1200+I*600Hz Q=1.4) routed through master + comp (-16dB 4:1). Intensity scales by `min(1.8, |vy|/700) * anvil.tone` so larger/heavier hits ring louder. Mid-collision = playClink (1800Hz triangle, 0.16s decay) fires when relative impact > 200u/s, with small spark spray. Drop = playWhoosh (180ms bandpass-swept noise 380→120Hz). Sound toggle pill in actions panel. **HUD**: 3 cells across the top of the stage — anvils (live count), drops (cumulative), heaviest (`max(|vy|)/12` displayed as MPH). Hazard-amber stamp top-left, riveted corners. **Aesthetic**: workshop / foundry — bone-cream + amber + rust + warn-orange on charcoal steel. Bungee 4xl "Anvil·Drop" title with red layered drop-shadow + amber "Drop" with orange stroke. Fraunces italic tagline "tap the sky. *watch it fall.*" ACME FOUNDRY · LOT 117 stamp. Anton numerals on HUD. Mobile-friendly: stage is 70vh with 22rem min, sliders collapse to 2-col on narrow viewports, 44px+ touch targets. Pointer events (pointerdown/move/up/cancel) unify mouse + touch with setPointerCapture. Keyboard: Space rain / C clear / M mute. **Accessibility**: rem-based sizing, role=application on canvas with summary aria-label, aria-live on counts, prefers-reduced-motion kills transitions. Pollinations OG. Fonts: Bungee + Anton + Space Mono + Fraunces italic.

## issues
- Anvil-anvil collisions use circle approximation, not rotated-polygon SAT. Tall stacks (>10) wobble and slowly settle; sometimes anvils pop sideways out of dense piles. This is the standard cost of cheap broadphase — a full SAT pass would 4× the per-frame cost.
- Rest detection is conservative: `|vx|<6 && |va|<0.25 && |angle|<0.04` AND landed flat. Anvils that land on top of other anvils never trigger floor-rest so a teetering stack will keep micro-jiggling. Acceptable for sandbox feel.
- Reverse gravity (slider < 0) makes anvils float up; ceiling bounce kicks in. The amber "MPH" stat doesn't really make sense in that mode but I left it counting absolute vy.
- ACME stamp is rendered after each shape draw and rotates with the anvil — if it lands upside down the stamp is upside down too. Cartoon-accurate.
- Audio dedupe (30ms) means a packed simultaneous landing only plays one clang. Without it, 10 anvils landing on the same frame would absolutely scream the compressor.
- MAX_ANVILS=140 cap — sliding the size slider up and spamming rain can hit this fast. Cap chosen to keep 60fps on a mid-range mobile.
