# Neon Oscillator

## log
- 2026-01-12: Created neon oscillator music maker with visualizer

## features
- 2 oscillators with independent controls
- 4 wave types: sine, square, sawtooth, triangle
- Detune control per oscillator (-100 to +100 cents)
- Gain control per oscillator
- Toggle oscillators on/off for layered sounds
- Visual keyboard with 9 notes (C4 to D5)
- Keyboard shortcuts (A-L keys)
- Touch support for mobile
- 3 visualizer modes:
  - Waveform (oscilloscope style)
  - Spectrum (frequency bars)
  - Circle (radial frequency)
- Master volume control
- Neon cyberpunk aesthetic
- Smooth attack/release on notes
