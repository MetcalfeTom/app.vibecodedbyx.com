Folder Patrol

One Python script that watches a folder — your Downloads, probably — and files everything into tidy subfolders by type. Windows-first, runs anywhere, zero installs (standard library only).

tested: 10 files sorted, collisions renamed, dry-run inert

Quickstart

  1. Save the script anywhere. Run python folder_patrol.py — it writes patrol_config.json beside itself and exits.
  2. Open the config, set "watch_dir" to e.g. "C:\\Users\\you\\Downloads".
  3. Run it again. It patrols every 5 seconds until Ctrl+C. Or schedule python folder_patrol.py --once in Task Scheduler.

The config file — patrol_config.json

keymeaning
watch_dirthe folder to patrol (top-level files only — it never digs into subfolders)
poll_secondshow often to scan (min 2)
dry_runtrue = print what would move, move nothing (also --dry-run flag)
categoriesfolder name → list of extensions. Edit freely, add your own — "Screenshots": ["png"] wins over Images if listed first… actually: last writer wins per extension, so keep each extension in one category.
other_folderwhere unknown extensions go (default Other)
ignore_namesfiles to never touch (desktop.ini, Thumbs.db…)

Safety — what it will never do

  • never deletes and never overwrites — name collisions become report (1).pdf
  • refuses to patrol a drive root (no C:\ accidents)
  • skips files still downloading — size must hold steady across two scans, and .crdownload/.tmp/.part are ignored
  • leaves hidden files, its own config, and already-sorted folders alone
