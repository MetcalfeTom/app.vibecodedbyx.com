# Homelab Dashboard

Simulated high-tech server monitoring dashboard with live stats, health rings, and neon indicators.

## log
- 2026-04-12: Initial build. 5 simulated servers (Atlas/Nexus/Vault/Edge/Lab) with drifting CPU/RAM/disk/temp stats. Canvas health rings with neon glow. CPU sparkline history per server. 12 services with up/down status and latency. Network throughput sparklines (in/out). Storage pool bar charts. Live system log with 14 templates and color-coded severity. Alert strip with rotating notifications. Cluster status summary card. Auto-updates every 2s. Chakra Petch + Share Tech Mono typography, dark blue-black with cyan/green/amber/red neon accents.

## features
- 5 simulated servers with realistic stat drift
- Canvas-drawn health rings with glow
- CPU history sparklines per server
- 12 monitored services with latency
- Network throughput dual sparklines
- Storage pool bar charts
- Live system log (14 event templates)
- Alert notification strip
- Cluster status summary (hosts, services, uptime)
- Color-coded health indicators (green/amber/red)
- Responsive grid layout
- Mobile-friendly
