# Neon XP

## log
- 2026-01-12: Created neon Windows XP desktop simulator

## features
- Windows XP style desktop with neon cyberpunk aesthetic
- Draggable windows with minimize/maximize/close
- Start menu with programs and system options
- Taskbar with quick launch, window buttons, system tray, clock
- Desktop icons (double-click to open)
- Multiple apps:
  - My Computer (shows drives)
  - My Documents (file browser)
  - Notepad (working text editor)
  - Paint (working canvas drawing with neon colors)
  - Calculator (working basic calculator)
  - Internet Explorer (mock browser)
  - Media Player (mock player)
  - Games folder
  - Settings panel
  - error.exe (fun error dialog)
  - Recycle Bin
- Window stacking with z-index management
- Focused/unfocused window styling
- Taskbar shows open windows
- Click taskbar items to minimize/restore
