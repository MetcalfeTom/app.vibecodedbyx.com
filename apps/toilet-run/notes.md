# Toilet Run

Sprint through meme obstacles to reach the bathroom! Endless runner with fast music.

## log
- 2026-01-22: Initial creation with bald runner, meme obstacles, procedural music

## features
- Bald running man with panic expression and sweat
- 12 meme obstacles: banana, moyai, skull, doge, sus mailbox, clown, alien, duck, pepe, crying emoji, fire, poop
- Procedural fast EDM music (160 BPM kicks + hi-hats + synth)
- Jump sound effects
- Urgency meter that fills as you run
- Distance counter with bathroom sign showing remaining distance
- Particle effects for dust and death
- Death messages with Gen-Z meme speak
- High score saved to localStorage
- Touch + keyboard controls
- Progressive difficulty (speed increases with distance)

## controls
- SPACE / UP ARROW: Jump
- Click/Tap canvas: Jump
