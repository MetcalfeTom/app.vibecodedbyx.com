## issues
- **`makeFormant` linearRampToValueAtTime cascade** can stack if pads overlap — earlier ramps don't cancel. AudioParam queues serialize so each phrase gets its own oscillator+filter pair (no shared nodes), so collisions are rare.
- **Squeak only fires on wail-tagged pads** (OWW! and POW indirectly via kick); other pads stay grounded — intentional, matches MJ phrasing.
- **Moonwalk slide can be re-triggered before previous finishes** — slide value resets to 1 and direction can flip mid-glide. Looks chaotic but on-brand.
- **Glove sparkle radius scales with `mj.glove`** which decays 0.9/frame — fades quickly enough that subsequent triggers re-snap, no buildup.
- **Stutter LFO** at 18Hz feels right for tee-hee but lower-pitched stutter pads (hee-hee at 14Hz) sound more like staccato — a tradeoff between tee-hee giggle and hee-hee cadence.
