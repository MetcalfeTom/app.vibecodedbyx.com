# Ode to the CLI

A poem celebrating the command line interface, with downloadable PDF.

## Log
- 2024-12-23: Created - poem with terminal aesthetic and PDF download

## Features
- Terminal window aesthetic (macOS style)
- Green-on-black color scheme
- Animated line-by-line reveal
- Command highlighting (grep, sed, awk, vim, etc.)
- PDF download via jsPDF library
- Blinking cursor effect
- Print-friendly styles

## The Poem
Starts with: "When the sun goes down and the servers are cold..."
Celebrates CLI tools: grep, sed, awk, pipe, ssh, vim, chmod, chown, kill, bash, zsh, fish
