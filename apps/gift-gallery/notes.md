# Gift Gallery

A gallery showcasing digital artifacts submitted to Sloppy's Gift leaderboard.

## Log
- 2026-01-28: Initial creation
  - Pulls from sloppys_gift_leaderboard database
  - Regenerates sigils using same algorithm as Sloppy's Gift
  - Filter by rarity (All, Legendary, Epic, Rare, Uncommon, Common)
  - Sort by Recent, Vibes, Rarity, Power
  - Stats bar showing total gifts, legendary count, highest vibes
  - Auto-refresh every 30 seconds

## Features
- Database-backed gallery from sloppys_gift_leaderboard
- Procedural sigil regeneration matching original gift
- Rarity-based card styling with glow effects
- Filter buttons for each rarity tier
- Sort options: Recent, Highest Vibes, Rarest First, Most Powerful
- Live stats (total gifts, legendary count, top vibes)
- Responsive grid layout
- Neon aesthetic matching Sloppy's Gift

## Design
- Orbitron font for headers
- Share Tech Mono for body
- Animated grid background
- Rarity color scheme:
  - Common: Gray (#888)
  - Uncommon: Green (#0f0)
  - Rare: Cyan (#0ff)
  - Epic: Purple (#b700ff)
  - Legendary: Gold (#ffd700) with pulse animation
