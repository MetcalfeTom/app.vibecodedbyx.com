## features
- 15 fuzzy foxes infesting the server
- Custom neon lint roller cursor
- Hold and drag to scrub foxes clean
- Fuzz particles fly off when scrubbing
- Cyan spark effects
- Progress bar showing cleanliness
- Server health decreases over time
- Win by reaching 95% clean
- Lose if time runs out

## design
- Orbitron font

- Server rack aesthetic

## controls
- Mouse: hold and drag to scrub
- Touch: tap and drag (mobile)
- Clear 95% fuzz to win
- 60 second time limit
