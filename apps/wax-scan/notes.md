# Wax Scan

Camera-based fake fruit detector with pseudo-scientific analysis.

## log
- 2026-04-10: Added 3 scan modes — Lab (default blue), Neon (purple/magenta with glow effects), Radioactive (toxic green with geiger flicker). Each mode has unique themed verdicts and SFX. Mode bar below header.
- 2026-04-10: Initial build. Camera access via getUserMedia, center-region pixel sampling (300 samples), shine/saturation/variance metrics for wax detection scoring. Scan ring animation with corner brackets, metric reveal sequence, entertaining verdict comments (REAL/WAX/UNKNOWN). Instrument Serif + DM Mono typography, dark botanical aesthetic with green accents.

## features
- Camera access with front/back toggle
- Center-region pixel sampling (300 random samples)
- Shine index (highlight detection)
- Saturation uniformity analysis
- Surface variance measurement
- Pseudo-scientific wax probability score
- Animated scan ring with corner brackets
- Metric reveal sequence
- Entertaining verdict comments
- Mobile responsive
