# The Harsh Grader

Exam paper grading game — read student answers and stamp PASS or FAIL with your red pen. 60-second time limit.

## log
- 2026-04-11: Initial build. 27 questions across 12 subjects with 4 answer variants each (2 pass, 2 fail). Random paper generation with 2-3 questions per paper. Scoring: +10 correct + streak bonus (up to +10), -5 wrong. 6-tier ranking system. Stamp animation with paper glow feedback. Swipe left/right on mobile, F/P keyboard shortcuts. Ruled paper aesthetic with red margin line, Caveat handwriting + Special Elite typewriter + Crimson Pro serif fonts. Wooden desk CSS texture background.

## features
- 27 questions across 12 academic subjects
- 4 answer variants per question (funny wrong answers + proper correct ones)
- Paper passes only if ALL student answers are correct
- 60-second timed grading session
- Streak system with escalating bonus points
- PASS/FAIL stamp animation with correct/wrong paper glow
- Keyboard shortcuts: F/← = Fail, P/→ = Pass
- Mobile: swipe left = Fail, swipe right = Pass
- 6 ranks: Expelled → Substitute → TA → Associate → Tenured → Legendary
- Ruled paper + red margin + wooden desk aesthetic
