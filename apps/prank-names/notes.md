# Prank Name Generator

## log
- 2025-12-29: Initial creation
  - Colorful animated gradient background
  - 7 name categories: classic, food, professions, nature, sounds, absurd, silly
  - Over 100 prank names in database
  - Click button or press Space/Enter to generate
  - Confetti emoji animation on each generate
  - Pop/boing sound effects
  - History shows last 8 names
  - Click any name to copy to clipboard
  - Toast notification on copy
  - Mobile responsive

## notes
### Name categories:
- Classic: Simpsons-style prank call names
- Food: Food-related puns
- Professions: Job/occupation puns
- Nature: Nature-themed names
- Sounds: Names that sound like phrases
- Absurd: More adult humor names
- Silly: Family-friendly goofy names

### Features:
- Keyboard support (Space, Enter)
- Click-to-copy on displayed name and history
- Confetti uses emoji and colored dots
- Pop and boing sound using Web Audio API
