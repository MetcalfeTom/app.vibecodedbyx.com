# liminal-doom

## log
- 2026-05-01: Created. **Wolfenstein-style raycaster set in the Backrooms — yellow wallpaper walls, flickering fluorescents, damp brown carpet, find the EXIT.** Single-file vanilla JS. **Engine**: classic DDA raycaster, casts 1 ray per ~2 horizontal pixels for perf, draws textured vertical strips. Map is a 32×32 Uint8Array (0=floor, 1=wall, 2=exit-wall) procedurally generated as ~70 random horizontal/vertical wall partitions (2–8 cells each) with ~90 random doorways carved through afterward — feels more like an endless office with cubicles than a maze. Start position randomized in top-left quadrant; exit cell randomized in bottom-right. Outer ring is solid wall. **Wall texture** (64×64 offscreen canvas, generated once): yellow→deep-yellow vertical gradient + 18px-spaced vertical seam lines (dark + cream alternating for paneling), dark baseboard at bottom, 800 grain dots (cream + amber speckle), random dark splotch stains. **Exit texture** is a separate red wallpaper variant with a glowing white-cream EXIT box stenciled mid-wall. **Lighting**: `lightLevel` (0..1) drives ceiling/floor/wall brightness. Base wobble via `0.92 + sin(t*0.012)*0.04 + sin(t*0.078)*0.03`. Every 4–13s a flicker event spawns: 200–500ms of chaotic 0.18..0.83 randomized brightness with 4%/frame fluorescent re-strike `flickerZap()` audio. **Audio** (Web Audio synth): continuous fluorescent hum = 60Hz square + 120Hz sawtooth harmonic through 320Hz lowpass (mains buzz + ballast hum); each step plays a 160ms low-passed (220–320Hz) noise burst (carpet squish); flicker zaps fire 1600–2200Hz square clicks via highpass for fluorescent re-strike; win plays a 4-note ascending arpeggio. Mute toggle bottom-right. **Player**: position + radian dir + 60° FOV. Movement 2.6 u/s, rotation 2.2 rad/s. Head-bob via `sin(walkPhase)*4px` while walking. Collision is per-axis with a 0.18-unit shoulder offset for wall-sliding. Step counter increments every 0.7 world-units traveled, shown in HUD top-left. **Render layers**: ceiling (yellow-orange flat fill, modulated by lightLevel) + horizontal stripe pattern at top to suggest tile grid, floor (linear gradient of brown carpet shades), walls via strip-blit from texture canvas with side-shading (×0.65 for y-walls), distance fog (`max(0, 1 - dist/14)`), and `lightLevel` global multiplier. Atmospheric yellow haze overlay scales with `1 - lightLevel` so flicker-darkened moments get gauzier. Exit cell pulses with red overlay at `0.3 + sin(t*0.005)*0.18` for that warning-sign feel. **HUD**: VT323 step counter + proximity hint ("something pulses…" at 8 units, "EXIT IS CLOSE" at 4 units). 120×120 minimap top-right shows wall layout + player dot with direction indicator. **Title screen**: Press Start 2P "LIMINAL · DOOM" with sodium glow + 4-second flicker animation; italic Special Elite lore line; ENTER button; controls hint with `<kbd>` styled keys. **End screen**: "ESCAPED — N steps" yellow-bordered card with rematch button. **Mobile controls**: touch d-pad (W/A/S/D + strafe) on left + 2-button look pad (←/→) on right, shown only at width ≤760px or coarse pointer. **CRT scanlines + radial vignette** on top of the canvas via fixed-position `::after`/`::before`-style divs at z-index 5/6, mix-blend-multiply for the scanlines so they read as dark stripes rather than washing out the yellow. **Win condition**: walk into the exit cell → ESCAPED screen with step count + win chord. **Controls**: WASD or ↑↓← →move, Q/E strafe, A/D rotate, ESC return to title.

## issues
- The map generation is purely random partition placement → no guaranteed connectivity, but the 90-doorway carve usually produces a reachable exit. Rare failures could leave the exit walled off; could fix with a flood-fill validation + retry.
- No enemy / threat AI yet. Chat asked for "yellow walls and flickering lights" — those are present, but the Backrooms genre usually implies a Smiler/Hound following you. Future iteration.
- Distance fog is purely brightness-based; no texture mip-mapping, so far walls show full-detail texture (slightly aliasing at edges). Acceptable for the small map size.
- Procedural wall texture has decent grain but no actual cracks/peeling — could add per-cell damage variants for visual variety on long playthroughs.
- Mobile look-rotate pad is 2 buttons (left/right). Adding swipe-to-look on the canvas would feel better but requires extra touch handling.
- Footstep audio fires per ~0.7-unit travel, which sounds slightly faster than realistic walking pace. Could tune.
- No saved progress between sessions — each entrance is a fresh map. Intentional for the "infinite Backrooms" feel but could add a "best escape" leaderboard.
