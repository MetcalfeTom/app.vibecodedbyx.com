# Activity Overlay

## log
- 2026-01-19: Initial creation
  - Glowing animated progress bar with shine effect
  - Floating particle system with glow
  - 5 color themes (cyan, magenta, gold, green, purple)
  - Built-in 25-minute focus timer
  - Session counter
  - Adjustable particle density
  - Toggle-able control panel
  - Responsive design

## features
- Animated progress bar with:
  - Gradient fill
  - Glow effects
  - Shine animation
  - Glass highlight
- Floating particles with:
  - Pulsing size/opacity
  - Glow halos
  - Screen wrapping
  - Adjustable count (10-150)
- Focus timer:
  - Start/pause/reset
  - 25-minute goal (pomodoro style)
  - Auto session counting
  - Progress auto-updates
- 5 color themes:
  - Cyan (default)
  - Magenta
  - Gold
  - Green
  - Purple
- Hideable controls for clean overlay

## design
- Orbitron font (futuristic)
- Dark gradient background
- CSS custom properties for theming
- Shimmer text animation

## controls
- Theme dots: change color scheme
- Progress slider: manual progress control
- Particle slider: adjust particle density
- Start/Pause: toggle timer
- Reset: clear timer and progress
- Toggle Controls: hide/show control panel
