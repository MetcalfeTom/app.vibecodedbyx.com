# CRT Terminal

Retro CRT monitor simulator with fake hacker commands and green phosphor aesthetic.

## log
- 2026-04-16: Initial build. Full CRT monitor with scanlines, curvature, flicker, and glow effects. 20+ commands including ls/cd/cat with fake filesystem, hack sequence, matrix rain, network scan, traceroute, neofetch, cowsay, fortune, SSH, ping, decrypt. Command history with arrow keys. Boot sequence animation. VT323 + Share Tech Mono typography, green-on-black phosphor aesthetic.

## features
- CRT monitor frame with bezel, scanlines, curvature vignette, flicker
- Fake filesystem with directories and humorous file contents
- 20+ commands: help, ls, cd, cat, pwd, whoami, date, uname, top, ping, ssh, hack, matrix, scan, decrypt, traceroute, neofetch, cowsay, fortune, clear, reboot, exit, sudo, rm, echo, yes
- Animated hack sequence with progress steps
- Matrix rain mode (auto-stops after 30 lines)
- Network vulnerability scanner with fake CVEs
- Boot and reboot sequences
- Command history (arrow up/down)
- Mobile-friendly responsive layout
- Touch support
