# unicorn-reef

## log
- 2026-04-29: Created. **Digital aquarium with neon jellyfish + rainbow unicorn seahorses + Nemo the neon clownfish hiding in coral.** Single full-viewport canvas. Background is a 5-stop linear gradient `#040818 → #0c1738 → #1a2960 → #2c3aa0 → #4d52c8` (deep abyss → shallow blue) plus 2 radial glows (top white sun-bloom, bottom violet warmth) + drifting sparkle dust via dual radial-dot CSS background-image with `sparkleDrift` 90s linear keyframe. **God-ray light shafts**: 6 trapezoidal `lighter`-blend gradients rooted at top, fanning ~1.4× wider at the floor, with `sin(time*0.1)` lateral drift; rgb shifts amber→cool blue from top to floor for sun-through-water feel. **Bubbles**: 50-cap pool, spawn from bottom with random `r`/`vy`/`sway`, sin-wobble path, `rgba(220,240,255)` rim+fill, removed past top. **Seafloor**: violet sandy mound (sin contour) at `y = H-60`. **Kelp**: `floor(W/90)+4` strands, 8–18 segments, sea-green/teal hue 140–240°, sway via `sin(time*sway + phase + u*3) * (4 + u*14)` per-segment, glowing tip dot at top. **Jellyfish (`Jelly` class)**: bell pulse via `sin(pulsePhase)`, bell width grows 1.05→1.23 / height shrinks 0.78→0.62 on contraction; outer halo radial-gradient + bell radial-gradient (3-stop hue-shifted) + 3 concentric rim ribbons + frill wavy under-edge (frillCount 8–13) + 6–10 tentacles with `sin(time*1.6 + u*6) * (10 + u*14)` sway + 3 frillier oral arms; thrusts up `60 px/s` per peak pulse, settles back at `+6 px/s²`, wraps screen edges. **Rainbow unicorn seahorse (`Seahorse` class)**: 12-point spline along an "S"-curve (`sin(u*π*1.4) * 22`) with a tail-curl rotation of `π*1.3*tailCurl` blended over the last 30%; each segment stroked with HSL `hue = (hueOffset + u*320) % 360` for a flowing rainbow body, dark base-stroke underneath for definition, light shimmer pass on top, and per-segment perpendicular dark "bony plates" for that signature seahorse armor. **Mane**: 7 wavy rainbow strands at neck/back, `lighter` blend, `sin(time*2.4 + ph + u*4) * 8` sway. **Dorsal fin**: 12-segment fluttering rainbow fan at u=0.35 along the back. **Head**: pastel rose ellipse + horse-snout quadratic + sparkle eye (white sclera + dark pupil + tiny highlight) + 3 ink eyelashes. **Unicorn horn**: amber→pink linear-gradient triangular cone with `lighter` blend, 5 spiral grooves (alternating curve direction) + tip sparkle that pulses with `sin(time*6)`. **Nemo the neon clownfish**: behavioral state machine (`hide`/`peek`/`venture`/`return`) anchored to the **coral hideout** at `(W*0.78, H-60)` — 14-tendril swaying anemone with hue 12° warm-orange, base mound, tip dot glow. While `hide` Nemo's draw is clipped to area outside the anemone column so only stripes+tail fragments poke through tendrils. While `peek` he's offset `(-22, -6)` from home. `venture` sends him into a small loop `(-80–140, -30–70)` from home; `return` eases back. Body is 18×11 ellipse with an orange radial-gradient (`#ffd166→#ff8a3d→#e54e12`), three white stripes (head/mid/tail) clipped to body ellipse, black ink outline, tail wag via `sin(time*12)*4`, top fin + pec fin flutter, eye with highlight. Outer additive halo 0.45→0.18→0 amber-orange. **Italic Cormorant Garamond name label "Nemo"** fades in (`showName` lerp at 3/sec) only when ventured/returning, with amber `shadowBlur:10` glow above his head. **Click anywhere** spawns a creature at click point (55% seahorse / 45% jelly weight), 10–14 sparkle twinkles in matching hue, plus a chime triad (triangle root + 1.5× and 2× harmonics, 1.4s exp decay). **Toolbar** glassy pill bottom-center with `+ jelly` / `+ seahorse` / `tide pool` (reset). **Audio** muted by default, `♪` toggle bottom-left enables 4-voice drone chord (55/82.4/110/164.8 Hz sine+triangle through 520Hz lowpass with per-voice 0.04–0.10Hz LFO detune ±4–8¢), 2s fade-in. Title "unicorn reef" in Pacifico cursive with magenta→gold→cyan→violet rainbow gradient text + 24px drop-shadow glow. Tagline "— a kingdom of glow and curl —" in Cormorant Garamond italic. HUD top-left: JELLIES + SEAHORSES counts in Major Mono Display. Mobile: tagline + kbd hint hidden, toolbar buttons compact. Pollinations OG.

## issues
- `Jelly.draw` rebuilds gradients every frame — fine at 12 jellies but could be cached if pool grows past 30.
- Seahorse spline is 12 points so the rainbow stroke shows discrete segments at large scales. If a future scale slider is added, increase `N` to 24+ for a smoother gradient.
- Nemo's `hide` clip rect is rectangular cutouts (top half + left + right of coral column) — works because the anemone column is narrow. If coral moves, update clip math.
- Coral position is fixed at `W*0.78` regardless of viewport. On portrait mobile this might overlap the toolbar — visually OK because the toolbar has backdrop-blur.
- Audio drone start is gated on user gesture (browser policy). Mute toggle starts it; first creature-spawn click also unlocks audioCtx but doesn't start drone.
- `bubbles` use `vy` in px/s but the wobble in `b.x + wob` uses absolute position — wobble is screen-relative, not bubble-relative. Looks correct for ambient drift.
- Drawing order: kelp drawn before bubbles which is drawn before sparkles which is drawn before coral+entities. Means bubbles can pass through entities; if you want bubbles in front, move bubble draw after the entity sort.
