# REDACT-O-MATIC

## log
- 2026-01-13: Created GDPR compliant redaction tool for Toby

## features
- Black bar redaction on any text
- Auto-detect patterns:
  - Email addresses
  - Phone numbers
  - ID numbers
  - Dates
  - Common names
- 5 redaction levels:
  - Light (10% random words)
  - Medium (25% random words)
  - Heavy (50% random words)
  - Paranoid (75% random words)
  - Toby Mode (100% everything)
- "Redact Everything" nuclear option
- Click redacted bars to peek underneath
- Yellow highlight on reveal
- Copy redacted text button
- Stats: words processed, items redacted, GDPR score
- Classified document watermark
- Special Elite typewriter font
- Official-looking stamp design
- Notice specifically for Toby
- Mobile responsive
