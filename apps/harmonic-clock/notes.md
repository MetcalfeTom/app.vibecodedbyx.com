## issues
- The piano shows ONE octave (C-B), but Circle of Fifths visits the same 12 notes in a different sequence ‚Äî so the piano is a "where on the keyboard does this pitch live" reference, not a "sequence preview". If chat wants the piano to show the next-fifth interval relationship visually, we'd need to draw an arc/line between the two highlighted keys.
- All sounds use the same single octave (C4-B4). Each "fifth up" in the circle would technically go up an octave every 5-6 steps if we tracked real frequency progression, but we reset to the same octave for consistency with the piano roll display.
- Enharmonic labels (C‚ôØ/D‚ô≠, F‚ôØ/G‚ô≠, etc.) only shown in the small text below the main note label, and only on the clock ‚Äî piano keys just use sharp notation.
- The hint arrow uses a single arc segment (always sweeps the short way around) which means going from F (position 11) back to C (position 0) draws a tiny arc rather than a clockwise rotation. Visually still correct since it's the adjacent slice.
