# Megascience Roast Machine

Giant button roast machine that dishes out randomized insults about cursed AI experiments.

## log
- 2026-04-17: Added wiki-style red links below roast box. 12 fake articles (Sentient Gradient Descent, Recursive Hamster Theory, Haunted Loss Function, etc.) that open error modal with unique HTTP error codes and absurd explanations. 3-5 random links shown, refresh on each roast. Error sound on open, Escape/click-outside to close.
- 2026-04-17: Initial build. Giant neon circle button with pulse ring and rotating dashes. 45 roasts across 7 categories: Neural Nets, Robotics, Cursed Labs, Data Science, AI Gone Wrong, Megascience, Existential. Typewriter effect for roast text. 8-bit WebAudio sounds (descending square wave + noise burst, chord stab every 10th roast). Screen flash on roast. Roast counter, category badges with per-category neon colors, scrolling history (max 20). Keyboard support (Space/Enter). Press Start 2P + Silkscreen typography, neon pink/cyan on dark aesthetic with scanline overlay.

## features
- Giant circular neon roast button with pulse animation
- 45 unique roasts across 7 categories
- Typewriter text reveal effect
- 8-bit WebAudio: descending "bwaaah" + noise burst per roast
- Chord stab every 10th roast (milestone sound)
- Screen flash on button press
- Roast counter
- Color-coded category badges
- Scrolling roast history (last 20)
- Keyboard: Space/Enter to roast
- No duplicate consecutive roasts
