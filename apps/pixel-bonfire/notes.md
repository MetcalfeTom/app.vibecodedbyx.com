# Pixel Bonfire

## log
- 2026-01-17: Initial creation
  - Standalone ambient pixel art bonfire
  - Doom-style fire algorithm for realistic flames
  - 12 color gradient from white-hot to ember
  - Rising ember/spark particles with flickering
  - Smoke particles floating upward
  - Pixel wood logs with glowing embers
  - Stone ring around campfire
  - Twinkling star background
  - Ambient glow effect on page
  - Crackling fire sound (toggleable)
  - Adjustable fire size (small/med/big)
  - Press Start 2P pixel font
  - Dark cozy atmosphere

## features
- Realistic fire simulation algorithm
  - Fire spreads upward with random decay
  - White-hot core to dark ember color gradient
  - Natural flickering effect
- Pixel art aesthetic (4px chunky pixels)
- Ember particles:
  - Rise from fire
  - Random drift and flicker
  - Fade out over time
  - Yellow and orange variants
- Smoke particles:
  - Rise slowly
  - Drift with sine wave motion
  - Gradually fade and grow
- Environment:
  - Three crossed logs
  - Stone ring
  - Glowing embers on wood
  - Twinkling star field
  - Ambient orange glow
- Audio:
  - Procedural crackling sound
  - Lowpass filtered noise
  - Toggle on/off
- Controls:
  - Sound toggle
  - Fire size (small/medium/big)
