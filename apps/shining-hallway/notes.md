# shining-hallway

## log
- 2026-04-27: Created. **Cinematic blood hallway** — pixelated 1-point perspective Overlook hotel hallway with 3D fluid blood physics. Three stacked canvases at low internal res (`IRES.w = clamp(280, 420, W/4)`) scaled up via CSS `image-rendering: pixelated` for chunky retro look. **Layers**: `#bg` (baked scene + elevator redraw on tick), `#blood` (pool + airborne drops), `#fx` (dust + REDRUM glow). **Perspective**: single-point `VP={x:0.5, y:0.5}`, `FOCAL=0.6`, world coords (x∈[-1..1], y=0 floor / 1 ceiling, z=0 camera / 8 vanishing). `project(wx,wy,wz)` returns `{sx, sy, persp = 1/(z*FOCAL+1)}`. Inverse `screenToWorldFloor(sx,sy)` solves for z then x so clicks splash blood at the correct world floor coordinates. **Carpet**: row-by-row scanline `ImageData` painter — for each y between horizon and bottom, compute z, persp, halfW, draw carpet pattern in world (u,v=z) coords. Pattern is iconic Overlook hex-ish: center diamond `#b6491c` (carpet-orange), ring `#6c2310` (rust), outer `#2a160d` (brown), with cream `#d4a86e` corner dots. Depth darkening `1 - rowProg*0.85`. **Walls**: column scanline — for each x compute z where wall surface intersects, then per-y draw cream wallpaper with deeper-tinted vertical stripes (4-state by `floor(z*6.5) % 4`), wood wainscoting bottom 33%, dado rail at 64%, ceiling cornice top 5%, deterministic damage spots from `(x*131+y*191+stripeBand*53) & 1023`. Depth tint `1 - z*0.16`. **Doors**: 5 z-positions on each wall, drawn as trapezoidal frame + body + 2 panels (lerp-projected) + brass knob + "237"/"217" room number on close ones. **Elevator** at z=7.5 — gold/brass frame, two doors that slide with `state.elevatorOpen` (eases toward target). When open, reveals black void + red gradient inner glow. **Sconces**: 4 z-positions × 2 sides, radial amber halo `rgba(245,177,90,0.55)→rgba(40,12,4,0)` + bulb + arm. **Ceiling beams**: 5 z-positions converging toward VP. **Blood physics 3D**: `state.drips[]` particles `{x,y,z,vx,vy,vz,life,sz,hue,kind}`, `MAX_DROPS=1400`. Gravity `vy -= 1.6*dt` (y is up, gravity reduces it). Air drag `vx,vz *= 0.985^(dt*60)`. Bounce off walls at `|x|=0.96*HALL_HALF_W` with -0.4 restitution + 0.7-amount **wall splat** entry. Floor-hit at y≤0 → `addPoolDot` + 25% small upward splash + remove. **Pool**: 64×64 `Float32Array poolGrid` indexed by (u,v=z/8), each cell 0..1 fluid amount; rendered as projected quads from far-to-near with red fill — bright `#d80f1a` if cell>0.85 else `#8c080c`, alpha = cell*1.4. Lazy diffusion every 3rd frame redistributes fluid to neighbors when local gradient > 0.03 (gentle viscous flow). **Wall splats** on `state.splats[]`: 14–22s life, drip extends downward over time (`drip += dt*0.05`). **Elevator wave**: opens via Space or button, target eases to 1.0, while open spawns 8–12 drips/frame from elevator opening (z=7.0, y=0.05–0.15) flying toward camera (vz=-1.6 to -2.8 - surge*1.2, vy=0.4–1.1). 4% chance per frame of **chunk burst** (24 drops + scream stinger) at surge>0.6. **Audio**: 4-voice drone chord (41.2/55/82.4/110Hz sine+triangle through 320Hz LP Q=1.4 with 0.04–0.16Hz detune LFOs ±2.5¢, fades in over 2.2s) + 3.2s convolver reverb. `playElevatorOpen` = 880Hz sine bell + 56→28Hz sub thud + delayed bandpass-noise creak (1800→420Hz Q=6). `playSplash` = throttled bandpass noise 800–1200Hz. `screamStinger` = sawtooth 380→120Hz through `tanh(*4)*0.7` waveshaper for distortion. `playClick` = 660→110Hz sine blip. **FX**: 14 dust motes drifting cyan, red radial glow at VP scaled by elevator open, REDRUM/ROOM 237/COME PLAY/FOREVER text flickers above doors when open and `sin(t*1.7)>0.92`. **Aesthetic**: void `#07050a` body bg with vertical blood-tint vignette, multi-layer overlays (grain `radial-gradient` repeating + `scanlines` repeating + flicker `mix-blend: screen`). IM Fell English italic body, Cormorant Garamond italic taglines, VT323 HUD ("THE HALL · 237" + "LITRES SPILLED"), Special Elite intro quote ("great party, isn't it?"). Toolbar buttons: alert-red OPEN ELEVATOR + ♪ ON/OFF + ↺ RINSE. Intro card with horror quote requires click (browser audio policy). Pollinations OG painted as cinematic Kubrick-style hallway prompt.

## issues
- **Internal res is intentionally low** (~320×200) for chunky pixel feel. shadowBlur is NOT used — instead per-particle highlight fillRect for "wet" specular. Scales perfectly to any viewport via CSS pixelation.
- **Carpet/walls are baked once per resize**, not redrawn each frame — perf is dominated by blood pool rendering (64×64 cells projected as quads) + airborne drops. On older mobile, pool can be expensive at full saturation; could downsample poolGrid to 48×48.
- **Elevator door state lives outside the bake** — uses `bgSnapshot` ImageData taken pre-elevator, then `putImageData + drawElevator + drawSconces` re-overlays each frame so doors animate without rebuilding the whole scene.
- **Web Audio requires user gesture** — handled via intro overlay's STEP INSIDE button which both unlocks AudioContext and starts the drone.
- **Pool diffusion is approximate** — runs every 3rd frame with simple neighbor-flow when gradient exists. Real fluid sim would be FLIP/PIC; this is a vibey approximation.
- **screenToWorldFloor returns null for clicks above horizon** — clicks on walls/elevator/ceiling don't splash on floor. Could add wall-clicks → wall splats; left out for simplicity.
- **REDRUM text is intentional camp** — the Shining is referenced loud and proud (not a copyright concern in a parody/horror tribute). Pulls from a 4-message rotation when elevator >50% open.
