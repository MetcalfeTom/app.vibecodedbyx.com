# Cyber Notes

## log
- 2026-01-03: Initial creation - cyberpunk glassmorphism note-taking app

## features
- Create, edit, delete notes
- LocalStorage persistence
- Search/filter notes
- Glassmorphism card design
- Neon gradient accents
- Animated grid background
- Edit modal with blur backdrop
- Note timestamps and IDs
- Keyboard shortcuts (Ctrl+Enter to save, Esc to close)
- Responsive grid layout

## design
- Cyberpunk dark theme
- Glassmorphism cards with backdrop blur
- Neon cyan/magenta gradients
- Share Tech Mono for code-like text
- Rajdhani for UI elements
- Subtle grid pattern background
- Hover effects with neon glow
- Side accent bar on card hover

## technical
- localStorage for persistence
- Escape HTML to prevent XSS
- Sort by last updated
- Real-time search filtering
- Unique ID generation
- Modal overlay with transitions

## keyboard shortcuts
- Ctrl+Enter: Save note (new or edit)
- Escape: Close modal
