# Cyber Evidence Board

## log
- 2026-01-13: Created neon cyber-forensics evidence board with draggable logs and red connection lines

## features
- Draggable evidence cards with neon styling
- Red animated connection lines (dashed, animated)
- Evidence types with color coding:
  - System Log (green)
  - IP Address (orange)
  - Suspect (magenta)
  - File (cyan)
  - Alert (red)
- Connection mode to link evidence cards
- Add new evidence with type, title, details
- Delete individual evidence cards
- Clear all connection lines
- Reset to initial state
- Pre-populated cyber incident scenario
- Pin styling on cards
- State persists in localStorage
- Keyboard shortcuts (Escape to cancel)
- Dark cyber-forensics aesthetic
- Mobile touch support for dragging

## initial scenario
- Intrusion detection alert
- Internal and external IP addresses
- Auth log with brute force attempts
- Insider threat suspect
- Encrypted data exfiltration file
- Firewall log blocking darkweb
- Data exfiltration alert
