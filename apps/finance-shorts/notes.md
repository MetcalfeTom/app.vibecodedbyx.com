## features
- 9:16 vertical format for TikTok/Reels/Shorts
- Chart types:
  - Line chart with glow effect
  - Bar chart with gradients
  - Candlestick chart (green/red)
  - Pie chart with percentages
  - Area chart with gradient fill
  - Stats card with key metrics
- 6 neon color options
- AI script generator creates 3-frame storyboards
- Multi-frame timeline with thumbnails
- Add/delete frames
- Frame navigation
- Real-time preview updates
