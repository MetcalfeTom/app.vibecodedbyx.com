# ABC Mysteries

Find hidden letters buried in visual chaos. Sharp eyes only.

## log
- 2026-04-13: Initial build. Procedural scene generation with shapes, noise letters, symbol clutter. 8-14 target letters per level (scaling), 5 color palettes rotating by level, letters blend into scene with matching colors/opacity. Hit detection with radius check, found letters highlight green. Hint system (flashing circle, -10 pts), time bonus on completion, miss penalty (-3 pts). Level progression increases clutter, reduces letter visibility. Anybody + DM Mono typography, dark purple/gold aesthetic.

## features
- Procedural scene with 80+ shapes, noise letters, symbol clutter
- 8-14 hidden target letters per level (increases with level)
- Letters camouflaged with scene colors, varying opacity, rotation, fonts
- 10 different fonts to disguise letters
- 5 color palettes (purple, green, red, blue, yellow) rotating by level
- Hit/miss feedback with expanding ring animations
- Hint system with flashing glow indicator
- Time-based scoring with level multiplier
- Letter bar showing targets and found letters
- Touch support
