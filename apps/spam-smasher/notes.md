# Spam Smasher

Whack-a-bot game — spam-bots pop up from server nodes, smash them with a neon hammer before they flood your inbox.

## log
- 2026-04-11: Initial build. 12-hole grid (4×3) of server nodes, bots pop up with timed lifespans. 4 bot types: basic (red, 1hp), shield (orange, 2hp), speed (pink, fast), mega (green, 3hp). Bots have robot faces with tracking eyes, antenna, jagged teeth, spam message bubbles. Neon hammer cursor with green glow, swing animation, impact shockwave ring. Combo system with score multiplier up to ×8. Missed bots add to spam meter (10 = game over). Gear/screw debris on kills, floating score text, spam envelope particles on escape. Circuit board background. Spam inbox bar at bottom. Difficulty ramps with score (faster spawns, harder bots, double spawns). Rubik Glitch + IBM Plex Mono typography, green/dark hacker aesthetic.

## features
- 4 bot types: basic, shield (2hp), speed (fast timer), mega (3hp + glow)
- 12 server node holes in 4×3 grid
- Neon hammer with swing animation and impact shockwave
- Bot eyes track the hammer
- Spam message bubbles on bots
- Combo system with ×8 max multiplier
- Spam meter: 10 escaped bots = game over
- Gear debris and floating score text on kills
- Escaped bots become flying spam envelopes
- Timer bar under each bot
- Circuit board background
- Difficulty scales with score
