# Neon Crawl

First-person dungeon crawler with raycasting engine, D20 combat, and neon wireframe monsters.

## log
- 2026-04-12: Initial build. Raycasting engine with fisheye correction, wall edge highlighting for wireframe look. Procedural dungeon generation (rooms + corridors, size scales with floor). 5 monster types (Wraith/Skeleton/Demon/Lich/Slime) each with unique wireframe art, D20 combat stats (AC, atk bonus, damage dice). Full D20 system: roll d20 + modifier vs AC, critical hits on nat 20 (double damage dice). Player leveling via XP (atk/ac/dmg scale). Health potions, stairs to next floor. Minimap with player direction. Combat log with color-coded messages. Death screen with stats. Mobile touch controls. Press Start 2P + Share Tech Mono typography, cyan/teal neon wireframe aesthetic.

## features
- Raycasting 3D engine with FOV 60°
- Procedural dungeon (rooms + corridors)
- 5 wireframe monster types with unique shapes
- D20 combat: roll + modifier vs AC, crits, damage dice
- XP-based leveling (atk, ac, damage die scale)
- Health potions (d8+2 heal)
- Multi-floor progression (stairs)
- Minimap with monsters/items/player
- Combat log (hit/miss/crit/kill/heal)
- Death screen with respawn
- Mobile touch controls (d-pad + attack)
- Click/spacebar to attack
- WASD movement, Q/E strafe
