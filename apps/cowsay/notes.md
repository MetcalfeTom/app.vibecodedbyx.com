# Cowsay

## log
- 2026-01-16: Initial creation
  - Classic cowsay ASCII art generator
  - 12 different animals to choose from
  - Real-time preview as you type
  - Copy to clipboard functionality
  - Retro green terminal aesthetic
  - VT323 + Fira Code fonts
  - Proper word wrapping for speech bubbles

## features
- 12 animals: Cow, Tux, Dragon, Cat, Bunny, Ghost, Duck, Elephant, Stegosaurus, Owl, Frog, Shark
- Auto-generates as you type
- Word wrapping for long messages
- Speech bubble with proper borders
- Copy to clipboard button
- Terminal window styling
- Scanline effect
- Flickering neon title
- Mobile responsive
