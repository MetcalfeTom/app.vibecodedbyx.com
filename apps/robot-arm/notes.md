# Robot Arm Sandbox

## log
- 2026-01-12: Created physics sandbox with 3-segment robot arm

## features
- 3-segment articulated robot arm
- Keyboard controls for each joint:
  - Q/A: Rotate base
  - W/S: Segment 1 angle
  - E/D: Segment 2 angle
  - R/F: Claw angle
  - SPACE: Grab/Release
- Physics simulation with gravity, friction, bounce
- Neon crates in random colors (cyan, magenta, yellow, green)
- Neon T-Rex dinosaur with glowing eyes
- Throw velocity based on arm movement
- Objects collide with ground and walls
- Spawn buttons for crates and dinos
- Reset scene button
- Neon cyberpunk aesthetic
- Trail effect for motion blur
