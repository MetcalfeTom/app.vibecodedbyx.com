# Robo Café

Serve motor oil and synthetic blends to impatient robot customers before they overheat.

## log
- 2026-04-12: Initial build. Canvas-based robot coffee shop game. 5 drink types (Motor Oil, Synth Blend, Coolant, Rust Juice, Gear Lube). 5 robot body types with expressive faces (happy/impatient/angry moods). Drag drinks from counter to customers. Order bubbles show what each robot wants. Patience bar counts down — robots overheat if not served in time. Wrong order = strike. 3 strikes = game over. Wave system with increasing difficulty (more simultaneous customers, faster patience decay). Tips earned based on speed. Café interior with metal wall panels, menu board, counter, floor tiles. WebAudio SFX (serve, wrong, ding, boom, tip). Bungee Shade + Oxanium typography, dark orange industrial aesthetic.

## features
- 5 drink types with distinct colors and descriptions
- 5 robot body types with antenna, chest lights, arm animations
- 3 mood states with facial expression changes
- Drag & drop drink serving mechanic
- Order bubbles with mini cup preview
- Patience bars with color-coded urgency
- Wave progression system (more customers, less patience)
- Tip calculation based on serve speed
- 3-strike system (wrong orders + timeouts)
- Café interior with wall panels, menu board, counter
- Steam wisps on drink cups
- Particle effects on serve/overheat
- Float text feedback
- WebAudio SFX
- Touch + mouse support
- Vignette overlay
