# The Logic Grove

Bioluminescent digital terrarium where chat messages grow into glowing fractal plants.

## log
- 2026-01-25: Initial creation
  - Canvas-based fractal plant generation
  - L-system inspired branching algorithm
  - Message text seeds plant properties (color, complexity, height)
  - Bioluminescent glow effects with pulsing
  - Particle system for floating spores
  - Gentle sway animation
  - 8 color palette (cyan, magenta, violet, lime, orange, teal, purple, green)
  - Chat input to plant messages
  - Random seed button for inspiration

## features
- Message-seeded plant generation
  - Message length affects plant height and complexity
  - Character values determine base color
  - Unique random seed per message
- Bioluminescent effects
  - Canvas shadowBlur for glow
  - Pulsing light intensity
  - Floating particle spores
- Terrarium aesthetic
  - Dark background with subtle gradients
  - Glass dome light reflection
  - Soil/ground layer at bottom
- Smooth growth animation
- Mobile responsive

## design
- Cormorant Garamond serif for titles
- JetBrains Mono for technical elements
- Dark teal/black background
- Neon glow palette
- Minimalist controls
