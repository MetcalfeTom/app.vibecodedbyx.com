# Cube Escape

## log
- 2026-01-18: Initial creation
  - Glowing neon cube player
  - 5 productivity tool obstacles
  - Level progression system
  - Mobile touch controls

## features
- Control a glowing gradient cube
- Dodge falling productivity tools:
  - Spreadsheets (Google Sheets style)
  - Trello cards
  - Excel files
  - Notion pages
  - Jira tickets
- Score increases for each dodge
- Level progression (faster spawns)
- Particle explosion on death

### Visual Effects:
- Gradient cube (cyan to magenta)
- Player trail effect
- Pulsing glow on all elements
- Grid background with motion
- Scanline overlay
- Particle explosions

### Controls:
- Arrow keys / A, D
- Mouse movement
- Touch drag (mobile)
- Mobile left/right buttons

## design
- Orbitron + Share Tech Mono fonts
- Neon cyan/magenta color scheme
- Dark background with grid
- Retro arcade aesthetic
