# aggressive-horticulture · notes

## log
- 2026-05-14: v1 — pixel-art gardening game where **microphone loudness drives plant growth**. Chat request: "create a pixelated gardening game called aggressive horticulture where yelling into the microphone makes plants grow." **Pixel-art pipeline**: logical canvas 160×100 px upscaled by an integer scale to fit the viewport (`Math.floor(viewport / 160)`, clamped ≥ 1), `image-rendering: pixelated` + `image-rendering: crisp-edges`, `ctx.imageSmoothingEnabled = false`, `ctx.setTransform(scale, ...)` so every drawn pixel is a single visible block. **Plant sprites**: 5 plant types × 5 growth stages, each defined as a 16×16 ASCII-art grid (`'.'` = transparent, letters map to a 17-entry palette) parsed into `{x, y, c}` blits and rendered with per-pixel `fillRect`. Stages 0-4: seed-in-soil → sprout → bush → unripe fruit → ripe with glow. The 5 plants: **tomato** (gnarled bush + red fruit), **sunflower** (tall stalk, finally bursting into a yellow halo with purple seed-center), **cactus** (segmented column with two arms, blooms with a tiny red flower at the top), **mushroom** (white stem grows a red+white-spot cap), **carrot** (orange root pokes up and unfurls a sprig of green leaves). Each plant has a unique score value (mushroom 4 fastest growth · sunflower 8 slowest+biggest payout) and growth duration in ms. **Microphone**: `navigator.mediaDevices.getUserMedia({ audio: { echoCancellation:false, autoGainControl:false, noiseSuppression:false } })` so the raw signal hits the AnalyserNode unaltered. AnalyserNode `fftSize: 1024`, `getByteTimeDomainData` → compute per-sample `(v-128)/128`, sum-of-squares, sqrt = RMS. Map `[0.04 silent floor, 0.5 yell ceiling] → [0, 1]` then exponential-smooth `level = level·0.55 + lvl·0.45`. **Yell → grow**: each frame `grow += (yellMultiplier / growDuration) · dt` where `yellMultiplier = max(0.04 baseline trickle, level·1.6 + (level > 0.5 ? (level-0.5)·1.2 : 0))` — so silence still nudges plants forward, normal volume grows them at a few seconds per stage, peak yelling grows them ~6× faster with a non-linear bonus past 50%. **Mic UI** top-right: parchment-card volume meter with a black-bordered gradient bar (green → amber → orange → shout-red) filling proportional to level, plus a tiny "ON" pulse-dot (green) when the mic is live; turns red on permission denial. **Fallback**: if mic is blocked, the button changes to "🚫 NO MIC · SPACE = YELL" and the spacebar simulates a 0.7 yell with slight wobble — so the game's still playable. **Game loop**: 8 plots in a horizontal strip at logical y=68. Click an empty plot to plant the currently-selected seed (5 chunky palette buttons at the bottom, each showing a 24-px-rendered stage-4 thumb as a `<img>` and a `+N` cost line). When a plant hits `grow >= 1` it flips to `ripe = true`, gets a colored glow halo behind its sprite, and a "RIPE!" toast pops above. Click a ripe plant to harvest: score += plant value, plot empties, "+5/+8/etc" toast. localStorage best-score persists across visits. **HUD** top-left: SCORE / HARVESTED / BEST in Silkscreen on parchment. **Atmosphere**: sky gradient (cyan → pale-cyan), pixel sun at top-right with 8 rotating yellow ray-pixels that intensify with mic level, three drifting pixel clouds at different speeds, grass strip just above the plot row. When the user is screaming (level > 0.7) a translucent red overlay tints the whole screen for an unhinged feedback effect. Slight bobbing wobble dots appear on near-mature plants when yelled at hard. **Aesthetic**: Press Start 2P chunky title with a red text-shadow halo + ink-black drop shadow (looks like an old NES game splash), Silkscreen for all UI text, VT323 loaded but currently unused (held in reserve for future numerals). Palette: NES-y greens, sky cyans, soil browns, NES-red. **WCAG basics**: canvas `role="application"` with descriptive aria-label, palette buttons `aria-pressed`, mic button text changes verbosely on state, HUD has `role="status"` + `aria-live="polite"`, ≥44 px button targets, `prefers-reduced-motion` disables the mic-pulse animation. Pollinations OG flux seed 42420.

## issues
- `getUserMedia` requires HTTPS or localhost. On sloppy.live it works because the site is served over HTTPS. If a user blocks the prompt, they get the spacebar fallback but lose any sense of "yelling actually does something" — could add a synthesized voice-fake (sloppy_ai shouts) as another fallback.
- The mic threshold (RMS 0.04 floor, 0.5 ceiling) is calibrated for a typical laptop mic in a quiet room. Loud environments (cafés, streams with music in the room) may sit around 0.2 RMS at baseline and the plants will grow continuously without yelling. Could add an adaptive baseline (running mean of the last 2 s) to subtract ambient.
- The 8-plot strip caps the simultaneous-plant count. Enough for the bit but could be tiled vertically into a fuller garden grid for more strategic play.
