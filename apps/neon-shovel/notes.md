# neon-shovel

## log
- 2026-04-27: Created. **3D synthwave dig simulator** built on **three.js r128** (CDN: jsdelivr). Click anywhere on the cyan grid plane and a shovel mesh swings down, dirt flies, and ~22% of digs uncover a glowing **neon loot box** that you click to claim. **Scene**: PerspectiveCamera (60° FOV) at `(0, 9, 14)` looking at `(0, 0.4, 2)` with subtle `sin(t)*0.6` x-drift. `THREE.Fog(0x150930, 18, 90)` + black void clear color. **Lights**: AmbientLight 0x6b3aff intensity 0.5 + magenta DirectionalLight from upper-back + cyan PointLight at right. **Backdrop**: pre-rendered 256×256 canvas with magenta radial sun gradient + 5 horizontal scanline cuts (ink-black 2-6px stripes), used as `THREE.Sprite` 34u wide at z=-45. Stylized purple ShapeGeometry mountain silhouette (50-point sin-wave + cos terrain heights) at z=-38. **Ground**: 80×80 PlaneGeometry rotated horizontal with a **1024×1024 canvas-painted neon-grid texture** — radial purple→indigo gradient base, 64px cyan major-grid lines (rgba(94,224,255,0.55) 2px stroke), 3×3 cyan glow dots at every intersection. CanvasTexture with `RepeatWrapping` 2×2. **Shovel** is a Group: 4.2u brown CylinderGeometry handle + 0.22u Torus pink-emissive grip + 0.9×1.2 cyan-emissive BoxGeometry blade with `EdgesGeometry` cyan wireframe overlay + small Cone tip below blade. Sits at idle position `(2.4, 0.5, 4)` with `rotation.z = 0.2` slight tilt; idle-bobs via `sin(now*0.002)*0.08`. **Dig animation** state machine across `swing 0..1` (4.2 units/sec): 0–0.4 raise+tilt over target (lerp x/z, sine y arc to 2.6u), 0.4–0.7 strike down (y → 0, x rot → 0.4 forward), 0.7–1.0 return to home. At swing > 0.5 the `performDig()` callback fires once per dig (guarded by `struck` flag): spawns particles + hole + maybe loot. **Dirt particles**: 22 small BoxGeometry (0.08–0.22u) brown emissive with random outward velocity + 4–9u upward + per-axis rotation; gravity -18u/s²; ground bounce at y<0.05 with 0.32 restitution. Life 1.4–2.1s with last 0.4s opacity fade. **Holes**: black CircleGeometry disc + magenta RingGeometry outline pinned 0.005u above ground; FIFO 28-cap. **Loot boxes**: BoxGeometry 0.65–0.95u (size by rarity) with `transparent:true opacity:0.78` + emissive rarity color + LineSegments wireframe edges + additive-blending Sprite halo (128×128 canvas radial gradient). Float bob `0.7 + sin(age*2)*0.16`, rotate y `0.9rad/sec`, x rotate sine wave. Legendary boxes pulse emissive intensity `0.6 + sin(age*4)*0.35`. **4 rarities** weighted: COMMON cyan 60% +10c, RARE magenta 25% +50c, EPIC gold 12% +250c, LEGENDARY violet 3% +1500c. **Open box** = spawn 22/40/70 reward particles (count by rarity) at box position with cone-shaped upward burst, +coin total + per-rarity counter + DOM toast (`.toast.{rarityId}` class with rarity color, `toastFloat` keyframe 1.6s or 2.4s legendary). **Audio** (Web Audio): **dig** = 120→40Hz sine sub thump 0.25s + lowpass-filtered (900Hz) noise scatter 0.28s. **rumble** (when box appears) = 3-note rising bell triad (1×, 1.25×, 1.5×) at 7-segment-stagger, base Hz scaled by rarity (legendary 196Hz lowest = ominous). **open** = 4–6-note ascending arpeggio (1×/1.25×/1.5×/2×/2.5×/3×, count by rarity) in triangle+sine octave layer. **legendary stinger** = 4-voice sawtooth chord through 1800Hz LP after 0.4s delay, 1.2s exp decay. **Raycast input**: `pointerdown` → first try active loot boxes (intersect against box meshes), then ground plane. Hit ground → `digAt(point)`, hit box → `openBox(box)`. **HUD**: Monoton "NEON SHOVEL" title with magenta+cyan chromatic-split top-center; Press Start 2P stats panel top-left (DIGS/BOXES/COINS); rarity scoreboard top-right (4 pills with color swatches + counts, legendary has stronger violet glow); VT323 hint bar bottom-center; CTRL bar bottom-left (RESET FIELD + ♪ ON/OFF). **Intro overlay**: Monoton title + "Click anywhere to start digging" + violet "⛏ START DIGGING" button (also unlocks AudioContext). **Persistence** in `localStorage['neon-shovel-v1']` `{digs, boxes, coins, common, rare, epic, legendary, muted}`. **Aesthetic**: deep void `#04020c`, magenta `#ff5ea1`, cyan `#5ee0ff`, violet `#c47bff`, gold `#ffd25b`. Monoton + Audiowide + Press Start 2P + VT323 + Space Mono. Pollinations OG.

## issues
- **three.js r128** is CDN'd (~600KB cached after first load). If CDN is blocked, scene won't render. Could pin a specific cdnjs mirror as fallback.
- **Fog + transparency interplay** — loot box transparent material can show seams through fog edges at certain angles. Acceptable, looks slightly "ghostly".
- **No GPU pixel-density cap on mobile** beyond `min(devicePixelRatio, 2)` — Pixel 7+ class phones still hit 2x. Could lower to 1.5 if perf complaints.
- **Particle gravity** (-18) and ground bounce (0.32 restitution) are tuned by eye. Reward particles linger because their `life` only halves on slow bounce — gives rainbow scatter feel.
- **Camera is fixed angle** — no orbit controls. Intentional: keeps the dig site always framed cleanly. Could add subtle drag-to-pan later.
- **Box has no inventory after claim** — they're claimed once and gone. Could persist a "vault" of icons later.
- **Click-through on intro** — pointerdown listener checks `intro.style.display === 'none' || intro.classList.contains('gone')` so clicks before "START DIGGING" don't trigger digs.
- **Holes don't actually deform geometry** — they're flat CircleGeometry decals. Real terrain deformation would require a vertex-shader pass.
- **Reset confirms via window.confirm()** — old-school but prevents accidents.
