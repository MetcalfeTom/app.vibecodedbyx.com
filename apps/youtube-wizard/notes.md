## features
- Channel name generator
  - Topic/niche input
  - Optional creator name
  - 4 style options: Professional, Fun, Minimal, Epic
  - Pattern-based generation
  - Random variations with adjectives/suffixes
- Channel preview card
  - Avatar initial
  - Handle preview
  - Subscriber/video placeholders
- Save favorite names to localStorage
- Copy to clipboard
- Stats tracking (generated, saved, copied)
- Toast notifications
- Script Generator (AI-powered)
  - 4 styles: Tutorial, Review, Vlog, Top 10
  - Structured output: Hook, Intro, Main Points, CTA, Outro
  - Copy script feature
  - Uses Pollinations text API
- AI Thumbnail Previewer
  - 4 styles: Vibrant, Minimal, Dramatic, Retro
  - Generates 4 variations per request
  - Download thumbnails
  - Uses Pollinations image API

## design
