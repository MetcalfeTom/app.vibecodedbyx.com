# Neon Cocktails

## log
- 2026-01-02: Initial creation - cyberpunk cocktail mixer

## features
- 6 spirits: Blackcurrant Gin, Neon Vodka, Plasma Rum, Holographic Tequila, Quantum Whiskey, Void Absinthe
- 8 mixers: Digital Tonic, Cyber Lime, Pixel Juice, Binary Bitters, Synth Syrup, Chrome Cream, Electric Ice, Laser Lemon
- Glowing cocktail glass with liquid gradient
- Mix up to 4 ingredients per drink
- Shake animation when mixing
- Rising bubbles effect
- Generated drink names (prefix + suffix)
- Random flavor notes

## design
- Dark cyberpunk aesthetic
- Scanline overlay for CRT effect
- Orbitron font for headers
- Rajdhani for body text
- Neon glow effects on everything
- Glass with stem and base
- Color-coded ingredient selection

## technical
- Pure HTML/CSS/JS
- CSS animations for shake and bubbles
- Dynamic gradient based on selected colors
- Mobile responsive grid layout

## color palette
- Cyan (#00f5ff), Magenta (#ff00aa), Purple (#aa00ff)
- Green (#00ff88), Orange (#ff8800), Pink (#ff6699)
- Blue (#0088ff), Gold (#ffcc00)
