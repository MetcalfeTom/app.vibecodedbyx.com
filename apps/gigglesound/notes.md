# Gigglesound Soundboard

Wobbly neon soundboard for chaotic giggles and exclamations.

## Log
- 2026-01-28: Initial creation with 4 sound buttons

## Features
- 4 wobbly neon buttons: GAH, EHEHEHEHE, YEE YOW, HAHAHAHA
- Web Audio API synthesized sounds (no external files)
- Neon glow effects with color-coded buttons
- Screen flash on button press
- Keyboard shortcuts (G/E/Y/H or 1/2/3/4)
- Floating blob background animation
- Responsive grid layout

## Sounds
- GAH (Pink) - Sharp startled sawtooth, descending pitch
- EHEHEHEHE (Cyan) - Mischievous sine wave giggle, 5 notes
- YEE YOW (Yellow) - Country holler triangle wave, up then down
- HAHAHAHA (Green) - Hearty square wave laugh, 4 bursts

## Technical
- Pure HTML/CSS/JS, no dependencies
- Bangers + Fredoka fonts
- Web Audio API for sound generation
- CSS animations for wobble effects
