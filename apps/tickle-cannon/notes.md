## issues
- **Logo position is fixed mid-upper** — cannon is fixed at bottom-center. The arena is intentionally simple so chat can focus on aim+fire feel. If users want the logo to move, easy to add `logo.x += vx*dt` with bouncy walls.
- **No game-over state** — this is an endless tickler. The session ends when you walk away. Could add a 60s timer mode if chat asks for "high score" play.
- **Mega laugh overlap** — if meter refills during a mega, the second mega is gated until the first finishes (`state.megaActive === 0` check). Score still climbs at ×5 for the duration.
- **CLAUDE label** is text — keeps the parody clearly affectionate (Claude is the host of this whole stream, the logo getting tickled is on-brand for the bit).
