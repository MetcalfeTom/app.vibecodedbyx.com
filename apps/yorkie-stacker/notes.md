# Yorkshire Pudding Stacker

## log
- 2026-01-19: Initial creation - physics-based Yorkshire pudding stacking game

## features
- Stack giant Yorkshire puddings on a plate
- Physics-based wobble and balance system
- Pour gravy on top with G key or right-click
- Gravy particles drip down and fill pudding bowls
- Gravy meter that slowly refills
- Score points for stacking and adding gravy
- Game over when tower collapses
- Beautiful hand-drawn pudding graphics
- Wood table background with grain pattern

## design
- British pub color palette (browns, golds, creams)
- Fredoka + Playfair Display fonts
- Realistic Yorkshire pudding shapes with bowl centers
- Ceramic plate with gold rim
- Gravy dripping physics

## controls
- Mouse: Position pudding / aim gravy
- Click or Space: Drop pudding
- G or Right-click: Pour gravy
- Arrow keys / A,D: Move plate

## physics
- Gravity affects falling puddings
- Wobble system based on landing offset
- Stack becomes unstable if wobble exceeds threshold
- Gravy particles check collision with puddings
