# Neon Fox Trap

## log
- 2026-01-11: Created neon fox trap game with digital pest catching

## features
- Click/tap to place glowing fox traps
- 5 types of digital pests (glitch, pixel, bug, virus, worm)
- Each pest type has unique appearance, speed, and point value
- Combo system (up to x10) for catching pests quickly
- Particles and sound effects on catch
- Pests escape after time limit (lose combo)
- Neon fox mascot in corner with wagging tail
- Cyberpunk grid background
- Mobile and desktop friendly
