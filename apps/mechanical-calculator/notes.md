# Mechanical Calculator

## log
- 2026-01-05: Initial creation - vintage calculator with division by zero chaos

## features
- Vintage 1960s mechanical calculator aesthetic
- Animated spinning gears during calculations
- Mechanical number dials
- Division by zero triggers infinite spinning mode
- Emergency stop button to halt the madness
- Smoke particles during chaos
- Screen shake and explosion flashes
- Survival time counter
- Keyboard support
- Nixie tube style display

## division by zero behavior
1. Display shows "ERR: ÷0"
2. All gears spin at maximum speed
3. Number dials spin infinitely
4. Calculator shakes violently
5. Random warning messages cycle
6. Smoke particles emit
7. Occasional explosion flashes
8. Emergency stop button appears
9. Pressing stop shows survival time

## design
- Special Elite + Orbitron fonts
- Bakelite/brass color scheme
- 3D button effects with shadows
- Gear SVG animations
- Status indicator light
- Warning label at bottom

## controls
- Number buttons (0-9)
- Decimal point
- Operations (+, -, ×, /)
- Equals for calculation
- Clear (C) - also stops spinning
- Backspace (←)

## keyboard shortcuts
- 0-9: Numbers
- +, -, *, /: Operations
- Enter or =: Calculate
- Escape or C: Clear
- Backspace: Delete last digit

## technical
- Pure HTML/CSS/JavaScript
- SVG gear graphics
- CSS animations for all effects
- No external dependencies
