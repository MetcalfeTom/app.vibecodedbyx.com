# Neon Crash

## log
- 2026-01-04: Initial creation - 2D platformer inspired by Crash Bandicoot

## features
- Side-scrolling 2D platformer
- Player with run and jump mechanics
- Regular crates - jump over for points (+10 each)
- NITRO boxes - pulsing red, instant death on touch
- Stacked crate obstacles
- Floating platforms with crates
- 3 lives system
- Progressive difficulty (speed increases over time)
- High score saved to localStorage
- Distance tracker (meters)
- Neon visual aesthetic with glow effects
- Explosion effects on nitro collision
- Keyboard controls (WASD/Arrows + Space)
- Mobile touch controls

## controls
- Arrow keys / WASD: Move left/right
- Space / Up / W: Jump
- Mobile: On-screen buttons

## design
- Press Start 2P + Orbitron fonts
- Cyan player with glow
- Orange/gold crates
- Red pulsing nitro boxes
- Magenta ground line
- Purple platforms
- Dark space background with stars

## technical
- Canvas-based rendering
- Gravity/physics simulation
- Platform collision detection
- Procedural obstacle generation
- Explosion particle effects
