# Sock Hole

Catch falling socks in your laundry basket before they hit the floor.

## log
- 2026-04-10: Initial build. Canvas arcade game with trapezoid basket, 7 sock types + sneaker (bad item) + 3 rare bonus items (crown/star/diamond). Wave system with escalating speed and spawn rate. Combo system (catch streaks for multipliers up to 5x). Wobble physics on some socks. Particle effects on catch/miss. Lives as sock emojis in HUD. Wave progress bar. Mouse/touch/keyboard controls. Best score in localStorage. Lilita One + Azeret Mono typography, deep purple/pink laundry aesthetic.

## features
- Canvas-based arcade gameplay
- 7 colored sock types + sneaker hazard
- 3 rare bonus items (crown, star, diamond)
- Wave system with progressive difficulty
- Combo multiplier (up to 5x)
- Wobble physics on falling socks
- Trapezoid basket with grid detail
- Particle burst effects
- WebAudio SFX
- Mouse/touch/arrow key controls
- Best score persistence (localStorage)
- Funny game over messages

## issues
- No leaderboard (localStorage only)
- Basket width doesn't scale much on very wide screens
