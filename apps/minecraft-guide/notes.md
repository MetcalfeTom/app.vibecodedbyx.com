# Learn Minecraft - Interactive Guide

## Log
- Initial creation: Interactive tutorial app for learning Minecraft
- Features:
  - 10 comprehensive lessons covering basics to advanced topics
  - Progress tracking with visual progress bar
  - Interactive quiz for each lesson
  - Minecraft-themed UI with blocky design
  - Topics covered:
    1. Getting Started
    2. Crafting Tools
    3. Building Shelter
    4. Combat Basics
    5. Food & Farming
    6. Mining & Ores
    7. The Nether
    8. Enchanting
    9. The End & Ender Dragon
    10. Advanced Building & Redstone
  - LocalStorage to save progress
  - Modal system for lesson details
  - Completion tracking with checkmarks
