# Flip Clock

Retro split-flap clock with 3D CSS flip animations and synthesized sound effects.

## log
- 2026-04-16: Initial build. 6 flip tiles (HH:MM:SS) with 3D CSS rotateX animations, top-half flips down then bottom-half flips up. WebAudio "oof" sound on minute change (low thud + mechanical click + noise burst), subtle tick on second change. 12-hour format with AM/PM. Date display. Pulsing separator dots. Ambient floating particles. Sound toggle button. Bebas Neue + Red Hat Mono typography, deep navy/red accent aesthetic.

## features
- 3D CSS flip animation with perspective
- Split-flap style: top half flips down, bottom half reveals up
- Shine highlight during flip, shadow on bottom tile
- WebAudio synthesized sounds:
  - "Oof" on minute change (low sine thud + square click + noise burst)
  - Soft tick on second change
- 12-hour format with AM/PM indicator
- Date display (day, month, date, year)
- Pulsing red separator dots
- Ambient floating particles
- Sound toggle (on/off)
- Mobile responsive (3 breakpoints)
- Touch support for audio unlock
