## features
- Organic liquid blob background animations
- Glassmorphism cards with backdrop-filter blur

- Smooth scroll-triggered reveals
- Film grain texture overlay
- Premium typography (Instrument Sans + Playfair Display)
- Gradient text effects
- Responsive mobile design
- Stats counter section
- Testimonial section
- CTA with gradient background

## design elements
- Dark luxurious color palette (#0a0a0b base)
- Purple/cyan/pink accent gradients
- Subtle glass borders (8% white)
- 32px border radius on large elements
- Staggered animation timing
- Floating ambient blobs

- GSAP 3.12 + ScrollTrigger plugin
- CSS backdrop-filter for glass effect
- CSS custom properties for theming
- SVG noise filter for grain texture
- requestAnimationFrame via GSAP
- Intersection-based scroll triggers
