# Simp Meter

## log
- 2026-01-13: Created simp meter for +poopypee6741 with explosion effect

## features
- Thermometer-style simp meter
- Fills up as you type
- Neon pink/magenta gradient
- 8 simp levels with status text:
  - DORMANT
  - CURIOUS
  - INTERESTED
  - CRUSHING
  - OBSESSED
  - FULL SIMP
  - CRITICAL
  - EXPLOSION
- Critical pulse animation at 85%+
- Explodes at 100% with:
  - Screen shake
  - Pink flash
  - Warning popup
  - 50 emoji particles
  - Explosion sound
- Floating heart background
- Beep sounds increase pitch with level
- Reset button
- Percentage display
- Target name display (+poopypee6741)
- Gradient animated title
- Mobile responsive
