# Fox Font Quest

A word puzzle adventure game where you play as a fox-eared character solving puzzles to progress through themed worlds.

## log
- 2026-04-11: Initial build. CSS-drawn fox character with ears, body, tail, eyes, nose, mouth — reacts to correct/wrong answers (happy squint, sad frown, bounce animation). 6 themed worlds (Whispering Meadow, Amber Thicket, Moonlit Cavern, Crimson Peak, Frozen Archive, The Void Spire). 4 puzzle types: anagram (click scrambled letter tiles), fill-blanks (type the word), definition (4-choice multiple choice), riddle (type answer). 3 difficulty tiers per world (easy 3-letter, medium 5-letter, hard 7-letter). Word pools with clues/definitions. 12 riddles. Level/score/progress persistence via localStorage. Vollkorn + Azeret Mono typography, dark fox-orange/gold/teal aesthetic.

## features
- CSS-only fox character with animated reactions
- 6 themed worlds with unique color palettes
- 4 puzzle types (anagram, fill-blanks, definition, riddle)
- Progressive difficulty (3/5/7-letter words)
- Score system with streak bonus
- localStorage progress persistence
- Mobile-friendly
