Two players. One screen. Broken code.
Race to click the buggy line.

How it works:
• Code appears with a hidden bug on one line
• P1 clicks 🔵 buttons (or keys 1-9)
• P2 clicks 🔴 buttons (or keys Q W E R T Y U I O)
• Wrong guess → bug jumps to a new line + 1.5s cooldown
• First to find it scores a point — first to 5 wins
