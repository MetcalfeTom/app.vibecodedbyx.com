## features
- Skull rock formation with carved features
- Cursed lantern with flickering green flame
- Skull's eyes glow with flame intensity
- Ghost fog creeps in from all directions
- Some fog has skull faces (haunted spirits)
- Spirits fly out when haunted fog is burned
- Cursed flame beam burns away ghost fog
- Fog drains flame power when it reaches beacon
- Survival time counter
- Spirits banished counter
- Global leaderboard (Supabase)
- Sickly green moon in haunted sky
- Dark ocean with eerie waves

## controls
- Mouse movement / Touch drag: Aim cursed beam
- Beam automatically follows cursor

## mechanics
- Ghost fog spawns faster as time progresses
- Beam length tied to flame power
- Fog in beam slowly dissolves with green particles
- Multiple fog clouds near beacon drain power faster
- Game over when cursed flame dies

## design
- Dark haunted atmosphere (#050508 to #0d1015)
- Ghostly green color scheme (#7cfc00, #98fb98)
- Skull rock with glowing eye sockets
- Wispy fog tendrils with pulsing animation
- Sickly yellow-green moon
- Pirata One / IM Fell English SC fonts

## technical
- Canvas 2D rendering
- Angle-based beam collision detection
- Particle systems for flame and spirits
- Smooth beam rotation with lerping
- Fog with random skull faces
