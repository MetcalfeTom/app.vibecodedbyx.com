# Mystic Omen

High-fantasy auto-battler with D&D classes. Choose your fate.

## log
- 2026-04-10: Initial build. Solo hero auto-battler with 6 D&D classes (Fighter, Wizard, Cleric, Rogue, Paladin, Ranger). Each class has unique skill type, stat modifiers, and signature weapon with endless upgrades. 8 monster types scaling by wave, 5 boss monsters every 10 waves (Red Dragon, Arch Lich, Kraken, Pit Fiend, Tarrasque) with unique skills. Enemy HP persists across defeats. CSS animations: fadeSlide cards, fadeScale VS badge, hitPulse on damage, staggered class card entrance. Uncial Antiqua + Cormorant Garamond typography, deep purple/arcane aesthetic. localStorage save/load.

## features
- 6 D&D classes with unique skills and weapons
- Endless weapon forging (damage + speed)
- Level-up system
- 8 monster types with wave scaling
- 5 boss encounters every 10 waves with skills
- Enemy HP persists across defeats
- Smooth CSS entrance animations (fadeSlide, fadeScale, stagger)
- Hit/attack flash animations
- Boss gold border glow
- Battle log with color-coded events
- localStorage persistence
- New Run button
- Mobile responsive
- WebAudio SFX
