# Seahorse

## log
- 2026-01-06: Initial creation - massive seahorse display with bubbles

## features
- Giant SVG seahorse centered on screen
- Floating animation
- Rising bubble effects
- Ocean gradient background
- Glowing cyan seahorse with detailed features

## design
- Custom SVG seahorse with body, snout, eye, dorsal fin, belly ridges, curled tail, crown
- Ocean-inspired color palette (#40c8ff cyan, #30b0e8 darker accents)
- Deep blue gradient background
- Ambient rising bubbles

## technical
- Pure CSS animations
- SVG for crisp scaling at any size
- min(80vw, 80vh) sizing for responsive massive display

## notes
- There is no seahorse emoji in Unicode standard
- Created custom SVG seahorse instead
