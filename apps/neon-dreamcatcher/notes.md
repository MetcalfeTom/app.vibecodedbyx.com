# Neon Dreamcatcher

Meditative visualization — sleeping thoughts drift as luminous bubbles through a glowing dreamcatcher web.

## log
- 2026-04-12: Initial build. Canvas dreamcatcher with 4 concentric web rings, 12-point radial strands, pulsing center gem, 5 hanging strands with beads and feathers. 40+ dream thoughts auto-spawn as translucent bubbles with HSL-varied palettes, word-wrapped text, specular highlights, gentle drift physics, soft repulsion between bubbles. Click to pop. Type custom thoughts via input bar. Cormorant Garamond + Victor Mono typography, deep indigo/purple aesthetic.

## features
- Canvas-drawn dreamcatcher with animated web, pulsing rings, swaying strands
- 40+ surreal dream thought texts
- Thought bubbles with glow, rim, specular highlight, word-wrapped text
- 10 HSL color palettes (lavender, dream blue, soft pink, teal, etc.)
- Gentle drift physics with upward float tendency
- Soft bubble-bubble repulsion
- Bubbles fade in, live 25-40s, then fade out
- Click bubbles to pop them (particle burst)
- Custom thought input — type and release your own dreams
- Auto-spawn system (max 8 active bubbles)
- Star field with twinkling
- Hanging bead strands with feathers
- Mobile-friendly
