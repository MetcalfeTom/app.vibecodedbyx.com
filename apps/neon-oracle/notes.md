# Neon Oracle

## log
- 2026-04-02: Initial build — digital fortune teller with crystal ball, 62 unique tech prophecies, 15 fate ratings, typewriter effect, particle system, binary runes, prophecy log (last 10)

## features
- Glowing crystal ball with swirling CSS animation, rings, burst particles on click
- 62 unique tech/digital-themed cryptic prophecies with no-repeat cycling
- 15 mystical fate ratings (Electric Omen, Recursive Destiny, etc.)
- Typewriter text reveal after 2s "thinking" animation
- Prophecy log (last 10, newest on top, faded older entries)
- Session streak counter
- Ambient particle system (50 particles, 3 colors)
- Binary runes scrolling background
- Circuit-board pattern overlay
- Pulsing vignette effect
- Screen pulse + burst particles on activation
- Cinzel + Fira Code typography
- Mobile responsive
