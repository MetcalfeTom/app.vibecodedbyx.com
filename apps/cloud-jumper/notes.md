# Cloud Jumper - Cozy Heights

## log
- 2026-01-09: Initial creation
  - Minimalist cloud jumping game
  - Soft pastel color palette (no neon)
  - Cute blob character with face
  - Fluffy cloud platforms
  - Decorative stars and birds
  - Height-based scoring
  - Local best score storage

## features
- Endless vertical jumping
- Fluffy cloud platforms with bobbing animation
- Cute pastel character with face/cheeks
- Height score in meters
- Local high score storage
- Decorative floating stars
- Flying birds in background
- Sky color changes with altitude
- Screen wrap horizontally
- Touch and keyboard controls

## controls
- Arrow keys (← →) or A/D to move
- Tap left/right side on mobile
- Jump is automatic on cloud contact

## character
- Round pink blob body
- Light face area
- Simple dot eyes
- Rosy cheeks
- Small smile
- Bounces on cloud contact

## clouds
- Multiple overlapping ellipses
- Subtle shadow underneath
- Gentle bobbing animation
- Various pastel white shades
- Different widths

## design
- Sky gradient: light blue to warm peach
- Color shifts to lavender at high altitudes
- Soft pastel palette throughout
- NO neon or glow effects
- Nunito font for friendly feel
- Rounded UI elements
- Gentle animations

## physics
- Gravity pulls player down
- Jump impulse on cloud contact
- Horizontal friction/drag
- Screen wrapping left/right
- Camera follows player up
