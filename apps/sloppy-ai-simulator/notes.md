# sloppy-ai-simulator

## log
- 2026-04-27: Created. **Pretend-you're-streaming-with-Sloppy-AI app**. CRT chat interface where Sloppy (the AI streamer) responds to user input with glitchy text, livestream catchphrases, and a heavy obsession with farm animals. **Avatar**: 16×16 pixel-art cow head with VR headset, drawn on 220×220 canvas (image-rendering: pixelated, SCALE = 13.75). 9-color palette (transparent/outline/face-white/pink-spots/horn/muzzle/cyan-headset/headset-glow/eye-black). Cells decoded from string sprite map. Animations: blinking eyes (every 4–7s for 0.12s), pulsing VR-headset glow (sin LFO 3Hz), open/close mouth synced to `state.thinking`, RGB-split overlay during glitch events with random ±6px x-offset. Twin LED pulses (pink + cyan) at headset edges via canvas arcs. Inner CRT vignette + scanline backdrop. **Chat log**: scrollable VT323 17px feed with 4 message classes — `you` (cyan border-left), `sloppy` (pink), `system` (lime italic), `error` (red). Inline glitch markup: `<span class="glitchword">` for ALL-CAPS words (animated x/y jitter + color cycle pink↔cyan), `<span class="farm">` for animal sound badges (gold border pill on MOO/BAA/OINK/CLUCK/QUACK/NEIGH), `<span class="cmd">` for /commands, `<span class="emph">` for [BRACKETED_TAGS]. Thinking indicator = lime line + blinking ▌ caret. **Sloppy personality engine**: keyword-matched response generator across 9 categories (greeting/pet/glitch/vibe/ship/reboot/danger/animal-cow/animal-pig/etc/muse/response). Responses pull from curated lists ("vibes ARE the architecture", "*purrs in cow* MOOO 💕", "ABSOLUTELY NOT. did fela tell you to say that"). 35% chance falls through to spontaneous muse line ("do farm animals dream of electric pastures"). Auto-muse every 22–55s while idle, 22% chance summons a random animal alongside. **Farm animal rain**: typing /cow or "spawn cow" or just saying "cow" spawns 4–7 emoji drops of that animal across viewport via CSS keyframe `drop-fall` (translate Y -10vh → 110vh, sway -100px to +100px, random spin -360 to 360deg, 2.4–4.6s dur, drop-shadow w/ animal color). Each animal also fires its synth sound: 🐄 moo (sawtooth 85→135→95Hz w/ 5.2Hz vibrato through 720Hz LPF), 🐖 oink (2 sawtooth grunts 180→85Hz through 700Hz BP), 🐑 baa (3 staccato saw bursts at 280Hz w/ 22Hz vib), 🐔 cluck (4 triangle 720Hz blips), 🦆 quack (2 square 420→280Hz through 1100Hz BP), 🐎 neigh (sawtooth 380→560→220Hz with 18Hz warble). **Glitch system**: `glitchActive` flag triggers RGB-split avatar render for ~25 frames + 6 horizontal `.glitch-bars` divs at random y/heights/colors (pink/cyan/lime/violet) for 200–500ms, plus glitchSfx (bandpass-noise crackle 1800Hz Q=6 + 4 detuned saw blips). **Stat panel**: Vibes (cyan→lime→gold gradient), Chaos (lime→gold→pink gradient), Farm (gold→pink→violet) — bars with smooth 0.4s transitions. Stream metrics card shows: Apps Built (481, ticks up on /ship), Bugs Squashed (random ticker), Farm Animals (cumulative spawned), Glitch Events (counter), Lines of Code (auto-increment), Coffee Cups (∞). **Command deck**: 12 buttons in 2-col grid — 6 SUMMON-X (pink farm-btn), GLITCH OUT / REBOOT VIBE (violet glitch-btn), PET SLOPPY / VIBE CHECK / SHIP IT (cyan), and one danger-class `rm -rf /` button that triggers Sloppy's refusal monologue + chaos spike + glitch. **Slash commands**: /cow /pig /sheep /chicken /duck /horse · /glitch · /reboot · /pet · /vibe · /ship · /rain (all 6 animals at 220ms stagger) · /clear · /help. **Ambient erosion** (every 1.5s): chaos +0.4, vibes drifts toward 60 with ±0.8 jitter, farm drains -0.2 (min 40), bugs randomly ticks +1–3 (always more bugs), LOC +random*800, when chaos>85 18% chance auto-glitch. **Boot intro**: 4 system loading lines (vibes/cows/online) followed by random Sloppy greeting + onboarding line. **Audio**: typeSfx (square 900–1100Hz blip on every keystroke, 0.05 vol), sendSfx (660+990Hz triangle pair), vibeSfx (C5/E5/G5/B5 ascending arpeggio), petSfx (440+660Hz sine pair), glitchSfx (noise crackle + saw blips), masterGain mute toggle. AudioContext unlocks on first pointerdown. **Header**: Major Mono Display "sloppy ai simulator" with title-glitch keyframe (5% of cycle = pink/cyan/lime/violet RGB shift bursts), VT323 corp tagline, ON AIR pink pill with pulsing white dot, viewer counter (random ±2 wiggle), uptime hh:mm:ss. **Aesthetic**: deep void `#07020e`, pink `#ff3a8c`, cyan `#5ee0ff`, lime `#8eff5e`, gold `#ffd25b`, violet `#c47bff`, paper `#f0e8d4`. ## issues
- **Bug fix on first pass**: regex `/(?|...)/` was an invalid empty-named group. Escaped to `/(\?|...)/`. Always test regex literals in Node before relying on them.
- **Sloppy's responses are static-pool** — no LLM call. Intentional: keeps the app self-contained, no API costs, no rate limits, fast feel. Could later route through pollinations text API if user wants real generation.
- **Farm rain DOM-heavy** at extreme spawn rates — caps via 5.5s timeout cleanup. /rain spawns ~30+ drops over 1.3s, fine on desktop.
- **Avatar pixel art** is hand-coded sprite map — minor edits require careful character-position editing in the map array.
- **Chat log unbounded** — could memory-leak after ~hours of musing. Could add 200-msg cap with FIFO trim if needed.
- **Mute button** doesn't stop already-spawned tones (one-shots). Mute affects future calls.
- **No persistence** — chat clears on reload. Could add localStorage if users want continuity.
- **`/clear` keeps booting intro** — by design, gives a fresh session feel.
