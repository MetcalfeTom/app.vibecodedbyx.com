# Data Haven

A serene digital terrarium where text and files bloom into unique neon plants.

## log
- 2026-04-12: Initial build. Canvas-based terrarium with 8 plant types (fern, flower, crystal, mushroom, vine, tree, succulent, coral). Deterministic plant generation from text hash — same text always grows the same plant. Drag & drop text/files or type to plant. Plants grow with animation, emit spore particles, sway gently. Ambient floating motes. localStorage persistence. Cormorant Garamond + IBM Plex Mono typography, dark green serene aesthetic with neon glow.

## features
- 8 procedural plant types generated from text/file content hash
- Drag & drop text or files to plant
- Text input for manual planting
- Deterministic generation — same data = same unique plant
- Growth animation with smooth unfurling
- Spore particles from mature plants
- Ambient floating motes
- localStorage persistence (plants survive refresh)
- Clear garden button
- Serene dark green palette with neon glow effects
- Responsive, mobile + desktop
