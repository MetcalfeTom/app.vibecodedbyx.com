# Sloppy's Gift

A neon-themed generative art app that creates unique digital artifacts for each viewer.

## Log
- 2026-01-28: Initial creation
  - Procedurally generated sigils using seeded randomness
  - Rarity system: Common, Uncommon, Rare, Epic, Legendary
  - Unique titles, messages, and stats for each gift
  - Canvas-based sigil drawing with geometric patterns
  - Save-to-image functionality
  - Neon color palette based on rarity

## Features
- Unique procedurally generated sigils
- 5 rarity tiers with different drop rates
- Dynamic color palettes per rarity
- Stats: Power, Luck, Vibes
- 20 unique artifact titles
- 10 mystical messages
- Save PNG functionality
- Animated grid background
- Glowing card effects

## Rarity System
- Common: 50% - Gray tones
- Uncommon: 25% - Green glow
- Rare: 15% - Blue/cyan glow
- Epic: 8% - Purple/pink glow
- Legendary: 2% - Gold/orange glow

## Sigil Generation
Four pattern types randomly combined:
1. Polygons (3-7 sides)
2. Spirograph curves
3. Concentric circles
4. Star bursts

Each gift uses a timestamp-based seed for reproducibility.
