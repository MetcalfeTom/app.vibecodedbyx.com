# Infinite Abyss

## log
- 2026-01-18: Initial creation
  - Fractal tunnel with 60 layers
  - Code snippets floating into void
  - Three visual modes
  - Performance optimized (no actual recursion)

## features
- Hypnotic endless falling effect
- 60 simultaneous layers creating depth illusion
- No browser crashes - uses iterative layer recycling
- Depth counter tracking infinite descent

### Visual Modes:
- Code: Recursive code snippets orbiting shapes
- Geometric: Fractal patterns within shapes
- Matrix: Green code rain aesthetic

### Code Snippets:
- Recursive function examples
- Lambda calculus (Y combinator)
- Haskell infinite structures
- Assembly loops
- Type theory notation

### Technical:
- Layer recycling (no actual infinite objects)
- Z-depth sorting for proper rendering
- Alpha fading for depth perception
- Radial vortex center effect
- Character particles pulled to center
- Trail effect with partial canvas clearing

### Shapes:
- Hexagons
- Squares
- Triangles
- Circles

## design
- Fira Code monospace font
- HSL color cycling based on depth
- Radial vignette overlay
- Dark void aesthetic
- Subtle title animation

## controls
- Mode buttons: Code / Geometric / Matrix
- Speed: Slow / Normal / Fast / Hyperspeed
- Keyboard: 1, 2, 3 for modes, Space for speed

## performance
- 60 layers max with recycling
- 100 particles max
- requestAnimationFrame loop
- Partial canvas clear for trails
- No recursive functions (iterative simulation)
