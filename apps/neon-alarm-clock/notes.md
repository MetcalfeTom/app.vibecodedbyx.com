# Neon Alarm Clock

## log
- 2026-01-09: Initial creation
  - Massive shaking neon time display
  - Web Audio API synthesizer sounds
  - Multi-layered siren alarm
  - Screen flash effects
  - Synth sound buttons

## features
- Real-time clock with huge neon digits
- Individual digit shake animation on change
- Set alarm with time picker
- LOUD multi-oscillator alarm sound
- Siren effect with frequency modulation
- Pulsing volume during alarm
- Screen flash overlay
- 4 synth sound buttons
- Volume control
- "WAKE UP" urgent prompt

## alarm effects
- Massive number shaking animation
- Rapid color cycling (magenta/cyan/red/yellow)
- Screen flashes red
- Multi-layer sawtooth + square wave siren
- Frequency sweeps 240-640 Hz
- Pulsing volume modulation
- Giant STOP button appears

## synth sounds
- Bass: Sawtooth 55-110Hz sweep
- Lead: Square wave arpeggio 440-660Hz
- Arp: Triangle wave C-E-G-C pattern
- Noise: White noise burst

## design
- Dark background (#0a0a0f)
- Cyan time display with heavy glow
- Magenta date display
- Green SET button
- Orange CLEAR button
- Red STOP button (pulsing)
- Orbitron font for digital feel

## controls
- Time picker for alarm
- SET ALARM button
- CLEAR button
- STOP ALARM button (during alarm)
- Volume slider
- 4 synth test buttons
