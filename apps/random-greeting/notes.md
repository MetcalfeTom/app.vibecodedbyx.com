# Random Greeting

## log
- 2026-01-06: Initial creation - simple random greeting generator

## features
- 30 unique greetings with emojis
- Click or press space/enter for new greeting
- Wave animation on emoji
- Fade-in animation on text
- Greeting counter
- No repeat consecutive greetings

## greetings include
- Classic: Hello, Hey, Hi
- International: Bonjour, Hola, Ciao, Aloha, Namaste, Konnichiwa
- Regional: Howdy, G'day mate, Top of the morning
- Casual: Yo, Sup, Wassup, What's up
- Fun: Ahoy, Greetings Earthling, Hey superstar

## design
- Deep blue gradient background
- Playfair Display serif for greetings
- Space Mono for UI elements
- Gradient text (coral to gold)
- Minimal, clean layout

## technical
- Pure vanilla JS
- CSS animations
- No external dependencies
- Click and keyboard support
