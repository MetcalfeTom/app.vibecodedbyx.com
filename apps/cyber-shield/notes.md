## features
- 3D mansion interior view
- 3 gothic arched windows with shield glass
- Shooting star pings from random directions
  - Green (safe), yellow (caution), red (threat)
  - Trails and glow effects
- Splash particles on window impact
- Shockwave ring effect
- Dashboard stats:
  - Pings blocked counter
  - Active threats count
  - Ping rate per second
  - Shield uptime timer
  - Shield integrity bar
  - Threat level (low/medium/high)
- Live ping log with timestamps
- Random IP generation
- Wireframe shield dome visualization
- Ambient floating particles
