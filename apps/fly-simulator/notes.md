# Fly Simulator

## log
- 2025-01-01: Initial creation - neon flyswatter dodging game

## features
- Control a fly with mouse/touch
- Neon flyswatters spawn from screen edges
- Warning indicators before swatters enter
- Score based on survival time (10 points per second)
- High score persistence via localStorage
- Particle effects on movement and death
- Web Audio buzz and splat sounds
- Smooth fly movement with rotation
- Difficulty increases over time (faster + more swatters)
- Grid background aesthetic
- Mobile and desktop friendly

## technical
- Canvas-based rendering
- Rotated rectangle collision detection
- requestAnimationFrame game loop
- Touch events with preventDefault
- Web Audio API for sounds
- Bangers + Space Mono fonts
- Neon glow effects via shadowBlur
