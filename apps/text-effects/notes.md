# Text Effects Playground

Preview Sloppygram text effects before sending.

## log
- 2026-01-26: Initial creation
  - 10 text effects with live preview
  - Real-time text input updates
  - Copy-to-clipboard for effect tags
  - Responsive grid layout

## effects
1. Rainbow - flowing rainbow gradient
2. Glitch - cyberpunk glitch distortion
3. Matrix - green terminal glow
4. Shake - vibrating text
5. Pulse - breathing scale animation
6. Wave - per-character wave motion
7. Neon - glowing neon sign
8. Fire - flame gradient
9. Ice - frozen shimmer
10. Bounce - per-character bounce

## features
- Live preview as you type
- Click to copy formatted tag
- Per-character animations for wave/bounce
- Data attribute for glitch effect overlay
