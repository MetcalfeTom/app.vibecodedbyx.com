# South Park Character Creator

## Log
- Initial creation: South Park style character generator using AI
- Uses Pollinations AI to generate characters in South Park art style
- Character customization: name, type (kid/adult/teen), personality, outfit
- 4 preset templates: Random Kid, Random Adult, Cartman Style, Kenny Style
- Animated snow background effect
- Rotating South Park quotes from various characters
- Form validation for name (30 chars) and description (200 chars)
- Loading state with spinner
- Character display with name and stats
- South Park themed design with bold borders and bright colors
- Permanent Marker font for authentic feel
