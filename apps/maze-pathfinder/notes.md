# Maze Pathfinder

## log
- 2026-01-18: Initial creation
  - Recursive backtracking maze generation
  - A* pathfinding with visualization
  - Bilingual support (English/Spanish)
  - Neon cyan/magenta aesthetic
  - Click to draw/erase walls
  - Speed control slider

## features
- Random maze generation (recursive backtracking)
- A* pathfinding algorithm
- Step-by-step visualization:
  - Open set (cyan)
  - Closed set (magenta)
  - Final path (yellow)
- Language toggle (EN/ES)
- Click/drag to toggle walls
- Speed control slider (1-100)
- Statistics display:
  - Nodes visited
  - Path length
  - Time elapsed
- Mobile touch support
- Glowing start/end/path cells

## design
- Orbitron + Share Tech Mono fonts
- Dark blue/purple gradient background
- Neon colors for visualization
- Grid with subtle lines
- Glow effects on important cells

## controls
- Generate Maze: create random maze
- Find Path: run A* algorithm
- Clear Path: reset visualization
- Language button: toggle EN/ES
- Speed slider: adjust animation speed
- Click/drag on cells: toggle walls

## translations
- English (EN)
- Spanish (ES)
- All UI elements translated
