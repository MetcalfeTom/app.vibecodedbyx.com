# Mundane Life Adventure

## log
- 2026-01-18: Initial creation
  - Choose your own adventure about mundane choices
  - AI-generated images via Pollinations
  - Branching story about socks, breakfast, phones
  - Multiple endings based on choices
  - Dramatic presentation of trivial decisions

## features
- Choose your own adventure format
- AI-generated scene images (Pollinations)
- Branching narrative paths
- Stats tracking:
  - Decisions made
  - Time elapsed
  - Mood changes
  - Inventory items
- Story chapters:
  - Chapter I: The Awakening (alarm, phone, bed)
  - Chapter II: The Sock Drawer (sock choices)
  - Chapter III: Breakfast (food decisions)
  - Epilogue: Various endings
- Sock options:
  - Black socks (safe)
  - Pizza fun socks (whimsical)
  - Mismatched (chaos)
  - No socks (rebellion)
- 4 different endings
- Play again functionality

## design
- Merriweather + Cinzel fonts
- Dark theme with gold accents
- Book/narrative aesthetic
- Dramatic chapter headers
- Fade-in animations
- 16:9 scene images

## story paths
- start → phone_check → doom_scroll → late_start
- start → five_minutes → late_start
- start → sock_drawer (main path)
- sock_drawer → [black/fun/mismatched/no socks] → breakfast
- breakfast → [cereal/coffee/pizza/skip] → endings
