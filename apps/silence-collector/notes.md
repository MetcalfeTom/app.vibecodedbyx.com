# Silence Collector

An idle clicker game about harvesting silence from the void.

## log
- 2026-01-24: Initial creation - click the void to collect silence, 7 upgrades, milestones, localStorage save

## features
- Click the void to harvest silence
- Passive income through upgrades
- 7 upgrade types (whisper catcher, meditation mat, quiet library, etc.)
- Milestone celebrations at 100, 1K, 10K, 100K, 1M
- Ambient floating particles
- Dark, zen aesthetic with Cormorant Garamond font
- Auto-save to localStorage
- Mobile-friendly

## upgrades
1. Whisper Catcher - +0.1/sec
2. Muffled Thoughts - +1 per click
3. Meditation Mat - +0.5/sec
4. Quiet Library - +2/sec
5. Pocket Void - +5 per click
6. Silent Monastery - +10/sec
7. Silence Dimension - +50/sec
