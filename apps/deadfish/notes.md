## notes
### Deadfish language:
- Only 4 commands: i, d, s, o
- i: increment accumulator
- d: decrement accumulator
- s: square accumulator
- o: output accumulator value
- If acc becomes -1 or 256, it resets to 0
- All other characters are ignored

### Famous programs:
- "iisiiiisiiiiiiiio" outputs 72 (ASCII 'H')
- Used to output "Hello World" character by character
