# Boiler Rave

Top-down rave visualizer with 200 ravers, strobes, lasers, and a DJ booth.

## log
- 2026-04-12: Initial build. Canvas-based top-down rave with 200 bouncing ravers (colored circles with arms), DJ booth with spinning turntables, sweeping laser beams, moving spotlight circles, pulsing dance floor tiles, smoke particles, strobe flashes. WebAudio beat engine: kick drum, hi-hat noise, clap. Bass drone (sawtooth 55Hz + sub 36Hz). BPM control (←→, 80-200), DROP mode (space/click) intensifies everything — bigger bounces, faster lasers, strobes, wider spots. Vignette overlay. Orbitron + Azeret Mono typography, neon magenta/cyan aesthetic.

## features
- 200 ravers with individual bounce, sway, arm animations
- Top-down perspective with colored circle bodies
- DJ booth with dual spinning turntables
- Sweeping laser beams that re-randomize every 4 beats
- Moving spotlight circles on dance floor
- Pulsing color floor tiles synced to beat
- Smoke/haze particles
- Strobe flashes (intensified in drop mode)
- WebAudio kick, clap, hi-hat synced to BPM
- Bass drone + sub bass
- BPM control (80-200) with arrow keys
- DROP mode toggle (space/click) — amplifies everything
- Vignette overlay for atmosphere
- Mobile + desktop, responsive canvas
