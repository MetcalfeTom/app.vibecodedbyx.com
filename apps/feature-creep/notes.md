# Feature Creep

Dodge falling innovative widgets that bloat your player. Every hit installs a feature, slowing you down and growing your hitbox. At 100% bloat = Stack Overflow. Grab coffee to reduce bloat.

## log
- 2026-04-12: Initial build. 24 feature widget types with weighted spawn rates (AI Chatbot, Blockchain, NFT Gallery, Dark Mode Toggle, etc.). Player bloat/speed system — hits install features, increase bloat %, decrease speed, grow hitbox. Coffee power-ups (8% chance) reduce bloat by 15%. Expressive player face changes with bloat level (happy → worried → suffering → dead). Installed features orbit player as emoji. Wave progression every 30s increases fall speed and spawn rate. Score based on survival time. Game over at 100% bloat ("Stack Overflow"). Touch + keyboard controls. Space Grotesk + Fira Code typography, dark gradient aesthetic with neon accents.

## features
- 24 innovative feature widget types with emoji icons
- Weighted spawn rates (common vs rare features)
- Player bloat system (speed decrease + hitbox growth)
- Coffee power-up to reduce bloat
- Expressive player face (4 mood levels)
- Installed features orbit player as emoji trail
- Wave progression (speed + spawn rate increase)
- Touch support (left/right tap zones)
- Title screen and game over with stats
- Feature install notification popups
