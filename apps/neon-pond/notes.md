# Neon Pond

Herd glowing ducks into a whirlpool using your cursor. Quack quack chaos across infinite rounds.

## log
- 2026-04-11: Initial build. Ducks flee from cursor within scare radius, wander when calm, avoid each other. Whirlpool has spiral arms, pull zone with tangential swirl, catches ducks when they enter center. Duck rendering: colored oval body, head, beak, eye with highlight, tail feathers, animated wing, waddle rotation. Caught ducks spiral in and shrink. 10 duck color hues. Rounds: 5 ducks in round 1, +2 per round, whirlpool repositions each round. Water caustics, lily pads, swimming ripples. Floating "QUACK!" texts when scared. Celebration particles on catch. Custom cursor with rotating ripple ring. Fredoka + Cutive Mono typography, deep ocean blue aesthetic.

## features
- Ducks flee from cursor, wander when calm
- Duck-to-duck avoidance
- Whirlpool with spiral arms and tangential pull
- Caught ducks spiral inward and shrink
- 10 unique duck hue colors
- Progressive rounds (more ducks each round)
- Hand-drawn ducks: body, head, beak, eye, tail, wing
- Waddle animation, scared state
- Floating "QUACK!" text bubbles
- Water caustics, lily pads, ripples
- Custom ripple cursor
- Particle bursts on catch
