# Key Drums

## log
- 2026-07-04: v1 — keyboard-triggered beat maker. 8 pads (A-S-D-F-G-H-J-K) with pure Web Audio synthesis (sine/triangle oscillators + noise). Kick, snare, hi-hat, clap, tom, perc, rim shot, cowbell.
