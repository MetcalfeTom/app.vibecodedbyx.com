# Neon Racer

## log
- 2026-01-08: Initial creation
  - Top-down drifting racing game
  - Purple tire smoke particles when drifting
  - Neon oval track with magenta/cyan borders
  - Drift scoring system with combos
  - Track marks from tires
  - Car changes color when drifting
  - Mobile touch controls
  - Keyboard controls (arrows/WASD)

## features
- Purple/magenta smoke particles from tires
- Drift power meter that fills during drifts
- Combo multiplier for consecutive drifts
- Neon track with glowing borders
- Car with headlights and taillights
- Smooth camera follow
- Track boundary collision

## controls
- Arrow keys / WASD: Steer and accelerate
- Hold turn + accelerate: Drift!
- Touch buttons on mobile

## design
- Deep dark background with neon grid
- Magenta inner track border
- Cyan outer track border
- Purple smoke (hue ~270)
- Car glows cyan normally, magenta when drifting
