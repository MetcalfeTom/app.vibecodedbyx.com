# terminal-herd

## log
- 2026-04-27: Created. **Neon CRT terminal where every keystroke spawns a wandering ASCII cow**. Full-screen phosphor green aesthetic with magenta/cyan chromatic-split title (Major Mono Display), CRT scanlines via `repeating-linear-gradient` multiply overlay + radial vignette + 4.6s flicker keyframe on title. **Layout**: `<header>` (title + HERD/KEYS pill counters + ♪ON/cull buttons) → `#pasture` (relative-position canvas where cows live) → `.promptbar` (`guest@pasture:~$` prompt with VT323 22px text input + amber blinking caret). **Boot text** top-left fades from opacity 0.7→0.12 over 12s (8 lines describing cow init). **8 cow breeds** as ASCII art rows: classic, sleepy (zz eyes), angry (XX eyes), tiny (3-line compact), cosmic (* eyes + ~ underbelly), derpy (asymmetric eyes), mini (4×3 micro-cow), lonk (extra-long body). **6 colors** picked at random: phosphor `#7dffb0`, amber `#ffd25b`, cyan `#5ee0ff`, magenta `#ff5ea1`, violet `#c47bff`, lime `#a8ff6f` — each cow has `text-shadow:0 0 3px currentColor` for phosphor bloom. **Spawn mechanic**: `input` event diffs new vs prev value; if a char was added (i.e. typed, not deleted), `state.keyCount++` and `spawnCow()` runs. Throttled to 28ms between spawns so paste doesn't flood (most natural typists never hit it; spam-paste gets sampled). MAX 80 cows — oldest culled FIFO. **Cow physics**: each cow has `{x, y, vx, vy, w, h, bobPhase, restUntil, facing, lastMooAt}`. Wander AI ticks per frame: 0.18% chance to start a 0.6–2s rest pause, 0.5% chance to flip direction. While walking `vx ∈ [-1, +1]` px-tick (`* dt * 0.06`), bobPhase advances (sin → ±1.4px y bob), facing flipped via `scaleX(-1)`. Wall collision = bounce + facing flip. Y stays in upper portion (clamped to `H-70-h`). **Moo speech bubbles**: 32% chance per rest pause to spawn a `.moo` Bungee text bubble above the cow (random pick from `MOO_VARIANTS` = 9 strings: MOO/moo!/MOOO/m·o·o/mrrhh/mooo~/MUUH/mawww/BWAH); CSS keyframe `mooFloat` (0→opacity 1 + translateY(-12px) → opacity 0 + translateY(-78px) + scale 1.1, 1.4s ease-out). Plays a **moo synth**: sawtooth 90Hz→130Hz→80Hz exp ramp with 5.4Hz vibrato (LFO sine through gain 4 to osc.frequency) through 720Hz LP Q=1.6, ~0.42s. Pitch slightly random each call. **Spawn blip**: square wave 1200–1800Hz, 60ms decay — every keypress. **Enter key**: empty input → collective moo (every cow speech-bubbles in 40ms cascade) + low moo synth; non-empty → picks 1 random cow to "say" the typed text (clipped to 24 chars) as a moo bubble + clears input. **Cull button + C key**: fades all cows via `.dying` class (opacity 0 over 0.5s) then DOM-removes; resets herd. Plays low `moo(0.6, 0.7)` farewell. **HUD pills**: HERD count (animates via `countBump` keyframe scale 1.16 + amber flash on each spawn) + KEYS (lifetime keystrokes counter). **Welcome cow** at x=80 with "welcome" speech bubble appears 500ms after load. **Aesthetic**: deep dark green void `#02110a→#06291a`, phosphor `#7dffb0`, amber prompt caret + `b` accent, magenta+cyan chromatic title shadows. VT323 + Major Mono Display + Space Mono + Bungee fonts. Mobile breakpoint @560px shrinks header pills + hides hint. Pollinations OG.

## issues
- **Audio is browser-gated** — first user pointerdown OR keydown calls `actx.resume()`. Until then no moo audio. Mute toggle also acts as unlock.
- **Cow throttle 28ms** — pasting "hello world" still drops 1 cow per char in fast bursts but caps spawn rate to ~35/s so the page doesn't seize. Long paste = some chars get no cow.
- **MAX_COWS = 80** — beyond that oldest cow is FIFO-culled (no fade animation, just removed). At 80 cows on screen the wander AI is still cheap (~0.05ms/frame); could push to 150 if needed.
- **Wall bounce** is axis-aligned-only — cows can briefly overlap each other (no inter-cow collision). Acceptable for an ambient pasture.
- **Cow art width estimation** uses `8.4 * longest_row` chars (VT323 is monospace ~8.4px at 15px font-size). Slightly fudgy on certain breeds with longer overhang rows but fine for boundary clamp.
- **Speech bubble placement** is at `c.x + c.w/2` which is the original (untransformed) position — for `scaleX(-1)` cows the bubble still sits over the cow body since the transform is around center.
