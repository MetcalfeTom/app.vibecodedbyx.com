# Magnet Board

Draw with magnetic colors that push each other apart. Same colors attract slightly, different colors repel.

## log
- 2026-04-13: Initial build. Up to 4000 particles with spatial grid acceleration for neighbor lookups. 9 colors. Different colors repel (inverse-distance force), same colors attract slightly (cohesion). Adjustable brush size (3-20). Toggle repulsion on/off, freeze particles, clear board. Particles have glow halos and specular highlights. Wall bouncing. Velocity damping and clamping. Brush preview circle. Subtle background grid. Touch support. Space Grotesk + DM Mono typography, dark minimal aesthetic.

## features
- 9 drawing colors
- Different colors repel each other
- Same colors attract slightly (cohesion)
- Up to 4000 particles
- Spatial grid for fast physics
- Adjustable brush size
- Toggle repulsion / freeze / clear
- Particle glow and highlights
- Wall bouncing
- Touch support
- Brush preview
