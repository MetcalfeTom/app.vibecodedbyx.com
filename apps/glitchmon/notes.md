# GlitchMon - Neon Pixel Monster Battler

## log
- 2026-01-14: Initial creation
  - 6 unique pixel art monsters with neon glow effects
  - Turn-based combat system (Attack, Glitch, Defend, Repair)
  - Monster selection screen
  - Battle log with color-coded messages
  - HP bars with color states (green/yellow/red)
  - Hit animations with glitch effects
  - Simple enemy AI weighted by HP
  - Victory/Defeat screens
  - Scanline overlay for retro CRT feel
  - Press Start 2P font for pixel aesthetic

## features
- 6 monsters: BYTEBUG, VIRALOX, NULLVOID, ERROREX, STACKMON, PIXLGHOST
- Each has unique stats: HP, Attack, Defense, Special
- 4 actions: Attack (basic), Glitch (special), Defend (halve damage), Repair (heal)
- Pixel art rendered with CSS grid
- Neon glow effects on all monsters
- Glitch animation effect on sprites
- Battle log tracks all combat
- Mobile responsive

## controls
- Click monster to select
- Click action buttons during battle
- Touch friendly for mobile
