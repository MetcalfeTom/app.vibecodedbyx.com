# Neon Fox Tickle Simulator

## log
- 2026-01-18: Initial creation
  - CSS-based neon fox character
  - Interactive tickle zones
  - Web Audio API laughter synthesis
  - Vibration effects
  - Laugh meter and combo system

## features
- Pure CSS neon fox character
  - Glowing orange body
  - Cyan glowing eyes
  - Pink cheeks when tickled
  - Animated tail wagging
  - Mouth opens when laughing
- Tickle zones:
  - Belly (3x multiplier)
  - Chin (2x multiplier)
  - Ears (1.5x multiplier)
  - Paws (1.5x multiplier)
- Synthesized laughter sounds
  - Web Audio API oscillators
  - Sawtooth waves with bandpass filter
  - "Ha ha ha" rhythm
  - Squeaky overtones at high intensity
- Visual effects:
  - Fox vibration animation
  - Manic shaking at high levels
  - Emoji/text particles
  - Screen shake effect
  - Blush cheeks
- Laugh meter (0-100%)
- Mood levels:
  - Calm, Giggly, Laughing, Hysterical, MANIC, CHAOS
- Combo system for rapid tickling
- Device vibration API support
- Sound toggle

## design
- Bangers + Nunito fonts
- Neon orange/cyan color scheme
- Dark purple gradient background
- Glowing text shadows
- CSS-only fox character

## stats tracked
- Total tickles
- Current mood
- Combo multiplier
