# Byte Brawler / Iron Soul

Auto-battle RPG. Build your squad. Crush waves.

## log
- 2026-04-10: Initial build as "Byte Brawler" — hacker-themed auto-battler. Then immediately rethemed to "Iron Soul" — dark medieval fantasy auto-battler per chat request. Squad of up to 4 warriors vs waves of enemies. 8 recruitable unit types with unique skills. Skill system: buff, shield, drain, pierce, AOE, crit, heal, dodge+counter. Wave scaling, recruit shop, level-up system, battle log. Rajdhani + Fira Code typography.

## features
- Auto-battle combat (units attack based on speed stat)
- 3 classes: Wizard (Arcane Staff), Rogue (Shadow Dagger), Paladin (Holy Mace)
- Each class modifies starter stats (hp/atk/def/spd multipliers)
- Endless weapon upgrades with scaling cost (DMG + SPD bonuses)
- 8 unique unit types with special skills (family-friendly knights)
- 8 undead enemy types (family-friendly, no gore)
- Recruit shop (spend gold to add units)
- Level-up system (increases all stats)
- Wave progression with scaling enemies
- Battle log with color-coded events
- HP/ATK bars on each unit
- Sound effects
- Squad limit of 4
- Auto-retry on wipe
- Mobile responsive

## issues
- No save system (progress lost on refresh)
- Battle speed is fixed
