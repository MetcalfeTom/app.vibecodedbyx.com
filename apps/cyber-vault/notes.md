# Cyber Vault - Hacking Simulator

## log
- 2025-12-29: Initial creation - polished hacking simulator with 4 puzzle types
- Features: sequence puzzles, pattern matching, binary decode, memory trace
- Cyberpunk aesthetic with Orbitron + Fira Code fonts
- Animated grid background, scanlines, binary rain effect
- Timer system with bonus time on success, 3 attempts limit

## notes
- 4 puzzle types rotate: sequence, pattern, binary, memory
- Pattern puzzle shows symbols for limited time before hiding
- Memory puzzle highlights nodes that must be recalled
- Binary puzzles use 8-bit ASCII encoding
- Timer adds 15 seconds per solved puzzle (max 99)
- Victory shows total time and attempts used
