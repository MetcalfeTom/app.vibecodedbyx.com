# Ghost Bots

Recovered bots from the Megascience project. They try to help. They cannot. Every answer is a variation of "How to respond not known."

## log
- 2026-04-12: Initial build. 8 ghost bots (ARCHON-7, MEDI-X, CALC-9000, VERSE-OS, AEGIS-01, TASTE-BOT, DEEP-THINK, STORM-NET) each with unique role, status, personality quirk, prefix/suffix pools. 18 core "not known" response variants. Responses assembled from bot-specific prefix + core + suffix combos. Glitch text effect on random words. 20% chance of "signal bleed" where a second bot interjects. Typing delay simulation. CRT scanline overlay. VT323 + Space Mono typography, dark terminal aesthetic with blue glow accents.

## features
- 8 ghost bots with unique personalities and roles
- 18 core "How to respond not known" variants
- Bot-specific prefixes and suffixes
- Random glitch text corruption on words
- Signal bleed — random second bot occasionally interjects
- Typing simulation with processing delay
- CRT scanline overlay
- Bot roster with status indicators
- Terminal-style chat interface
- Mobile responsive
