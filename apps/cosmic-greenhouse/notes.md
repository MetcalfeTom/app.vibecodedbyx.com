# cosmic-greenhouse

## log
- 2026-05-27: Initial build. **Important name decision**: the chat asked for `neon-garden` but a click-to-plant flower app already exists at that slug (from 2026-03-16). Per CLAUDE.md "never delete apps unless very good reason" and "prefer iterating over creating new ones", I shipped this fundamentally-different concept (passive, viewer-driven, alien greenhouse) under `cosmic-greenhouse` so both can coexist. Display title is "Cosmic Greenhouse" but it honors the chat brief: a cosmic greenhouse where glowing alien flora grows taller based on `viewer_count`. **Viewer count source**: Supabase Realtime presence channel `cosmic-greenhouse:room`. Each visitor joins with a random `gardener-XXXXXX` key. Presence sync event → `state.viewerCount = max(1, Object.keys(presenceState).length)`. Falls back to 1 if Supabase fails to import or connect (graceful degradation — garden still runs solo). Connection status indicator in HUD: lime pulsing dot when live, magenta dot on failure, grey when solo. **Growth model**: `rate = 0.04 + 0.05 × viewerCount` per second. At 1 viewer = 0.09/s = ~11s per stage = ~66s to full bloom. At 10 viewers = 0.54/s = ~1.85s per stage = ~11s to full bloom. **Six rotating slots**, each holding one plant in stages 0-5. When a plant reaches stage 5 it holds for 8 seconds (the "bloom" window), then reseeds with a new species (avoids picking the same species as immediate neighbors). Lifetime bloom counter persists in localStorage. **Six alien species**, each procedurally drawn per stage: (1) **PULSEWORT** — stem grows + leaves sprout + a pulsing radial-gradient bulb appears at stage 2+; (2) **SPIRAL FROND** — 60-segment 3D-feeling helix that adds turns + radius + tip-glow as it grows; (3) **LUMEN CAP** — mushroom with widening cap, glowing dot gills appear at stage 1.5+, spore halo at stage 4.5+; (4) **TRUMPET VINE** — stem + leaf + bezier-bell trumpet that "opens" via parameterized cubic curves with `openness` proportional to stage, glowing stamen at stage 4.5+; (5) **STAR BRAMBLE** — vertical stem with horizontal branches that gain glowing 5-pointed stars at their tips, plus a master star at the apex; (6) **BIO-CORAL** — recursive branching (max depth = 2 + stage), each branch forks at ±0.4 + sin-noise angle, leaf tips pulse. BIO-CORAL uses a deterministic seeded PRNG keyed by slot+species so the branching pattern is stable across redraws (otherwise it would jitter each frame). **Each plant's max height varies by species** (110-165 units), so the garden silhouette has natural variety. **Stage chimes** play on each stage advance (triangle 440 + 110·stage Hz, pitch sweeps up). **Bloom chord** plays on stage-5 advance: 5-note triangle major-7 ascending arpeggio at root 330Hz × (1, 1.25, 1.5, 1.875, 2.25). **Per-stage burst** spawns 8 particles; **bloom burst** spawns 28. Particles drift upward then fall under gravity (60 u/s²), fade over 0.7-1.5s, render with shadow-blur additive blend. **Ambient drone toggle** — 4-voice chord (55/73.42/110/138.59 Hz sine+triangle 0.5% detune) through resonant lowpass swept by 0.04-0.16Hz LFOs ±160Hz, 2.4s fade-in. **Background**: void with 3 layered radial nebula glows (violet top-left + jade mid-right + magenta bottom-center) + 90 deterministic stars + dome arch line drawn with `quadraticCurveTo` and shadow-blur cyan glow + faint highlight arch above it. **Soil band**: dark vertical gradient + 26 random bioluminescent flecks scattered across the soil + a glowing surface strip on top. **36 drifting spores** rise from the bottom in `lighter` blend mode, twinkling with `sin(tw)` opacity, wrap horizontally + reset to bottom when they leave the top. **Stage progress bars** drawn just below each plant's soil position — 5 little segments showing the current plant's growth progress through the stages with hue-matched fills. **HUD top**: viewers (cyan) + blooms (gold) + tallest growing species (magenta name). **Bottom strip**: italic Cormorant quote that rotates every 10s with idle observations, swaps to a species-specific bloom phrase when a plant tops out (e.g. "fully crowned. · trumpet vine has bloomed."). Drone + mute toggles. **Defensive init**: every step in try/catch including Supabase import, global error toast on window.error + unhandledrejection. **WCAG basics**: rem units, semantic main/h1, role=img + descriptive aria-label on canvas, aria-live on quote, aria-pressed on drone+mute toggles, focus-visible 2-3px outlines, ≥2.5rem touch targets, prefers-reduced-motion damps dt ×0.65 + freezes drift opacity. **Aesthetic**: Audiowide jade→magenta→gold gradient title, Major Mono Display tagline, Cormorant Garamond italic for the rotating quote (editorial feel), Share Tech Mono for HUD body. **Note on the original neon-garden slug**: the click-to-plant flower garden (2026-03-16 build) remains untouched at `/neon-garden/` — chat users can use either.

## issues
- Supabase presence count counts tabs, not unique users. Someone with 3 tabs open looks like 3 viewers. Acceptable for "vibe" purposes; could dedupe by anonymous user_id if needed.
- Growth rate is global (all 6 slots grow at the same rate at any moment). If viewerCount changes mid-stage, growth speed changes immediately — visible but not jarring.
- BIO-CORAL's stable seeded PRNG re-seeds based on slot + species index, so the same species in the same slot always produces the identical branching tree. Variety across slots is preserved.
