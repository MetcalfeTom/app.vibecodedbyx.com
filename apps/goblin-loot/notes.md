# Goblin Loot Bag

## log
- 2026-01-15: Initial creation
  - Neon goblin aesthetic with greens and golds
  - 10 treasure types: Gold Coins, Gold Bars, Rubies, Emeralds, Sapphires, Diamonds, Amethysts, Pearls, Crowns, Magic Rings
  - Each treasure has a value in gold
  - Add/remove buttons for each treasure
  - Quick loot buttons for common additions
  - Total value calculation with K/M formatting
  - Persists to localStorage
  - Goblin quotes that change when adding items
  - Sparkle effects on adding treasures
  - Clear bag option with confirmation
  - Creepster font for goblin vibes
  - Mobile responsive

## features
- Track 10 different treasure types
- Each has icon, name, count, and gold value
- Total hoard value calculated automatically
- Quick add buttons for fast looting
- Data persists in browser storage
- Fun goblin quotes
- Shimmer and sparkle animations
- Responsive design
