# Tombstone Todo

## log
- 2026-01-17: Initial creation
  - Gothic graveyard aesthetic
  - Tasks carved into stone tombstones
  - Crumbling animation when completed
  - Dust particle effects
  - localStorage persistence

## features
- Tasks appear as stone tombstones with R.I.P. header
- Carved text effect with inset shadows
- "Lay to Rest" button to complete tasks
- Crumbling animation with blur effect
- 40 dust particles scatter on completion
- Date carved shown on each tombstone
- Stats: Standing (active) vs Crumbled (completed)
- localStorage saves tasks and completion count
- Empty state with lone tombstone icon
- Fog overlay for atmosphere

## design
- UnifrakturMaguntia for gothic headers
- Cinzel for carved text
- Dark grays and muted purples
- Stone texture overlay on tombstones
- Rounded top tombstone shape
- Inset shadows for carved effect
