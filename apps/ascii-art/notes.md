# ASCII Art Generator

## log
- 2025-01-01: Initial creation - text to ASCII art converter with 6 font styles

## features
- 6 font styles: Block, Banner, Small, Slant, Big, Mini
- 6 color options (neon green, cyan, magenta, yellow, orange, white)
- Live preview as you type
- Copy to clipboard
- Full alphabet A-Z, numbers 0-9, basic punctuation
- Retro terminal aesthetic

## technical
- Pure JavaScript with pre-defined font character mappings
- No external libraries - all fonts defined inline
- VT323 font for output (classic terminal look)
- Space Mono for UI elements
