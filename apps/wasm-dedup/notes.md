# WASM Deduplicator

## log
- 2026-01-18: Initial creation
  - High-speed file deduplication simulation
  - Glowing hex-code particles
  - WASM purple/cyan color theme
  - Real-time stats and logging

## features
- Visual file block spawning with hashes
- Duplicate detection (30% chance of duplicate hash)
- Connection lines between duplicates
- Animated merging and dissolving
- Floating hex-code particles
- Scan line effect
- Real-time statistics:
  - Files scanned
  - Duplicates found
  - Space saved
  - Hash rate

## design
- JetBrains Mono + Orbitron fonts
- WASM-inspired purple/cyan palette
- Grid background effect
- Glowing neon elements
- Particle system with hex codes

## controls
- Start/Stop scan buttons
- Speed slider (1-10x)
- Auto-cleanup of old files
