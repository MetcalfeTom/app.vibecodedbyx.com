# Neon Ball Pit

## log
- 2026-01-17: Added third-person character
  - Neon humanoid with body, head, arms, legs
  - Physics-based movement with Cannon.js
  - Third-person camera follows character
  - WASD/Arrow keys to move
  - Spacebar to jump
  - Mobile joystick + jump button
  - Character pushes balls around
  - Enlarged pit for more room
- 2026-01-17: Initial creation
  - Babylon.js for 3D rendering
  - Cannon.js for physics
  - Neon colored spheres with emissive glow
  - Gravity toggles (on/off/reverse)
  - Chaos mode with random impulses
  - Click to spawn balls
  - Arc rotate camera

## features
- Third-person playable character
  - Magenta body, cyan head, yellow limbs
  - Glowing white eyes
  - Physics-enabled movement
  - Camera-relative controls
- Physics-enabled ball pit container (16x16 units)
- 8 neon colors (cyan, magenta, mint, hot pink, purple, yellow, orange, blue)
- Gravity controls:
  - Normal gravity (on)
  - Zero gravity (off)
  - Anti-gravity (reverse)
- Chaos mode: random impulses every 500ms
- +10 balls button
- Reset button (also resets character)
- Ball count display
- Semi-transparent walls with neon edge lines
- Mobile joystick and jump button
