## issues
- Headphones are deliberately non-removable, per the speech bubble joke. If chat wants a cosmetic system later (different headphone colors, sunglasses, hats), state.cosmetics would need to be added and SVG groups conditionally rendered.
- Stat decay is calibrated for desktop pacing (~3-5 min sessions). On mobile, smaller buttons + cooldowns may feel grindy. Could add a difficulty toggle.
- Caffeine over 85 escalates glitches, simulating the wired-coding state, but there's no in-game warning of this synergy. Players discover it via stat-watching.
- localStorage is NOT used yet ‚Äî refresh resets day + streak. Easy add for v2.
- No fail-state for "Stream attempted while not ready" beyond a toast. Could add a small glitch-bump or cooldown penalty.
- The creature's body is a single SVG path ‚Äî replacing the species (e.g. green slime variant, blue ghost variant) would mean swapping the body path + gradients. Could expose as a state.species enum.
