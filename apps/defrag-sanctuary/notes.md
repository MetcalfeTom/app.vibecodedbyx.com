# Defrag Sanctuary

Watch obsidian blocks drift into perfect alignment in an endless dark void.

## log
- 2026-04-12: Initial build. Canvas grid of obsidian blocks with spring-damped physics (position, rotation, scale). 4-phase cycle: scatter (explode apart) → seek (drift toward targets) → settle (snap into grid) → glow (aligned state with chime). Obsidian rendering: gradient faces, edge highlights, shadow edges, diagonal streak textures, radial glow on alignment. Shuffled target positions each cycle (Fisher-Yates). Ambient nebula drifts, ghost grid lines during settle/glow. Alignment counter + progress bar. Gentle sine chime arpeggio on full alignment. Click to restart scatter. Cormorant Garamond + Martian Mono typography, deep void purple/obsidian aesthetic.

## features
- Spring-damped block physics (position, rotation, scale)
- 4-phase endless cycle: scatter → seek → settle → glow
- Obsidian block rendering with gradient, edges, texture streaks
- Blocks glow faintly when aligned
- Shuffled grid targets each cycle
- Alignment counter and progress bar
- Ambient nebula background drifts
- Ghost grid lines during settling
- Gentle chime arpeggio on alignment
- Click to scatter blocks
- Responsive grid sizing
- Vignette overlay
