# Neon Sandbox

## log
- 2026-01-11: Created neon physics sandbox with balls, squares, triangles, stars, and hexagons
- 2026-01-11: Added robot arm with grab & throw mechanics
- 2026-01-11: Added neon pig shape type

## features
- Click/drag to spawn shapes
- 7 shape types: circle, square, triangle, star, hexagon, pig
- Robot arm tool to grab and throw shapes
- Physics: gravity, bounce, friction, collisions
- Neon glow trail effect
- Clear all, toggle gravity, shake buttons
- Max 200 shapes (removes oldest when exceeded)
- Mobile and desktop friendly
