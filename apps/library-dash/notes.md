# Library Dash

Return overdue books to the shelves while dodging an angry librarian. Shh!

## log
- 2026-04-13: Initial build. Grid-based stealth/collection game. Procedural library layouts with shelf aisles, walking gaps, and return desk. Books scattered on floor glow and must be returned to highlighted shelf locations. Librarians patrol waypoints, have line-of-sight detection, alert/chase states with "SHH!" speech bubbles. Player has 3 lives, invincibility frames on catch. Combo system for returning books quickly. Level progression increases book count, librarian count/speed/sight, shelf density. Libre Baskerville + Special Elite typography, warm library aesthetic with carpet tiles and wood shelves. Mobile d-pad controls.

## features
- Procedural library layout generation per level
- Books scattered on floor with glow indicator
- Pick up and return books to correct shelf location
- Dashed highlight showing where carried book goes
- Librarian AI with patrol, line-of-sight, alert, and chase states
- "SHH!" speech bubble and angry eyebrow animations
- Alert (?) and chase (!) indicators
- 3 lives with invincibility frames and screen shake
- Combo scoring for quick returns
- Level progression (more books, more/faster librarians)
- Depth-sorted character rendering
- Vignette overlay
- WASD + arrow key controls
- Mobile d-pad touch controls
- Game over with score display, press space to restart
