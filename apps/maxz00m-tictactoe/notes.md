# Neon Tic-Tac-Toe - maxz00m Edition

## log
- 2026-01-08: Initial creation for +maxz00m
  - Neon cyan (X) vs magenta (O) theme
  - Persistent scoreboard with localStorage
  - Win tracking for X, O, and draws
  - Animated win line effect
  - Mobile responsive

## features
- Neon aesthetic with cyan X and magenta O
- Scoreboard tracking X wins, O wins, and draws
- Scores persist in localStorage
- Animated piece placement
- Glowing win line across winning cells
- Turn indicator highlights current player
- New Game and Reset Scores buttons

## design
- Orbitron + Rajdhani fonts
- Dark background with neon glows
- Gradient text effects
- Pulsing animations on win
- Mobile responsive grid

## technical
- Pure vanilla JS, no dependencies
- localStorage for score persistence
- CSS animations for effects
- CSS Grid for board layout
