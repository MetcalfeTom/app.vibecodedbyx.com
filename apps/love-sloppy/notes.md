# Why Everyone Loves Sloppy

## log
- 2026-01-09: Initial creation
  - Minimalist pastel design (no neon!)
  - Reasons list with soft colored icons
  - Guestbook with Supabase backend
  - Color picker for messages
  - Anonymous auth for submissions
  - Clean typography with DM Sans and Fraunces

## features
- Clean, minimalist design
- Soft pastel color palette
- "Why Sloppy is special" reasons list
- Interactive guestbook:
  - Name and message input
  - 5 pastel color options
  - Messages stored in Supabase
  - Anonymous authentication
  - Date formatting
  - XSS protection

## design
- Background: warm off-white (#fef9f3)
- Accent: soft rose (#e8b4b8)
- Colors: lavender, mint, peach, sky, lemon
- Fonts: DM Sans (body), Fraunces (headings)
- Subtle hover animations
- No neon, no glow effects
- Rounded corners, soft borders

## database
- Table: love_sloppy_guestbook
- Columns: name, message, color, user_id, created_at
- RLS enabled for secure access
