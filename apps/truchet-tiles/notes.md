# Truchet Tiles Generator

Generate mesmerizing geometric patterns with Truchet tiles.

## log
- 2026-01-22: Initial creation with 5 tile patterns and anchor point editor

## features
- 5 built-in tile patterns: Quarter Arcs, Triangles, Diagonal Lines, Maze Lines, Half Circles
- Custom tile designer with anchor point editor
- Adjustable grid size (4-40 columns/rows)
- Color customization (background and foreground)
- Line width adjustment
- Click tiles to rotate them manually
- Export to PNG
- Responsive design

## anchor point editor
- Click to place anchor points
- Points snap to edges, corners, and center
- Select two points to create a connection
- Choose between arc or line connections
- Edge points connect with adjacent tiles for seamless patterns

## tile patterns
- **Quarter Arcs**: Classic Truchet pattern with quarter circles at opposite corners
- **Triangles**: Solid triangles creating diagonal patterns
- **Diagonal Lines**: Simple diagonal line connections
- **Maze Lines**: L-shaped connections creating maze-like patterns
- **Half Circles**: Semicircles on tile edges
- **Custom**: User-defined patterns via anchor editor
