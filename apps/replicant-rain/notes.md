# replicant-rain

## log
- 2026-04-27: Created. **Blade Runner 1982 cyberpunk 3D megacity** built on **three.js r128** (CDN: jsdelivr). Combines four queued chat requests (Tyrell pyramid, neon rain, billboards, post-apocalyptic streets) into one cinematic scene. **Scene**: PerspectiveCamera 58° at `(0, 28, 120)` looking at `(0, 50, -160)` with subtle sin/cos drift. `THREE.FogExp2(0x1a0a18, 0.0098)` for thick smog falloff. **Lights**: AmbientLight 0x331a26 0.4 + orange DirectionalLight `0xff7a3a` from upper-back + cyan PointLight + magenta PointLight. **Tyrell Pyramid** at `z=-240`: 9 stacked truncated CylinderGeometry cones (4-sided segments + rotated π/4 = pyramid faces), each 18u tall, baseR `70 → 10`, dark slate `0x18141c` w/ subtle emissive. Each level has TorusGeometry window-slits in concentric horizontal rings. Top: ConeGeometry spire + pulsing red beacon SphereGeometry (sin-pulse intensity) + AdditiveBlending sun-glow Sprite (orange→amber radial gradient halo). **City buildings**: 140 procedural BoxGeometry instances scattered around camera (avoiding pyramid footprint), heights 30–180u, widths 12–30u. Each building gets a per-instance `64×256` canvas-painted window texture: 5×22 grid of lit/unlit cells, ~55% lit, 4 color tiers (55% orange `#ffaa44` / 23% gold `#f5c453` / 14% cyan `#5ee0ff` / 8% magenta `#ff5ea1`) — gives every building a distinct lit-window pattern. Tall buildings (>140u) get 1–2 thin antenna CylinderGeometry rods on top. **Billboards** (10 spawned on tall building faces): each is a 18×9u PlaneGeometry with `512×256` canvas texture rendering 2 lines of bold neon text + double-stroke glow border, plus an additive-blending glow plane behind. 12-ad library cycles every 5–10s with random flicker pulse: `CALL 魂 / 2049`, `ATARI`, `COCA-COLA / Enjoy`, `OFF-WORLD / 殖民地`, `百年 SAKE`, `NEXUS-7 / SOON`, `TDK / TYRELL`, `JVC`, `BUDWEISER`, `PAN AM`, `RICHO / 理光`, `銀河 GALAXY`. **Rain**: 1400 streaks in single LineSegments BufferGeometry — each streak = 2 vertices (0.4u lean + 2.4u length), per-streak `vy ∈ [60, 120]u/s`, recycled to top of cloud volume when `y < 0`. Cyan→white emissive material. Cheap single drawcall. **Spinners** (6 initial, max 18 spawnable): each is a Group with body BoxGeometry + canopy + headlight Sphere + tail Sphere + AdditiveBlending glow Sprite (cyan or magenta). 18% chance police variant with siren-flashing red↔blue intensities (sin-pulse). Flight path = ellipse `(a, b, cx, cz, cy, speed, phase)` with random params; spinner faces flight tangent. **Searchlights**: 3 ConeGeometry beams (orange / cyan / magenta) tall `120u` w/ AdditiveBlending + low opacity, slow rotation around y-axis from rooftops. **Wet street ground**: 800×800 PlaneGeometry rotated horizontal, with 2048×2048 canvas texture — radial dark gradient base + 32px road grid lines + 60 random bright puddle highlights (random size + blur via shadowBlur). **Audio** (Web Audio): **ambient drone** = 4-voice chord (A1/E2/A2/C#3) sine+triangle through 480Hz LPF with `0.062Hz` LFO sweeping cutoff ±240Hz, plus per-voice `0.04–0.105Hz` detune LFOs (±2–6¢). Faded in over 1.8s on first user interaction. **rain noise loop** = AudioBufferSource of 4s pink-noise buffer through highpass1200Hz + lowpass4800Hz, gain 0.085, smoothed gain ramps. **thunder** = lowpass-filtered noise burst 2.4s + 70→28Hz sub thump triggered 360–780ms after lightning flash. **lightning** = DOM `.lightning` div white overlay animates opacity `0 → 0.85 → 0.2 → 0.7 → 0` over 460ms via WAAPI; auto-fires every 9–24s. **HUD**: Major Mono Display "REPLICANT · RAIN" title with cyan/magenta chromatic-split shadow, Noto Serif JP "記憶を持つ涙" eyebrow ("tears that hold memory"), Audiowide stat-pills top-left (SECTOR/RAIN/SPINNERS) + top-right (TYRELL.HQ/SMOG/FRAME). Bottom control bar: 4 Audiowide pill buttons — RAIN+/STORM, SPINNER+ (spawn extra), STRIKE (manual lightning), AUDIO ON/OFF. **Intro overlay**: Major Mono Display title + Noto Serif JP subtitle + Cinzel italic flavor text + "⌐ ENTER THE CITY" button (also unlocks AudioContext). **Effects layers** (DOM): SVG `feTurbulence fractalNoise` grain at 0.04 opacity, radial vignette overlay, scanline overlay (`mix-blend-mode: screen`), CRT subtle curvature. **Aesthetic**: deep void black `#08050d`, rust `#b55135`, neon cyan `#5ee0ff`, magenta `#ff5ea1`, gold `#f5c453`. Major Mono Display + Cinzel + Audiowide + VT323 + Noto Serif JP + Space Mono. Pollinations OG.

## issues
- **three.js r128** is CDN'd (~600KB cached after first load). If CDN is blocked, scene won't render.
- **140 buildings + 10 billboard textures** = ~150 unique CanvasTextures uploaded to GPU once at boot. Slight stutter on first frame, then steady. Could pool textures if perf complaints.
- **Rain BufferGeometry** uses single drawcall (1400 segments) — efficient, but repositioning happens on CPU. At 1800+ streaks, the per-frame Float32Array writes start to bite. 1400 is the sweet spot.
- **Spinners on ellipse paths** can overlap visually (no collision). Acceptable — it's atmospheric chaos, not a sim.
- **Police variant flash** uses sin-pulse intensity, not strobe — easier on eyes during long viewing.
- **Lightning auto-cadence (9–24s)** intentionally rare; user can hit STRIKE button for on-demand.
- **No camera orbit controls** — fixed dramatic angle. Intentional. Could add subtle drag-to-orbit later.
- **Billboards always face camera roughly** because they're attached to building faces — at extreme camera angles they may foreshorten oddly. Acceptable for distant buildings.
- **Pyramid windows** are TorusGeometry rings, not individual lit panes — gives distant-glow look but loses detail up-close. Could swap for InstancedMesh window panes if camera ever zooms in.
