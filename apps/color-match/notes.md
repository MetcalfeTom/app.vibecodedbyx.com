# Color Crush - Match 3 Puzzle Game

## Log
- Initial creation: Match-3 puzzle game with tile swapping
- Features:
  - 8x8 grid with colorful emoji tiles
  - Click two adjacent tiles to swap
  - Match 3+ in a row/column to crush
  - Cascade mechanics - tiles fall down
  - Auto-refill from top
  - Chain reactions for combos
  - Move limit system (30 moves)
  - Score tracking
  - Best combo tracking
  - Shuffle button (costs 5 moves)
  - Game over screen
  - Smooth animations
  - Combo notifications for 5+ matches
  - Prevents invalid swaps
