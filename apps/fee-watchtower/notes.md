# Fee Watchtower

## log
- 2026-01-17: Initial creation
  - 3D watchtower with glowing orbs
  - Four token types: ETH, USDC, WBTC, ARB
  - Orbs spawn around tower base and spiral upward
  - Orbs collected at beacon, stats update
  - Live feed shows recent fee events
  - Speed controls (1x, 5x, 10x)
  - Camera orbits tower slowly
  - Neon aesthetic with fog and particles

## features
- 3D watchtower structure (6 segments)
- Glowing beacon at top
- Neon rings between tower segments
- Glowing orbs for each token type
  - ETH: cyan
  - USDC: magenta
  - WBTC: green
  - ARB: yellow
- Orbs spiral upward toward beacon
- Burst effect when orb reaches beacon
- Real-time stats tracking
- Total USD value calculation
- Live feed of collected fees
- Speed multiplier controls
- Ambient particles
- Grid floor
- Auto-orbiting camera

## design
- Orbitron + Share Tech Mono fonts
- Cyan, magenta, green, yellow neon palette
- Dark blue foggy background
- CRT scanlines overlay
- Glowing effects on all elements
