# Neon Glitch Vending Machine

## Log
- 2026-01-10: Initial creation
  - Interactive neon vending machine with radioactive snacks
  - Glitch visual effects when dispensing
  - Web Audio API sounds for special items
  - 9 unique radioactive snacks

## Features
- **Credit System**: Add credits to purchase snacks
- **9 Radioactive Snacks**: Each with unique name and description
- **Glitch Effects**: Screen flickers and color shifts
- **Sound Effects**: Screaming chips scream, static soda makes noise
- **Radiation Particles**: Float up when items dispense
- **Scanline Effect**: CRT-style display screen
- **Hover Descriptions**: See snack descriptions on hover
