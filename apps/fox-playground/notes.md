# Fox Playground

## log
- 2026-01-07: Initial creation - neon fox playground with yarn balls and platforms

## features
- Neon fox character with animated tail wag
- Walk cycle animation when moving
- Interactive yarn balls with physics
- Click/tap yarn to bat it around
- Floating platforms with bobbing animation
- Yarn collection counter
- Double jump capability
- Ground friction and air control

## controls
- WASD / Arrow keys: Move left/right
- Space / W / Up Arrow: Jump (double jump available)
- Click / Tap on yarn: Bat it around

## design
- Dark purple night sky background
- Orange/coral neon fox with glowing effects
- Multi-colored yarn balls (pink, cyan, yellow, lime, orange)
- Cyan neon platforms with glow
- Pixel font (Press Start 2P) for UI
- Particle effects when collecting yarn

## technical
- Canvas 2D rendering
- Custom fox drawing with body parts animation
- Physics simulation (gravity, bounce, friction)
- Platform collision detection
- Yarn ball physics with click interaction
- Touch controls for mobile
