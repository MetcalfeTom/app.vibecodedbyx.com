# Hacker Terminal

## log
- 2025-01-01: Initial creation - Hollywood-style hacker terminal with Matrix rain and dramatic popups

## features
- Matrix-style falling green code (katakana + hex)
- CRT scanline and flicker effects
- Dramatic boot sequence with proxy chain
- Interactive commands: hack, scan, decrypt, download, bypass, trace, nuke, clear, exit
- ACCESS DENIED / ACCESS GRANTED popups with animations
- Progress bars for operations
- Sound effects for popups (Web Audio API)
- Fake IP addresses and server names
- Glitch text effects

## commands
- help - show available commands
- hack <target> - attempt to hack (random success/fail)
- scan - scan network for hosts
- decrypt - decrypt secure files
- download - download database files
- bypass - bypass security protocols
- trace - trace connection route
- nuke - nuclear option (always denied)
- clear - clear terminal
- exit - terminate session

## technical
- Canvas-based Matrix rain animation
- CSS animations for CRT effects
- Web Audio API for sound effects
- Async/await for timed sequences
- Custom progress bar component
