# Sloople

The world's most judgmental search engine. Every query gets roasted and beeped at in morse code.

## log
- 2026-04-12: Initial build. Parody search engine (not a brand clone). Colorful Sloople™ logo. Search box triggers roast messages + morse code beeps of insult words (LOSER, NOPE, LMAO, CRINGE, etc). 18 roast texts, 12 morse words. Keystroke beeps on every input. "I'm Feeling Unlucky" button. Search history with roast log. Body color flash on search. Morse display with dot/dash notation and translation. WebAudio square wave beeps. Gaegu + Inconsolata typography, clean white with red roast accents.

## features
- Parody search engine with colorful logo
- Morse code beeps insult words on every search
- 12 roast words beeped in morse
- 18 roast text responses
- Keystroke beeps on input
- "I'm Feeling Unlucky" button (beeps LOSER)

- Search history log
- Body color flash on search
- WebAudio square wave morse beeps
- Search counter
