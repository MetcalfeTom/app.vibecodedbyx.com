# Narnia Map Generator

## log
- 2026-01-13: Created fantasy map generator for Narnia with magical borders

## features
- Procedurally generated fantasy maps
- Parchment/old map aesthetic
- Animated magical rainbow border
- Terrain generation:
  - Enchanted forests
  - Snow-capped mountains
  - Winding rivers
  - Eastern Sea
- Named landmarks from Narnia lore:
  - Cair Paravel (always present)
  - Lantern Waste
  - Stone Table
  - Dancing Lawn
  - And more
- 4 seasons toggle (spring, summer, autumn, winter)
- Winter mode adds snow to trees
- Add random landmarks button
- Save map as PNG
- Corner decorations (lion, crown, sword, castle)
- Animated compass
- C.S. Lewis quote
- Cinzel fantasy fonts
- Mobile responsive
- Note for Toby about the wardrobe
