Defusal Manual

Three modules. All must be solved before the timer reaches zero. Three strikes detonates.

① Wires

  1. If no red wires: cut wire 2.
  2. Else if the last wire is white: cut wire 1.
  3. Else if there is more than one blue wire: cut the last blue wire.
  4. Else if the serial's last digit is even: cut wire 4.
  5. Otherwise: cut the last wire.

② Switches

Five switches labeled A–E. Set each ON or OFF using the serial number, then submit.

  • A on if serial contains a vowel.
  • B on if serial contains two or more digits.
  • C on if serial's first character is a letter.
  • D on if sum of digits is odd.
  • E on if serial contains the letter X or Z.

③ Sequence

Press the four symbols in this default order: ◯ ◆ ▲ ✕.

  • If the serial contains an odd count of vowels, reverse the order.
  • If the serial's last digit is ≥ 5, swap the first two presses.

④ Charm

Pick the line the bomb finds most charming. Two are cringe, one is sincere.

  • Charming: warm, genuine, a tiny bit literary, no vending-machine metaphors.
  • Cringe: pickup-line clichés (Wi-Fi, parking ticket, fell from the menu).
  • Solving this module also delays the explosion by 30 seconds.

Stand by

You have four minutes to solve all four modules. Three wrong inputs detonates. Manual rules use the bomb's serial number. The ④ Charm module also delays the explosion by 30s when solved.
