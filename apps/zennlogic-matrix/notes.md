## issues
- **Scoring bias**: 8 questions / 6 animals means the matrix can feel biased toward which animals were picked early. Intentional — empathy tests are meant to be slightly arbitrary.
- **No localStorage persistence**: each test is a one-shot. Could add a "history" of past verdicts later.
- **Animal voice synth quality** is whimsical not realistic — meant to evoke barnyard rather than imitate. Sheep "ba-ba-ba" especially abstract.
- **Replicant Index is decorative** — drives a bar but doesn't gate behavior. Could later trigger "FAILSAFE ENGAGED" UI at >80%.
- **Question pool is small** (10) — could expand to 20+ for more session variety.
- **Eye gaze is purely cosmetic** — doesn't track cursor or anything. Considered + decided no, more unsettling when it drifts on its own.
- **Audio context unlock**: must click ENTER button or any other button after first load — Web Audio gesture requirement.
- **Dominant animal tie-breaking**: first animal in declaration order wins ties. Should be fine for 8-question tests.
