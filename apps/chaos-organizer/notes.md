# Chaos Organizer

## log
- 2026-01-11: Created neon kanban board for tracking ghost stories

## features
- 4 columns: Haunting Ideas, Summoning, Manifested, Archived
- Drag and drop cards between columns (desktop + mobile touch)
- Add new ghost story cards with title, description, and vibe tag
- Tags: Spooky, Creepy, Horror, Paranormal
- Local storage persistence
- Pre-populated with sample ghost stories
- Delete cards with X button
- Floating ghost background decorations
- Neon purple aesthetic with Creepster font
- Card counts per column
- Mobile and desktop friendly
