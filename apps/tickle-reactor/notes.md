## features
- Giant neon reactor core (pink/magenta)
- Spinning orbital rings around core
- Click/tap to tickle the reactor
- Shake animation on each tickle
- Energy bar that fills with tickles
- Intensity percentage display
- Tickle count tracker
- Particle burst effects
- Pulse ring expansion
- Floating tickle text popups
- Core text reactions:
  - 0-25%: "TICKLE ME"
  - 25-50%: "Hehe! More!"
  - 50-75%: "AHAHAH STOP!"
  - 75-100%: "TOO MUCH!"
  - 100%: "MELTDOWN!"
- Overload mode at 100%:
  - Constant shaking
  - Screen pulse effect
  - Warning text flashing
  - Auto-particle spawning
- Energy slowly decays over time
- Reset button to restore calm

## controls
- Click/tap reactor core: Tickle it!
- Reset button: Calm the reactor down

## design
