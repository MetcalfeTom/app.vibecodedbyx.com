# VHS Deck

## log
- 2026-01-20: Initial creation - Walten Files style VHS horror aesthetic

## features
- Retro VHS tape deck interface
- Missing person report (Sophie Walten style)
- Heavy static/noise filters
- CRT scanline overlay
- Vignette darkening
- Screen flicker effect
- VHS tracking lines
- Color bleeding/ghosting
- Transport controls (play, stop, rewind, fast forward)
- Static/no signal mode
- Running timestamp
- Tracking bar indicator
- Glitchy text effects
- Random creepy message flashes
- Easter egg: click photo for jump scare

## aesthetic
- VT323 + Special Elite fonts
- Dark blue/black color scheme
- Yellow info text
- Red warning elements
- Simulated Panasonic VHS deck
- "Evidence locker" theme
- 1974 case date (Walten Files era)

## VHS effects
- Per-pixel static noise
- Horizontal distortion bands
- Tracking offset wobble
- Color channel bleeding
- Random heavy glitch bursts
- Rewind visual effect
- Fast forward scan lines

## creepy elements
- Silhouette in photo
- Glitchy white eyes (random)
- Hidden messages:
  - "SHE'S STILL HERE"
  - "THE RABBIT IS WATCHING"
  - "DON'T TRUST HIM"
  - etc.
- Text corruption
