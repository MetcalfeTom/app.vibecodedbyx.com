# Lingua Void

Decode a procedurally generated fictional language through context clues in ancient tablets.

## log
- 2026-04-11: Initial build. Procedural language generator using onset/vowel/coda syllable system. 24 core vocabulary words across 4 categories (nature/action/quality/being). 12 sentence templates with thematic illustrations. 20 tablet puzzles per game — each introduces 1-2 new words mixed with previously seen ones. 4-choice multiple choice with category-aware distractors (same-category wrongs are harder). Lexicon panel tracks all words (decoded vs unknown). Known words show inline translations in subsequent tablets. Contextual hints. Score with penalty for wrong answers. Fully replayable — new language generated each time. Cormorant Garamond + Fira Code typography, dark archaeological aesthetic with gold/purple accent.

## features
- Procedural language generation (unique each playthrough)
- Syllable-based word construction (onset+vowel+coda)
- 24 vocabulary words in 4 semantic categories
- 20 contextual tablet puzzles with gradual word introduction
- Previously decoded words show translations in later sentences
- Lexicon panel tracking decoded vs unknown words
- Category-aware wrong answers (harder distractors)
- Contextual hints from word associations
- Progress bar across all tablets
- Fully replayable with new language each time

## issues
- Some generated words may sound similar
- Sentence templates are fixed (word slots vary)
- No audio
