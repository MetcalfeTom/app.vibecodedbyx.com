# lidless

## log
- 2026-04-29: Created. **Fourth-wall-breaking sentient eye that watches your cursor and judges aloud.** Single-canvas full-viewport procedural eye + behavior detector + dialogue engine. **Eye render**: almond-shape sclera (flattened ellipse `rx=0.32*min(W,H)*1.6, ry=0.32*min(W,H)`) with cream radial-gradient body (`#f6ead0 → #e9dcc0 → #a89570`); **18 procedural red veins** rooted at the rim, each a 6-segment polyline with `sin(seed)` wobble + ≤3 levels of recursive branching, deterministic by index so they don't shimmer; **iris** offset toward `state.mouseX/Y` clamped to ellipse, drawn as 96 radial fibers at randomized lengths/widths/hues + outer rim radial gradient + collarette ring + base radial-gradient fill (hue shifts amber→hot when `moodTilt>0`, deeper amber when `moodTilt<0`); **pupil** pure-black disc with rim halo, dilation `state.pupilDilate ∈ [0.65, 1.6]` (clicks spike +0.18, idle drifts up, frantic shrinks down); two **catch-lights** (upper-left big specular + offset secondary highlight); **eyelid blink** every 4.5–11.5s — 4-phase state machine (`open→closing→closed→opening`) with bezier curves clipped to the sclera ellipse, top + bottom lids meet at 50% close; **outer almond outline** with 3px ink stroke + 6px lash hint along upper arc. **Mood-tinted halo**: outer radial-gradient swells red-orange when irritated, cool blue-cyan when calm. **Behavior detection**: `pointermove` samples speed (px/s) + erraticness (atan2 angle deltas, EMA at 0.88/0.12) + position; `mouseleave/mouseenter` fire `leaving`/`returning`; `click` increments + dilates pupil; `contextmenu` fires `rightclick` (and `preventDefault`s the menu); `keydown` catches F5 / Ctrl+R / Cmd+R → `refreshAttempt`, F12 → `devtools`; `visibilitychange` fires `tabAway` and (on return after >1s) `tabBack` with timed delta; long-session triggers fire after 90s. **Verdict** computed each frame from state buckets (observing / amused / unsettled / patient / suspicious / pleased / tired / wrathful) — shown in HUD top-left in IM Fell English italic with verdict-color (amber default, blood for sus, cool blue for calm). **Dialogue engine**: 13 categorized line pools (waking, idle, fast, erratic, slow, clicking, leaving, returning, tabAway, tabBack, rightclick, refreshAttempt, devtools, bored, pleased, judgement, longSession), `trigger(bucket)` picks a random non-current line, swaps the centered Cormorant Garamond italic dialogue with a 0.55s blur+rise keyframe, hard-throttled by `suppressUntil = now + 2400ms` so lines don't stomp on each other. `judgeTick` scans for triggers every 1.2s. **Lines include user-requested**: "are you still here?" / "stop moving. it hurts." / "is this all you do?" plus ~50 others ("the cursor is not a leash.", "blink. i dare you.", "you cannot un-see me.", "no menu can save you.", "the inspector is open. so am i.", etc.). **Intro overlay**: "open the eye" with letter-spacing pulse keyframe, click to wake; first words after waking are from the `waking` pool. **HUD top-left**: WATCHED FOR (mm:ss), MOVEMENTS count, CLICKS count, JUDGMENT verdict — IM Fell English SC labels + Major Mono Display numerals. **HUD top-right**: "VOL · I · ANNO MMXXVI" + "oculus indagator" + "· nothing escapes ·" crest. **Aesthetic**: void `#040206 → #0c0509` with 3 radial gradients (amber center, blood top-left, cold-cyan bottom-right) + film-grain dual radial dots @ 3px/7px screen-blend + radial vignette. Fonts: Cormorant Garamond italic (dialogue), IM Fell English (body), IM Fell English SC (labels), Major Mono Display (numerals). Cursor stays as `crosshair`. Mobile: HUD shrinks, dialogue 18px, title letter-spacing tightens. Pollinations OG.

## issues
- Procedural veins are deterministic per index but redrawn every frame — minor cost (~18 paths × 6 segments + branches). Could be cached to an offscreen canvas for cheaper redraw if FPS sags on weak GPUs.
- Dialogue cooldown is global (`suppressUntil`), so a fast-firing event right after a slow one gets dropped. By design — chattiness without throttle would be exhausting.
- F12 keydown is detectable on press but most browsers open devtools on F12 *only* via key, not message — we surface a line either way. Cmd+Opt+I / Ctrl+Shift+I aren't covered (intentionally — those are power-user combos and the line "the inspector is open. so am i." stays surprising).
- F5 / Cmd+R / Ctrl+R triggers a dialogue line BEFORE the page actually refreshes. The line will only be visible if the user pauses; on a real refresh it disappears — accepted as ephemeral easter egg.
- `mouseenter`/`mouseleave` fires on the window which is fine for desktop but on mobile there's no analogue; touch users won't see the `leaving/returning` lines. Acceptable — the rest still works via `pointermove`.
- `state.avgSpeed` uses 85/15 EMA — feels responsive but slow gestures take 5–10 samples to register as slow. Tune the weight if "slow" lines feel late.
- The eye's pupil tracking math has both a complex displacement calc and a simpler clamped fallback (`ix = dx * min(1, k)`), the simpler one wins. The complex calc is dead code — left for future refinement (e.g. angular projection onto the sclera ellipse boundary).
- No persistence yet — every page load starts fresh. Could store cumulative time-watched / total clicks in localStorage to mock returning visitors ("you again. that's the third time today.").
