# Neon Fox Chaos Visualizer

## log
- 2026-01-18: Initial creation
  - Audio reactive particle system
  - Neon fox shape made of particles
  - Microphone input for visualization
  - Chaos level control
  - Frequency mode selection

## features
- Web Audio API microphone input
- Fox shape made of orbiting neon particles
  - Particles pulse and orbit based on audio
  - Glowing eyes that react to bass
  - Nose glow effect
- Chaos particles that explode outward
  - Trail effects
  - Gravity simulation
  - Color variety (orange, magenta, cyan, green, yellow, pink)
- Frequency modes:
  - Bass: responds to low frequencies
  - Mids: responds to mid frequencies
  - Highs: responds to high frequencies
  - All: responds to full spectrum
- Chaos level control (normal to max)
- Audio level meter
- Waveform visualization
- Background glow pulses with bass

## controls
- Activate Mic: Start audio input
- More Chaos: Increase particle intensity
- Mode: Switch between bass/mids/highs/all

## design
- Orbitron font for futuristic feel
- Orange/magenta/cyan neon palette
- Dark background with radial overlay
- Particle trails with fade effect
- Glow effects throughout
