# Bear Cycle

## log
- 2026-01-19: Initial creation - grizzly bear on unicycle dodges campers

## features
- Grizzly bear riding unicycle (hand-drawn style)
- Spinning wheel animation
- Bear bobbing motion while riding
- Campers with colorful tents as obstacles
- Blueberry clusters to collect
- 3 lives system with heart display
- Score counter for berries
- Increasing difficulty (speed + spawn rate)
- Touch controls for mobile
- Forest background with trees and clouds

## design
- Cheerful cartoon aesthetic
- Fredoka + Outfit fonts
- Green/blue nature color palette
- Hand-drawn character style
- Colorful tent variations
- Berry glow effect

## controls
- Arrow Keys: Move in all directions
- WASD: Alternative movement
- Touch: Drag up/down on mobile

## gameplay
- Bear moves on left side of screen
- Campers scroll from right
- Berries worth 10 points each
- Hit camper = lose life
- Speed increases with score
- Game over at 0 lives

## entities
- Bear: 60x80, can move vertically and slightly horizontal
- Campers: Tent + person, random colors
- Berries: Cluster of 6 purple circles with glow
