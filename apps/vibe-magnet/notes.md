# Vibe Magnet

Your cursor is a powerful magnet. Your emotional state controls whether it pulls or pushes objects.

## log
- 2026-04-16: Initial build. Canvas physics game with 5 emotional vibes (Calm, Hyped, Angry, Dreamy, Chaotic). Each vibe changes magnet behavior — pull strength, push strength, speed, colors, background tint. Objects spawn from edges with matching vibe colors. Absorb matching-color objects for points, push wrong-vibe objects off screen for bonus. Wrong absorptions penalize score and energy. Combo system, vibe energy meter, cursor trail, force field visualization, particle effects on absorb/push/vibe-shift. Rubik Mono One + Lexend typography. Click to cycle vibes. Touch support.

## features
- 5 emotional vibes with distinct physics behavior
- Canvas-based magnet cursor with force field visualization
- Absorb matching objects, push wrong ones away
- Combo scoring system
- Vibe energy meter
- Particle effects and cursor trail
- Touch/mobile support
- Subtle grid background that shifts with vibe speed
