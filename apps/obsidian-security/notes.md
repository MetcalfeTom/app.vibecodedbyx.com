# Obsidian Security

## log
- 2026-01-07: Initial creation - neon security dashboard with obsidian armor theme

## features
- Armor integrity ring with animated gradient fill
- Live threat matrix grid (64 cells updating)
- Defense metrics with animated progress bars
- Real-time statistics counters
- Security activity feed with auto-updates
- Encryption stream visualizer (hex codes)
- Network perimeter visualization with animated packets
- Obsidian dark theme with neon accents

## design
- Dark obsidian background (#0d0d0f)
- Cyan/gold/magenta neon accents
- Rajdhani font for headers
- Share Tech Mono for data displays
- Animated shield logo
- Subtle grid overlay background
- Panel cards with gradient edges

## technical
- Pure CSS animations for loading effects
- Canvas for network visualization
- Live updating statistics
- Activity feed with newest-first insertion
- Responsive grid layout
- Hex encryption stream generator

## metrics displayed
- Armor Integrity (95%)
- Firewall Strength
- Encryption Level (AES-256)
- Intrusion Detection
- Obsidian Plating
- Threats Blocked counter
- Scans Completed counter
- Breaches (always 0!)
- Uptime percentage
